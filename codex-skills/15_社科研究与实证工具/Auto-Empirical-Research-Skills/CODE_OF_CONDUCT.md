# Code of Conduct

## Our Standard

This project is for researchers, builders, and maintainers improving agent skills for empirical research. Participants should be professional, specific, and evidence-oriented.

Examples of welcome behavior:

- Giving concrete feedback with file paths, links, or reproducible commands.
- Respecting upstream authors and preserving attribution.
- Explaining uncertainty when evaluating research tools or claims.
- Keeping disagreements focused on technical, methodological, or documentation quality.

Examples of unacceptable behavior:

- Harassment, insults, threats, or personal attacks.
- Publishing private information without consent.
- Submitting intentionally misleading, malicious, or license-violating skill entries.
- Repeatedly derailing issue or pull request discussions.

## Enforcement

Maintainers may edit or delete comments, close discussions, reject contributions, or block participants when behavior damages the project or its users.

If you need to report conduct concerns, open an issue with only the minimum public detail needed and ask for a private maintainer contact path.
