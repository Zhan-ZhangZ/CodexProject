# /evidence-rules

1. 读取 `~/.claude/plugins/config/claude-for-legal/china-law/CLAUDE.md` → 管辖与法律框架。
2. 通过北大法宝 MCP 检索《民事诉讼法》及其司法解释中的证据规定 `[PKULaw]`。
3. 根据案件类型和争议焦点确定举证责任分配。
4. 评估现有证据的证明力、可采性和潜在风险。
5. 拟定证据收集和补强策略。

---

## Purpose

中国的法官是事实认定者——没有陪审团。证据规则决定什么能进入法庭、什么会被排除、谁承担举不出证据的后果。本技能帮助你在提交证据前知道：你现有的证据是否足够，还缺什么，以及对方可能怎么攻击你的证据。

## 法律框架

**核心法律:**
- 《中华人民共和国民事诉讼法》(2023修正) 第六章 证据（第66-80条）`[PKULaw]`
- 《最高人民法院关于适用〈中华人民共和国民事诉讼法〉的解释》(2022修正) 证据部分（第90-124条）`[PKULaw]`
- 《最高人民法院关于民事诉讼证据的若干规定》(2019修正)(法释〔2019〕19号) `[PKULaw]` — 最核心的民事诉讼证据规则
- 《最高人民法院关于知识产权民事诉讼证据的若干规定》`[PKULaw]`
- 《人民法院在线诉讼规则》`[PKULaw]`
- 《最高人民法院关于互联网法院审理案件若干问题的规定》`[PKULaw]`

## Step 1 — 证据种类识别

### 八种法定证据（《民事诉讼法》第66条 `[PKULaw]`）

| 证据类型 | 说明 | 实务要点 |
|---------|------|---------|
| (一) 当事人的陈述 | 原告/被告的当庭或书面陈述 | 一般不能单独作为定案依据；须有其他证据印证 |
| (二) 书证 | 以文字、符号、图形记载的内容证明案件事实 | 原件优先；复印件需对方认可或与其他证据印证；**公文书证**推定真实 |
| (三) 物证 | 以物品的外形、特征、质量等证明案件事实 | 原物优先；难以搬动/不易保存可提供照片/录像 |
| (四) 视听资料 | 录音、录像等 | **2019证据规定第90条**：不得以"未经对方同意"为由否认录音合法性（以侵害他人合法权益或违反法律禁止性规定的方法取得的除外） |
| (五) 电子数据 | 邮件、微信记录、网页、博客、短信、电子签名、域名等 | **2019证据规定第93-94条**：电子数据真实性审查规则 `[PKULaw]` |
| (六) 证人证言 | 了解案件情况的人向法院所作陈述 | 一般应出庭；无正当理由未出庭的书面证言不能单独作为定案依据 |
| (七) 鉴定意见 | 鉴定人对专门性问题出具的意见 | 由法院委托/当事人自行委托；**2019证据规定第40条**：重新鉴定条件 |
| (八) 勘验笔录 | 法院对物证/现场的勘验记录 | 法院依职权或依申请进行 |

## Step 2 — 举证责任分配

### 一般规则：谁主张，谁举证（《民事诉讼法》第67条 `[PKULaw]`）

### 举证责任倒置的常见情形

| 案件类型 | 倒置内容 | 依据 |
|---------|---------|------|
| 环境污染侵权 | 加害人就免责事由及其行为与损害结果之间不存在因果关系承担举证责任 | 《民法典》第1230条 `[PKULaw]` |
| 医疗损害（推定过错情形） | 医疗机构就无过错/不存在因果关系承担举证责任（仅限《民法典》第1222条推定过错情形） | 《民法典》第1222条 `[PKULaw]` |
| 劳动争议 | 用人单位对开除、除名、辞退、解除劳动合同、减少劳动报酬、计算劳动者工作年限等决定的依据承担举证责任 | 《劳动争议调解仲裁法》第6条 `[PKULaw]` |
| 个人信息保护 | 个人信息处理者对其已履行PIPL规定的义务承担举证责任（不能证明无过错的 → 承担损害赔偿等侵权责任） | 《个人信息保护法》第69条 `[PKULaw]` |
| 证券虚假陈述 | 信息披露义务人对其无过错承担举证责任 | 《证券法》第85条 `[PKULaw]` |

### 文书提出命令（《民事诉讼法解释》第112条 `[PKULaw]` + 2019证据规定第45-48条 `[PKULaw]`）

- 书证在对方当事人控制之下 → 可申请法院责令对方提交
- 对方无正当理由拒不提交 → 可认定申请人主张的书证内容为真实
- 对方以妨碍对方使用为目的毁灭书证/使其不能使用 → 可认定申请人主张的书证证明事实为真实 + 可罚款/拘留

### 法院调查收集（《民事诉讼法》第67条 `[PKULaw]`）

法院依职权主动调查（第96条司法解释 `[PKULaw]`）：
1. 涉及可能损害国家利益、社会公共利益的
2. 涉及身份关系的
3. 涉及民事诉讼法第58条规定诉讼的（公益诉讼）
4. 当事人有恶意串通损害他人合法权益可能的
5. 涉及依职权追加当事人、中止诉讼、终结诉讼、回避等程序性事项的

## Step 3 — 证据三性审查

### 3.1 真实性

**书证真实性审查：**
- 原件 vs. 复印件（非原件须对方承认或能与其他证据印证）
- 公章/签名的完整性
- 形成时间和修改痕迹
- 与其他证据是否相互印证

**电子数据真实性审查（2019证据规定第93条 `[PKULaw]`）：**
- 电子数据生成、存储、传输所依赖的计算机系统硬件、软件环境是否完整、可靠
- 是否在正常的业务活动中生成、存储、传输
- 生成、存储、传输时主体是否适当
- 是否被完整保存、传输、提取，保持原始状态
- 是否系在正常往来中形成和存储
- 必要时通过鉴定或勘验来判断

以下电子数据推定真实（第94条 `[PKULaw]`）：
- 由当事人提交或保管的于己不利的电子数据
- 由中立第三方平台提供或确认的
- 在正常业务活动中形成的
- 以档案管理方式保管的
- 以当事人约定的方式保存、传输、提取的

### 3.2 合法性（2019证据规定第90条 `[PKULaw]`）

**非法证据排除：**
- 以严重侵害他人合法权益的方法获取的证据
- 以违反法律禁止性规定的方法获取的证据
- 以严重违背公序良俗的方法获取的证据

**录音证据：** 未经对方同意的录音不一定非法——关键在于是否"以侵害他人合法权益"或"违反法律禁止性规定"的方法录制。正常商谈中的秘密录音一般不视为非法。

### 3.3 关联性

- 证据与待证事实之间是否存在逻辑联系
- 是否存在过度推理的"证据链断裂"
- 间接证据能否形成完整的证据链

## Step 4 — 自认规则

### 自认的效力（2019证据规定第5-9条 `[PKULaw]`）

- 一方当事人在诉讼中（包括在起诉状、答辩状、代理词等书面材料中）明确承认对方主张的于己不利的事实 → 对方无需举证
- 在诉讼之外的书面自认也有证据效力

### 自认的撤回

- 法庭辩论终结前 → 对方同意或自认是在受胁迫/重大误解下作出的（须举证）
- 调解/和解中的让步 → 不构成自认（2019证据规定第7条 `[PKULaw]`）

### 不适用自认的情形（2019证据规定第8条 `[PKULaw]`）

- 涉及身份关系的事实
- 涉及国家利益、社会公共利益的事实
- 涉及法院依职权调查的程序性事项

## Step 5 — 证明标准

| 案件类型 | 标准 | 表述 |
|---------|------|------|
| 一般民事诉讼 | 高度盖然性（高度可能性） | "确信待证事实的存在具有高度可能性" |
| 欺诈、胁迫、恶意串通 | 排除合理怀疑 | "能够排除合理怀疑" |
| 口头遗嘱/赠与 | 排除合理怀疑 | "能够排除合理怀疑" |
| 民事诉讼程序事实 | 疏明（低于高度盖然性） | 对程序事项的证明程度可低于实体事项 |
| 刑事 | 排除合理怀疑 | 《刑事诉讼法》第55条 |

## Step 6 — 举证时限

### 举证期限（2019证据规定第50-52条 `[PKULaw]`）

- 一审普通程序：不少于**15日**
- 二审：不少于**10日**
- 简易程序：不超过**15日**
- 小额诉讼程序：不超过**7日**
- 当事人可协商一致并经法院认可

### 逾期举证的后果（《民事诉讼法》第68条 `[PKULaw]`）

- 逾期说明理由 → 拒不说明或理由不成立 → 法院可不予采纳或采纳但予以训诫/罚款
- 但"与案件基本事实有关的证据"即使逾期也应采纳

---

## Step 7 — 质证

### 质证内容（2019证据规定第83条 `[PKULaw]`）

围绕证据的**真实性、合法性和关联性**进行：
1. 对证据的真实性有无异议
2. 对证据的合法性有无异议
3. 对证据的关联性有无异议
4. 对证明目的有无异议

### 不得公开质证的情形

- 涉及国家秘密
- 涉及商业秘密 → 当事人申请不公开
- 涉及个人隐私

## 注意事项

- 中国民事诉讼无美国式"证据开示"（discovery）——不能要求对方交出所有相关材料，文书提出命令的范围远小于美国discovery。
- 电子数据（微信、邮件等）的证明难度通常低于实体书证——注意电子数据的完整性和原始性保全（截图+录屏+公证/时间戳/区块链存证）。
- 证人证言在庭审中的分量通常不如书证和电子数据——中国法官更重书证，证人证言属辅助证据。
- 逾期举证可能导致证据不被采纳——把握举证时限。
