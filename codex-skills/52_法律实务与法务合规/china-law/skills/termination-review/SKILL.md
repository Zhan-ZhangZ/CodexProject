# /termination-review

评估解除或终止劳动合同的法律风险、程序要求和补偿/赔偿义务。

## 指令

### 1. 事务上下文
- 解除事由？（劳动者过失/非过失/经济性裁员/协商一致/合同到期不续）
- 劳动者工作年限？
- 解除前12个月平均工资？（是否超过本地区上年度职工月平均工资三倍？）
- 是否有竞业限制协议？（解除后补偿义务）
- 是否涉及工伤、三期（孕期、产期、哺乳期）、医疗期？

### 2. 加载实践档案

### 3. 解除类型判定与法律框架

#### 3.1 劳动者过失性解除（《劳动合同法》第39条 `[PKULaw]`）

- 试用期被证明不符合录用条件（需有明确的录用条件和考核记录）
- 严重违反用人单位规章制度（需有合法有效的规章制度）
- 严重失职、徇私舞弊给用人单位造成重大损害
- 劳动者同时与其他用人单位建立劳动关系且对本职工作有严重影响
- 欺诈、胁迫或乘人之危使用人单位在违背真实意思下订立合同
- 被追究刑事责任

→ **无需支付经济补偿**

#### 3.2 非过失性解除（《劳动合同法》第40条 `[PKULaw]`）

- 劳动者患病或非因工负伤，医疗期满后不能从事原工作且不能从事另行安排的工作
- 劳动者不胜任工作，经培训或调岗后仍不胜任
- 客观情况重大变化，致使合同无法履行且无法协商变更

→ **提前30日书面通知或支付代通金（1个月工资）**
→ **需支付经济补偿（第46条）**

#### 3.3 经济性裁员（《劳动合同法》第41条 `[PKULaw]`）

条件（任一 + 人数要求）：
- 依照企业破产法重整
- 生产经营发生严重困难
- 转产、重大技术革新或经营调整，变更合同后仍需裁减
- 其他因客观经济情况变化

人数要求：裁减20人以上或不足20人但占职工总数10%以上
程序：提前30日向工会或全体职工说明 → 听取意见 → 向劳动行政部门报告方案

→ **需支付经济补偿**

#### 3.4 禁止解除情形（《劳动合同法》第42条 `[PKULaw]`）

第40条和第41条不适用于：
- 从事接触职业病危害作业未进行离岗前职业健康检查，或疑似职业病病人在诊断或医学观察期间的
- 在本单位患职业病或因工负伤并被确认丧失或部分丧失劳动能力的
- 患病或非因工负伤，在规定的医疗期内的
- 女职工在孕期、产期、哺乳期的
- 在本单位连续工作满15年且距法定退休年龄不足5年的
- 法律、行政法规规定的其他情形

→ **若以第40、41条为由解除 → 🔴 违法解除**

#### 3.5 协商一致解除（《劳动合同法》第36条 `[PKULaw]`）

→ **需支付经济补偿（用人单位提出）**

#### 3.6 合同到期终止（《劳动合同法》第44条 `[PKULaw]`）

- 合同期满 → 用人单位不续签 → 支付经济补偿
- 合同期满 → 用人单位维持或提高条件续签，劳动者不同意 → 无需支付
- 劳动者开始依法享受基本养老保险待遇 → 无需支付
- 劳动者死亡或被法院宣告死亡/失踪 → 无需支付
- 用人单位被吊销营业执照、责令关闭、撤销或提前解散 → 支付经济补偿

### 4. 经济补偿金计算（《劳动合同法》第47条 `[PKULaw]`）

经济补偿 = 劳动者在本单位工作年限 × 月工资

- 每满1年支付1个月工资
- 6个月以上不满1年 → 按1年计算
- 不满6个月 → 支付半个月工资
- 月工资高于本地区上年度职工月平均工资3倍 → 按3倍计算，年限上限12年
- 月工资定义：解除或终止前12个月的平均工资（《劳动合同法实施条例》第27条）

### 5. 违法解除赔偿金（《劳动合同法》第87条 `[PKULaw]`）

赔偿金 = 经济补偿标准的二倍

→ 支付赔偿金后，不再支付经济补偿（《劳动合同法实施条例》第25条 `[PKULaw]`）

### 6. 程序要求

- 需事先将理由通知工会（《劳动合同法》第43条）
- 出具解除或终止劳动合同证明（第50条）
- 在15日内为劳动者办理档案和社会保险转移手续

### 7. 输出

审查者说明 + 解除合法性分析 + 补偿/赔偿计算 + 程序要求清单 + 特别风险提示（工伤三期医疗期） + 决策树。

- 《劳动合同法》第36-50条、第87条 `[PKULaw]`
- 《劳动合同法实施条例》第25条 `[PKULaw]`

- 《劳动争议调解仲裁法》第27条（仲裁时效1年）`[PKULaw]`
