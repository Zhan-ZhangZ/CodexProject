# /labor-contract-review

逐条审查劳动合同的合法性、完整性和公平性。默认站在雇主立场（企业法务视角），
`--side employee` 切换为劳动者视角。

## 指令

### 1. 事务上下文
- 审阅立场？（雇主/劳动者/中立）
- 合同签订地？（省/直辖市——地方规定可能有差异，如上海、北京、广东有独立的劳动合同管理办法）

### 2. 加载实践档案
读取 `~/.claude/plugins/config/claude-for-legal/china-law/CLAUDE.md`，获取：
- 公司的竞业限制政策、补偿标准、违约金约定
- 社保和公积金缴纳政策
- 合同模板偏好

### 3. 核心审查清单

#### 3.1 必备条款检查（《劳动合同法》第17条 `[PKULaw]`）

| # | 必备条款 | 状态 |
|---|---------|------|
| 1 | 用人单位的名称、住所和法定代表人或者主要负责人 | ✓/✗ |
| 2 | 劳动者的姓名、住址和居民身份证或者其他有效身份证件号码 | ✓/✗ |
| 3 | 劳动合同期限 | ✓/✗ |
| 4 | 工作内容和工作地点 | ✓/✗ |
| 5 | 工作时间和休息休假 | ✓/✗ |
| 6 | 劳动报酬 | ✓/✗ |
| 7 | 社会保险 | ✓/✗ |
| 8 | 劳动保护、劳动条件和职业危害防护 | ✓/✗ |
| 9 | 法律、法规规定应当纳入劳动合同的其他事项 | ✓/✗ |

#### 3.2 试用期检查（《劳动合同法》第19条 `[PKULaw]`）

- 合同期限3个月以上不满1年 → 试用期 ≤ 1个月
- 合同期限1年以上不满3年 → 试用期 ≤ 2个月
- 合同期限3年以上或无固定期限 → 试用期 ≤ 6个月
- 以完成一定工作任务为期限的合同 / 合同期限不满3个月 → 不得约定试用期
- 同一用人单位与同一劳动者只能约定一次试用期
- 试用期工资 ≥ 合同约定工资的80% 且 ≥ 用人单位所在地最低工资标准（《劳动合同法》第20条）

#### 3.3 竞业限制检查（《劳动合同法》第23-24条 `[PKULaw]`）

- 竞业限制人员范围：高级管理人员、高级技术人员和其他负有保密义务的人员
- 竞业限制期限：≤ 2年（解除或终止劳动合同后）
- 竞业限制补偿：解除或终止后按月支付
  - 未约定补偿金额 → 解除前12个月平均工资的30%（司法解释一第36条 `[PKULaw]`）
  - 低于最低工资标准 → 按最低工资标准
- 用人单位3个月未支付补偿 → 劳动者可请求解除竞业限制约定（司法解释一第38条 `[PKULaw]`）
- 违约金应合理，不宜超补偿总额5倍（《企业实施竞业限制合规指引》第14条 `[PKULaw]`）

#### 3.4 服务期检查（《劳动合同法》第22条 `[PKULaw]`）

- 用人单位为劳动者提供专项培训费用，进行了专业技术培训 → 可约定服务期
- 违约金 ≤ 用人单位支付的总培训费用
- 违约金按未履行服务期比例递减

#### 3.5 违约金限制（《劳动合同法》第25条 `[PKULaw]`）

除竞业限制和服务期外，不得约定由劳动者承担的违约金。

### 4. 严重度分类

| 严重度 | 标准 |
|-------|------|
| 🔴 阻断性 | 违反强制性规定（如合同期限不满3个月设试用期、非法违约金）、缺失必备条款 |
| 🟠 高 | 竞业限制范围过大、补偿低于法定标准、试用期工资低于法定标准 |
| 🟡 中 | 条款不清晰可能引起争议、加班费计算基数约定不明确 |
| 🟢 低 | 格式问题、建议性补充（如送达地址条款） |

### 5. 输出

审查者说明 + 按严重度排序的发现清单 + 条款对照表 + 决策树。
