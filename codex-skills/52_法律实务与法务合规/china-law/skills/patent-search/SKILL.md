# /patent-search

中国专利在先权利检索——自由实施(FTO)分析、无效检索、新颖性评估。

## 指令

### 1. 事务上下文

- 技术领域？
- 检索目的？（FTO/无效检索/新颖性评估/现有技术调查）
- 拟实施的地域范围？（中国境内/含出口）
- 产品生命周期阶段？（研发/上市前/已上市）

### 2. 核心法律框架（《专利法》(2020修正) `[PKULaw]`）

#### 2.1 专利类型（第2条 `[PKULaw]`）

- **发明：** 对产品、方法或其改进提出的新的技术方案
- **实用新型：** 对产品的形状、构造或其结合提出的适于实用的新的技术方案
- **外观设计：** 对产品的整体或局部的形状、图案或其结合以及色彩与形状、图案的结合所作出的富有美感并适于工业应用的新设计

#### 2.2 授权条件

**发明和实用新型（第22条 `[PKULaw]`）：**
- 新颖性：不属于现有技术 + 无抵触申请
- 创造性：与现有技术相比具有突出的实质性特点和显著的进步（发明）/ 实质性特点和进步（实用新型）
- 实用性：能够制造或使用 + 产生积极效果

**外观设计（第23条）：**
- 不属于现有设计 + 无抵触申请
- 与现有设计或现有设计特征的组合相比具有明显区别

#### 2.3 现有技术与抵触申请

- **现有技术/设计：** 申请日以前在国内外为公众所知的技术/设计
- **抵触申请：** 申请日以前他人向专利局提出但申请日后公布的相同发明/实用新型申请

#### 2.4 专利保护范围（第64条 `[PKULaw]`）

- 发明/实用新型 → 以权利要求的内容为准，说明书及附图可用于解释
- 外观设计 → 以图片或照片中的外观设计为准

#### 2.5 不视为侵权的情形（第75条 `[PKULaw]`）

- 权利用尽后的使用/销售
- 先用权（申请日前已制造相同产品/使用相同方法/已作好必要准备）
- 临时过境的外国交通工具
- 专为科学研究和实验
- Bolar例外（药品和医疗器械行政审批）

#### 2.6 侵权赔偿（第71条 `[PKULaw]`）

- 实际损失 / 侵权获利 / 许可使用费倍数
- 法定赔偿：3万元至500万元
- 故意+情节严重 → 1倍以上5倍以下惩罚性赔偿

#### 2.7 诉讼时效（专利纠纷司法解释修正后第23条 `[PKULaw]`）

侵犯专利权诉讼时效为3年，自专利权人或利害关系人知道或应当知道侵权行为以及侵权人之日起计算

### 3. 检索策略

#### 3.1 检索步骤

1. **技术方案分解** → 提取技术特征 → 确定检索要素
2. **关键词构造** → 技术术语 + 同义词 + 上位/下位概念
3. **分类号确定** → IPC分类号（发明/实用新型）/ 洛迦诺分类号（外观设计）
4. **数据库检索** → 中国国家知识产权局(CNIPA)专利数据库 / 商业数据库
5. **引证追踪** → 前引和后引分析

#### 3.2 检索数据库

**中文优先：**
- 中国专利公布公告系统（CNIPA）
- 北大法宝专利检索

**国际补充：**
- WIPO PATENTSCOPE
- EPO Espacenet
- 各国专利局数据库

#### 3.3 FTO 检索清单

| 检索项 | 范围 | 状态 |
|-------|------|------|
| 有效发明专利（中国） | 技术领域 + 关键词 + IPC | ✓/✗ |
| 有效实用新型专利（中国） | 产品结构特征 | ✓/✗ |
| 有效外观设计专利（中国） | 产品外观特征 | ✓/✗ |
| 已公布未授权申请 | 可能构成临时保护 | ✓/✗ |
| 已过期/无效专利 | 排除 | ✓/✗ |
| 专利权属链 | 确认权利人 | ✓/✗ |

### 4. 检索结果评估

#### 4.1 风险分级

| 严重度 | 标准 |
|-------|------|
| 🔴 阻断性 | 发现有效专利，权利要求完全覆盖技术方案；无回避设计空间 |
| 🟠 高 | 发现相关有效专利，部分技术特征落入保护范围；回避设计可行但成本高 |
| 🟡 中 | 发现相关申请（未授权）；现有专利可挑战有效性（新颖性/创造性问题） |
| 🟢 低 | 未发现相关有效专利；相关专利已过期/无效；技术方案在公有领域 |

#### 4.2 应对策略

- **回避设计：** 修改技术方案绕开权利要求保护范围
- **无效宣告：** 检索现有技术，向专利局复审和无效审理部提起无效请求
- **取得许可：** 与专利权人协商许可
- **先用权抗辩：** 证明在申请日前已实施/已作好必要准备
- **等待到期：** 评估专利剩余有效期

### 5. 输出

检索报告 + 相关专利清单 + 技术特征比对表 + FTO风险分级 + 应对策略建议 + 决策树。
