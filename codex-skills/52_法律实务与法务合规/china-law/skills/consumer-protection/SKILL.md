# /consumer-protection

1. 读取 `~/.claude/plugins/config/claude-for-legal/china-law/CLAUDE.md` → 管辖与法律框架。
2. 确定消费者保护事项类型。
3. 通过北大法宝 MCP 检索适用法规 `[PKULaw]`。
4. 进行消费者法律分析（权利/义务/责任/救济）。
5. 出具附风险分级和应对建议的分析报告。

---

## Purpose

中国消费者保护法历来以"向消费者倾斜"为其核心立场。从《消费者权益保护法》(2013修正) 引入的"退一赔三"惩罚性赔偿制度，到《食品安全法》的"退一赔十"制度，再到《最高人民法院关于审理食品药品纠纷案件适用法律若干问题的规定》对"知假买假"的宽容态度，消费者权益在中国的法律保护力度仅次于刑法。本技能提供中国消费者保护法框架下的系统分析方法。

## 审查类型

| 类型 | 适用场景 |
|------|---------|
| 消费者纠纷分析 | 单次消费纠纷的法律地位评估和解决方案分析 |
| 消费者保护合规审查 | 企业经营活动是否符合消费者保护法规的全面审查 |
| 产品质量与产品责任 | 缺陷产品致人损害的责任分析和应对 |
| 电子商务消费者保护 | 平台/电商经营者特有的消费者保护义务 |
| 预付式消费（预付卡）合规 | 预付式消费的法律规制和风险 |
| 虚假广告与误导宣传 | 广告合规审查 + 虚假广告的法律后果 |

## Step 2 — 消费者权利体系

《消费者权益保护法》(2013修正) `[PKULaw]`：

| 权利 | 内容 | 法条 |
|------|------|------|
| 安全权 | 购买、使用商品和接受服务时享有人身、财产安全不受损害的权利 | 第7条 |
| 知情权 | 知悉其购买、使用的商品或接受的服务的真实情况的权利 | 第8条 |
| 自主选择权 | 自主选择商品或服务的权利（包括不买的权利） | 第9条 |
| 公平交易权 | 有权获得质量保障、价格合理、计量正确等公平交易条件 | 第10条 |
| 求偿权 | 因购买、使用商品或接受服务受到人身、财产损害的，享有依法获得赔偿的权利 | 第11条 |
| 成立社团权 | 依法成立维护自身合法权益的社会组织的权利 | 第12条 |
| 获得知识权 | 获得有关消费和消费者权益保护方面的知识的权利 | 第13条 |
| 人格尊严受尊重权 | 人格尊严、民族风俗习惯得到尊重的权利 | 第14条 |
| 个人信息受保护权 | 个人信息依法受到保护的权利 | 第14条/第29条 |
| 监督权 | 对商品和服务以及保护消费者权益工作进行监督的权利 | 第15条 |

## Step 3 — 经营者义务与责任

### 3.1 经营者义务清单

| 义务 | 内容 | 法条 |
|------|------|------|
| 安全保障义务 | 宾馆/商场/银行/车站/娱乐场所等经营场所的经营者，未尽到安全保障义务，造成他人损害的，应当承担侵权责任 | 第18条/《民法典》第1198条 |
| 缺陷产品召回义务 | 发现商品或服务存在缺陷有危及人身、财产安全危险的，立即报告并告知消费者+采取停止销售、警示、召回等措施 | 第19条 |
| 真实信息提供义务 | 不得作虚假或引人误解的宣传 | 第20条 |
| 明码标价义务 | 提供商品或服务应当明码标价 | 第20条 |
| 标明真实名称和标记义务 | 租赁他人柜台或场地的经营者也应标明真实名称和标记 | 第21条 |
| 出具凭证/单据义务 | 按规定或商业惯例出具发票等购货凭证或服务单据 | 第22条 |
| 质量保证义务 | 保证商品或服务应当具有的质量、性能、用途和有效期限 | 第23条 |
| "三包"义务 | 按规定或约定履行包修/包换/包退或其他责任 | 第24条 |
| 不得以格式条款等限制消费者权利 | 格式条款/通知/声明/店堂告示中含有排除或限制消费者权利、减轻或免除经营者责任等不公平、不合理内容的，其内容无效 | 第26条 |
| 个人信息保护义务 | 收集、使用消费者个人信息须合法、正当、必要 + 明示目的、方式和范围并征得同意 | 第29条 |

### 3.2 格式条款公平义务

《消费者权益保护法》第26条 `[PKULaw]` + 《民法典》第496-498条：

**无效的格式条款：**
- 排除消费者依法享有的权利
- 免除经营者应承担的责任（人身伤害责任不可通过格式条款免除）
- 加重消费者责任
- 采用格式条款订立合同时，对与消费者有重大利害关系的内容未以显著方式提请消费者注意 + 未按消费者要求予以说明 → 消费者可主张该条款不成为合同内容

### 3.3 安全保障义务

《民法典》第1198条 `[PKULaw]`：

| 要点 | 内容 |
|------|------|
| 义务主体 | 宾馆、商场、银行、车站、机场、体育场馆、娱乐场所等经营场所/公共场所的经营者/管理者或群众性活动的组织者 |
| 违反义务标准 | 未尽到合理限度范围内的安全保障义务 |
| 责任性质 | 侵权责任（不作为侵权） |
| 第三人侵权的补充责任 | 因第三人行为造成损害的，第三人承担责任；经营者未尽安全保障义务的，承担相应的**补充责任** |

## Step 4 — 惩罚性赔偿制度

### 4.1 "退一赔三"制度（欺诈）

《消费者权益保护法》第55条第1款 `[PKULaw]`：

| 要件 | 内容 |
|------|------|
| 适用前提 | 经营者提供商品或服务有**欺诈行为** |
| 赔偿标准 | 退还价款 + 赔偿价款**3倍（三倍）** |
| 最低赔偿额 | 增加赔偿的金额不足**500元**的，为**500元** |

**欺诈行为的认定：** 经营者故意告知虚假情况或故意隐瞒真实情况，诱使消费者作出错误意思表示的——"知假买假"者是否属于"被欺诈"存在争议（最高人民法院指导案例23号(2014)认为食品领域的"知假买假"不影响消费者权利，但食品以外的领域各地法院态度不一）。

### 4.2 "退一赔十"制度（食品安全）

《食品安全法》第148条第2款 `[PKULaw]`：

| 要件 | 内容 |
|------|------|
| 适用前提 | 生产不符合食品安全标准的食品或经营**明知**是不符合食品安全标准的食品 |
| 赔偿标准 | 退还价款 + 赔偿价款**10倍（十倍）**或损失**3倍**（以高者为准） |
| 最低赔偿额 | 增加赔偿的金额不足**1000元**的，为**1000元** |
| 标签/说明书除外 | 食品的标签、说明书存在不影响食品安全且不会对消费者造成误导的瑕疵的除外 |

**⚠️ "知假买假"在食品领域的处理：** 最高人民法院指导案例23号(2014)明确——消费者购买到不符合食品安全标准的食品要求赔偿的，法院应予支持，不论其购买时是否明知食品不符合安全标准（即"知假买假"不影响主张"退一赔十"）。但职业打假的限度在各地方司法实践中仍存在分歧。

### 4.3 产品责任惩罚性赔偿

《民法典》第1207条 `[PKULaw]`：

明知产品存在缺陷仍然生产、销售，或者没有依据前条规定采取有效补救措施，造成他人死亡或者健康严重损害的，被侵权人有权请求相应的**惩罚性赔偿**。

## Step 5 — 电子商务消费者保护

### 5.1 平台经营者特殊义务

《电子商务法》`[PKULaw]`：

| 义务 | 内容 | 法条 |
|------|------|------|
| 平台内经营者身份审核 | 要求申请进入平台销售商品或提供服务的经营者提交其身份、地址、联系方式、行政许可等真实信息，进行核验、登记，建立登记档案 | 第27条 |
| 平台自营与他营区分 | 平台在其平台上开展自营业务的，应以显著方式区分标记自营业务和平台内经营者开展的业务 | 第37条 |
| "先行赔付"责任 | 消费者通过平台购买商品或接受服务，其合法权益受到损害的，平台不能提供销售者或服务者的真实名称、地址和有效联系方式的，消费者可向平台要求赔偿（平台赔偿后有权向销售者或服务者追偿） | 第38条第1款 |
| "连带责任"情形 | 平台知道或应当知道平台内经营者销售的商品或提供的服务不符合保障人身、财产安全的要求，或有其他侵害消费者合法权益行为，未采取必要措施的，依法承担**连带责任** | 第38条第2款 |

### 5.2 直播电商

最高人民法院《关于审理网络消费纠纷案件适用法律若干问题的规定(一)》`[PKULaw]`：

| 直播主体 | 责任 |
|---------|------|
| 直播间运营者 | 标明实际销售者，未标明→承担销售者责任 |
| 直播营销人员（主播） | 如证明其提供的是广告代言，按广告法承担责任；自营自销的承担销售者责任 |
| 平台经营者 | 按电子商务法第38条承担平台责任 |

### 5.3 外卖平台

《网络餐饮服务食品安全监督管理办法》`[PKULaw]`：

- 入网餐饮服务提供者必须有实体经营门店 + 食品经营许可证
- 平台须对入网餐饮服务提供者的食品经营许可证进行审查和登记
- 配送人员的食品安全培训和管理

## Step 6 — 预付式消费（预付卡）

### 6.1 预付卡的法律规制

| 卡种 | 主要规范 | 发行主体 |
|------|---------|---------|
| 单用途商业预付卡 | 《单用途商业预付卡管理办法》(商务部) | 零售业、住宿和餐饮业、居民服务业企业法人 |
| 多用途预付卡 | 《非银行支付机构监督管理条例》+ 《支付机构预付卡业务管理办法》 | 持有支付业务许可证的支付机构 |

### 6.2 预付卡纠纷焦点

| 纠纷类型 | 法律分析 |
|---------|---------|
| 经营者"跑路" | 合同违约 + 可能构成诈骗罪（如具有非法占有目的） |
| 转卡/退卡限制 | 格式条款→公平性审查（转卡/退卡权利不可被不合理限制） |
| 余额退还 | 卡内余额属消费者财产，经营者不可拒绝退还 |
| 过期余额作废 | 通常认定为无效的格式条款（加重消费者责任） |

## Step 7 — 虚假广告与误导宣传

《广告法》(2021修正) `[PKULaw]` + 《反不正当竞争法》第8条 `[PKULaw]`：

| 禁止行为 | 典型示例 | 处罚幅度 |
|---------|---------|---------|
| 虚假广告 | 商品不存在/性能不实/虚称专利/虚假使用数据 | 广告费**3-5倍**罚款（广告费无法计算或明显偏低→**20-100万元**） |
| 误导性广告 | 表述模糊→消费者产生误认 | 同上 |
| "最好""第一""国家级"等绝对化用语 | 广告中使用最高级用语 | 20-100万元（《广告法》第57条） |
| 贬低其他经营者 | 对比广告中贬低竞争者 | 按《反不正当竞争法》处罚 |

| 路径 | 特点 | 时限 |
|------|------|------|
| 与经营者协商和解 | 最快速、成本最低 | 无限制 |
| 消费者协会调解 | 免费、专业 | 无严格时限 |

| 消费者权益保护公益诉讼 | 省级以上消费者协会+检察院 | 3年诉讼时效 |

## 注意事项

- 消费者权益保护法是"向消费者倾斜"的法律——在合同解释、格式条款公平性、举证责任分配等方面均偏向消费者。企业在起草消费者合同时需以此为出发点。
- "退一赔三"(欺诈)和"退一赔十"(食品)是中国法律中赔偿倍数最高的民事赔偿制度之一，且不与实际损失挂钩——意味着即使消费者无实际损失，经营者仍可能面临高额赔偿。
- 食品领域的"知假买假"在实践中仍属灰色地带——《最高人民法院关于审理食品药品纠纷案件适用法律若干问题的规定》(2020修正) 第3条保持了对食品领域"知假买假"的宽容态度，但非食品领域各地法院判决不一。
- 职业打假人对企业构成系统性诉讼压力——特别是针对食品和化妆品的标签标识合规（如生产日期标注不规范、配料表瑕疵等），批量发起的投诉和诉讼可耗费大量法务资源。
