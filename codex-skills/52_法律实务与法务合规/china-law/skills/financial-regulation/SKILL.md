## Purpose

中国金融体系实行分业监管——金融监管总局（银行、保险、信托、消费金融）、证监会（证券、基金、期货、上市公司）、央行（货币政策、支付体系、反洗钱）分别监管不同领域。2023年金融监管体制改革后，金融监管总局整合了原银保监会的全部职能并新增了消费者保护和金融控股公司监管等职责。金融监管规则密度极高——除法律和行政法规外，部门规章和监管规范性文件在执法中具有准强制力。本技能提供中国金融监管框架下的系统合规分析。

## Step 2 — 中国金融监管框架

### 2.1 监管机构与分工

```
中国金融监管体系（2023年改革后）

国务院
├── 中国人民银行（央行）
│   ├── 货币政策和宏观审慎
│   ├── 支付体系监管
│   ├── 反洗钱和反恐怖融资
│   └── 金融统计和征信
│
├── 国家金融监督管理总局（NFRA / 金融监管总局）
│   ├── 银行业监管（原银监会职能）
│   ├── 保险业监管（原保监会职能）
│   ├── 信托、消费金融、金融租赁、汽车金融等
│   ├── 金融控股公司监管（新增）
│   └── 金融消费者/投资者保护（新增，整合原人行和证监会的消保职能）
│
└── 中国证券监督管理委员会（证监会）
    ├── 证券市场（股票/债券/期货）监管
    ├── 证券公司和基金公司监管
    ├── 上市公司监管
    └── 投资者保护（部分）
```

## Step 3 — 金融牌照与市场准入

### 3.1 金融特许经营原则

《商业银行法》第11条 `[PKULaw]` + 《证券法》第118条等：

**⚠️ 金融业务特许经营是硬约束——未经批准从事需许可的金融业务构成非法经营（行政违法）甚至非法经营罪（刑事犯罪）。**

### 3.2 主要金融牌照类型

| 牌照类型 | 发放机关 | 主要法律依据 | 资本金要求（示例） |
|---------|---------|-------------|-----------------|
| 商业银行 | 金融监管总局 | 《商业银行法》 | 全国性≥100亿元（实缴） |
| 证券公司 | 证监会 | 《证券法》 | 按业务类型：5000万-5亿元 |
| 保险公司 | 金融监管总局 | 《保险法》 | 2亿元（实缴货币资本） |
| 信托公司 | 金融监管总局 | 《信托法》《信托公司管理办法》 | 10亿元（实缴） |
| 基金公司 | 证监会 | 《证券投资基金法》 | 1亿元（实缴） |
| 期货公司 | 证监会 | 《期货和衍生品法》 | 3000万-1亿元 |
| 支付机构 | 央行 | 《非银行支付机构监督管理条例》 | 1亿元（实缴） |
| 金融租赁公司 | 金融监管总局 | 《金融租赁公司管理办法》 | 10亿元（实缴） |
| 消费金融公司 | 金融监管总局 | 《消费金融公司管理办法》 | 10亿元（实缴） |
| 金融控股公司 | 央行 | 《金融控股公司监督管理试行办法》 | 50亿元（实缴） |

### 3.3 金融科技与新型牌照

| 业务类型 | 监管要求 |
|---------|---------|
| 网络借贷信息中介（P2P） | **已全部清零**，该牌照已被废除 |
| 互联网小额贷款 | 省级金融监管局审批；跨省经营须向金融监管总局申请 |
| 第三方支付 | 必须持有央行颁发的《支付业务许可证》 |
| 互联网保险 | 持牌机构通过自营网络平台或第三方网络平台开展 |
| 智能投顾 | 须持有投资顾问牌照（证券投资咨询机构） |
| 区块链/数字资产 | 除数字人民币外，已全面禁止虚拟货币交易和ICO |

## Step 4 — 上市公司合规

### 4.1 信息披露合规

《证券法》(2019修订) `[PKULaw]` — 2019年修订大幅加强信息披露要求和处罚力度：

| 审查项 | 合规要点 |
|-------|---------|
| 定期报告 | 年度报告（会计年度结束后4个月内）、半年度报告（下半年开始后2个月内） |
| 重大事件临时公告 | "应当立即"公告——发生重大事件时不能等到定期报告 |
| 内幕信息管理 | 建立内幕信息知情人登记制度 |
| 虚假陈述后果 | 罚款50-1000万元（发行人）+ 民事赔偿（代表人诉讼/特别代表人诉讼） |

### 4.2 内幕交易

《证券法》第50-54条 `[PKULaw]`：

| 禁止行为 | 说明 |
|---------|------|
| 利用内幕信息交易 | 内幕信息知情人或非法获取内幕信息的人在信息公开前买卖/建议他人买卖/泄露内幕信息 |
| 处罚幅度 | 没收违法所得 + **1-10倍**罚款（无违法所得的，处50-500万元罚款） |

### 4.3 操纵市场

| 操纵手段 | 示例 |
|---------|------|
| 连续交易操纵 | 单独或合谋，利用资金、持股或信息优势 |
| 约定交易操纵 | 与他人串通相互买卖 |
| 洗售操纵 | 自买自卖（对倒） |
| 虚假申报操纵 | 不以成交为目的频繁申报和撤销 |

## Step 5 — 资产管理产品监管

### 5.1 《资管新规》框架

《关于规范金融机构资产管理业务的指导意见》(2018年发布，过渡期至2021年底) `[PKULaw]`：

| 要点 | 内容 |
|------|------|
| 打破刚兑 | 金融机构不得以任何形式对资管产品进行保本保收益承诺 |
| 净值化管理 | 资管产品实行净值化估值和披露 |
| 限制嵌套 | 资管产品最多嵌套**2层**（含公募基金在内） |
| 限制杠杆 | 私募产品总资产/净资产不超过**200%** |
| 合格投资者 | 严格合格投资者标准（家庭金融净资产≥300万元/近3年年均收入≥40万元等） |
| 标准化资产 | 标准化债权类资产认定（非标转标） |

### 5.2 私募基金监管

《私募投资基金监督管理条例》(2023年9月1日施行) `[PKULaw]`：

| 要点 | 内容 |
|------|------|
| 管理人登记 | 私募基金管理人须在基金业协会登记 |
| 基金备案 | 私募基金募集完毕后须在基金业协会备案 |
| 合格投资者 | 单只基金投资≥100万元；个人金融资产≥300万元或近3年年均收入≥50万元 |
| 投资者人数 | 有限责任公司型/有限合伙型 + 契约型各自不超过**50人**（合格投资者） |
| 信息披露 | 按约定和监管要求向投资者披露基金信息 |

## Step 6 — 反洗钱与反恐怖融资

### 6.1 反洗钱义务主体

《反洗钱法》`[PKULaw]`：

| 义务主体 | 核心义务 |
|---------|---------|
| 银行/证券/保险/信托/基金/期货 | 客户身份识别 + 大额交易和可疑交易报告 + 客户身份资料保存 |
| 支付机构/互联网金融 | 同上 |
| 非银行支付机构 | 同上 |
| 特定非金融行业（房地产/贵金属/会计师等） | 特定交易报告义务（范围限缩于金融监管总局指定） |

### 6.2 客户尽职调查（KYC）

央行《金融机构客户尽职调查和客户身份资料及交易记录保存管理办法》`[PKULaw]`：

| 审查项 | 要点 |
|-------|------|
| 初次识别 | 建立业务关系时必须识别客户身份 |
| 持续识别 | 业务关系存续期间的持续身份更新 |
| 强化识别（EDD） | 高风险客户/业务须采取强化措施（了解资金来源等） |
| 受益所有人识别 | 穿透至最终自然人受益所有人（持股/控制权益≥25%） |
| 资料保存 | 客户身份资料和交易记录至少保存**5年** |

### 6.3 可疑交易报告

| 报告义务 | 要点 |
|---------|------|
| 大额交易报告 | 单笔或当日累计超过规定金额的现金/转账交易（如现金交易≥5万元人民币） |
| 可疑交易报告 | 有合理理由怀疑交易与洗钱/恐怖融资有关的，不论金额大小 |

**⚠️ "应当知道而未报告"的责任：** 金融机构如果对明显的可疑交易未报告，可能被认定为违反反洗钱义务——不仅仅是罚款，严重的可能涉及洗钱罪的帮助犯（《刑法》第191条洗钱罪的正犯行为包括"明知是...犯罪所得而...掩饰/隐瞒"）。

## Step 7 — 金融违法行为的处罚

### 7.1 行政处罚力度升级

《证券法》(2019修订)对违法行为的处罚大幅提高：

| 违法行为 | 原处罚幅度 | 2019年修订后幅度 |
|---------|----------|----------------|
| 欺诈发行 | 募集资金1%-5% | 募集资金**10%-100%** |
| 虚假陈述（发行人） | 30-60万元 | 50-**1000万元** |
| 内幕交易 | 违法所得1-5倍 | 违法所得**1-10倍** |
| 操纵市场 | 违法所得1-5倍 | 违法所得**1-10倍** |

### 7.2 市场禁入与职业禁止

- 证券市场禁入（证监会）：3/5/10年或终身
- 银行业/保险业高管任职资格取消（金融监管总局）
- 失信被执行人（法院）：全面的社会经济限制

## 注意事项

- 金融监管规则变动极快——金融监管总局和证监会每年发布大量部门规章和规范性文件。出具意见前须北大法宝检索确认规则为最新版本。
- "无证经营"是金融领域最严重的红线——不仅行政处罚重，还可能构成非法经营罪（《刑法》第225条），尤其涉及面向公众筹集资金时。
- 2023年《私募投资基金监督管理条例》将私募基金纳入行政法规监管，是私募基金领域的重大立法变化——在此之前私募基金主要以自律规则（基金业协会）规制。
- 反洗钱的"受益所有人"穿透识别义务在实践中执行日趋严格——2022年起央行加大了反洗钱处罚力度，单笔罚款可达数千万元。
- 金融消费者/投资者保护在2023年金融监管体制改革中升级——金融监管总局新设消保部门，统一行使原分散在人行和证监会的消保职能。
