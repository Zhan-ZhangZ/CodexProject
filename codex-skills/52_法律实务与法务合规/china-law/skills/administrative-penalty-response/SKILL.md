# /administrative-penalty-response

1. 读取 `~/.claude/plugins/config/claude-for-legal/china-law/CLAUDE.md` → 管辖与法律框架。
2. 确定行政处罚的阶段和适用法律。
3. 通过北大法宝 MCP 检索相关实体法和程序法 `[PKULaw]`。
4. 评估执法机关的处罚合法性、合规性和可能幅度。
5. 出具附应对策略和时限标注的应对指南。

---

## Purpose

中国行政处罚体系在2021年《行政处罚法》修订后经历了大幅重构——听证权扩大、"首违不罚"制度确立、处罚裁量基准的规范化等。2024年《行政复议法》修订进一步保障了相对人权利。但从企业实践来看，许多被处罚人在收到处罚决定书之前不知道有听证权、不知道可以申请不予处罚、不知道如何有效行使陈述申辩权。本技能提供行政处罚全流程的应对框架。

## 适用场景

| 阶段 | 场景 |
|------|------|
| 调查/现场检查阶段 | 执法人员现场检查、询问、调取资料、查封扣押 |
| 事先告知/听证阶段 | 收到行政处罚事先告知书（拟处罚），需要决定是否陈述申辩/申请听证 |
| 处罚决定阶段 | 收到行政处罚决定书，需要审查合法性并决定是否复议/诉讼 |
| 执行阶段 | 面临罚款缴纳、改正要求、停产停业等处罚执行 |
| 信用修复阶段 | 处罚公示后申请信用修复 |

## Step 2 — 行政处罚合法性审查框架

《行政处罚法》(2021修订) `[PKULaw]`：

### 2.1 主体合法性

| 审查项 | 依据 | 问题示例 |
|-------|------|---------|
| 处罚机关是否有法定职权 | 各单行法律 | 税务处罚是否由税务机关作出（非市监局） |
| 是否超越管辖范围 | 管辖规则 | 跨行政区域的处罚管辖是否合规 |
| 执法人员是否具有执法资格 | 《行政处罚法》第42条 | 协管人员/临时工不得单独执法 |
| 执法人员人数是否≥2人 | 《行政处罚法》第42条 | 单人执法不合法 |

### 2.2 程序合法性

```
行政处罚法定程序（一般程序）

    立案
      │
      ▼
    调查取证（≥2名执法人员，出示执法证件）
      │
      ▼
    案件调查终结报告
      │
      ▼
    ┌─ 拟作出处罚决定前：必须告知当事人
    │   ├── 告知拟处罚的事实、理由、依据
    │   ├── 告知当事人依法享有的陈述、申辩权
    │   └── 告知符合听证条件的听证权
    │
    ▼
    听取陈述和申辩（≥3日时限）
      │
      ▼
    符合听证条件且当事人申请 → 组织听证
      │
      ▼
    法制审核（重大处罚决定必经）
      │
      ▼
    负责人集体讨论决定（情节复杂/重大违法行为）
      │
      ▼
    作出行政处罚决定书 → 送达当事人
```

### 2.3 程序违法后果

| 程序违法 | 法律后果 |
|---------|---------|
| 未告知处罚事实、理由、依据 | 处罚决定**不成立**（《行政处罚法》第62条） |
| 未告知陈述申辩权/听证权 | 处罚决定**不成立**（《行政处罚法》第62条） |
| 拒绝听取陈述和申辩 | 违反法定程序 → 可撤销（《行政处罚法》第62条） |
| 应当进行法制审核而未审核 | 程序违法 → 可撤销 |
| 未送达处罚决定书 | 处罚决定对当事人不发生效力 |

### 2.4 实体合法性

| 审查项 | 依据 |
|-------|------|
| 事实认定是否清楚、证据是否充分 | 《行政处罚法》第40条 |
| 法律适用是否正确（法条、款项是否准确） | 《行政处罚法》第5条 |
| 处罚是否超过法定幅度 | 各单行法律的处罚幅度条款 |
| 是否在法定追责时效内 | 《行政处罚法》第36条 |

## Step 3 — 关键权利行使

### 3.1 陈述和申辩权

《行政处罚法》第44-45条 `[PKULaw]`：

| 要点 | 内容 |
|------|------|
| 行使时机 | 收到行政处罚事先告知书后，在告知的期限内（通常≥3日） |
| 行使方式 | 书面陈述申辩意见（建议书面，口头也可但需记录在案） |
| 不得加重 | 行政机关不得因当事人陈述、申辩而给予更重的处罚 |
| 效果 | 必须被听取；对合理意见应当采纳 |

**陈述申辩策略：**
1. **事实层面：** 指出事实认定错误或不完整
2. **证据层面：** 提供新证据（未在调查阶段提交的证明无违法或减轻情节的证据）
3. **法律层面：** 指出法律适用错误（如应适用较轻的法条）
4. **情节层面：** 主张从轻/减轻/不予处罚的法定情节
5. **比例原则：** 处罚明显不当的（畸轻畸重）→ "过罚相当"

### 3.2 听证权

《行政处罚法》第63-65条 `[PKULaw]`：

**听证适用情形（《行政处罚法》第63条）：**

| 处罚类型 | 是否可申请听证 |
|---------|-------------|
| 较大数额罚款 | ✅ 必须告知听证权 |
| 没收较大数额违法所得/较大价值非法财物 | ✅ 必须告知听证权 |
| 降低资质等级/吊销许可证照 | ✅ 必须告知听证权 |
| 责令停产停业/责令关闭/限制从业 | ✅ 必须告知听证权 |
| 其他较重的行政处罚 | ✅ 必须告知听证权 |
| 行政拘留 | ✅ 必须告知听证权 |
| 较小数额罚款 | ❌ 无听证权（但仍有陈述申辩权） |

**听证程序关键信息：**

| 程序节点 | 时限 |
|---------|------|
| 当事人申请听证 | 收到告知书后**5日**内 |
| 行政机关通知听证 | 举行听证的**7日**前 |
| 听证主持人 | 非本案调查人员（内部职能分离） |
| 公开原则 | 除涉及国家秘密/商业秘密/个人隐私外，公开举行 |
| 听证笔录 | 听证结束后，当事人核对无误后签字或盖章 |

**⚠️ 听证的意义：** 听证是行政处罚程序中最正式的当事人参与环节——可以当面对执法机关的证据、事实和法律依据进行全面质疑。虽然听证主持人来自同一行政机关（内部职能分离而非第三方独立主持），但对于复杂案件、大额罚款和吊销许可类处罚，听证是复议和诉讼前的最后防御阵线。

### 3.3 从轻/减轻/不予处罚的法定情节

《行政处罚法》第32-33条 `[PKULaw]`：

| 情节 | 效果 | 法律依据 |
|------|------|---------|
| 主动消除或减轻违法行为危害后果 | 从轻或减轻 | 第32条第1项 |
| 受他人胁迫或诱骗实施违法行为 | 从轻或减轻 | 第32条第2项 |
| 主动供述行政机关尚未掌握的违法行为 | 从轻或减轻 | 第32条第3项 |
| 配合行政机关查处违法行为有立功表现 | 从轻或减轻 | 第32条第4项 |
| 违法行为轻微并及时改正，没有造成危害后果 | **不予处罚** | 第33条第1款 |
| 初次违法且危害后果轻微并及时改正 | **可以不予处罚**（"首违不罚"） | 第33条第1款 |
| 有证据足以证明没有主观过错 | **不予处罚**（法律另有规定除外） | 第33条第2款 |

## Step 4 — 处罚后的救济路径

### 4.1 救济路径选择

```
收到行政处罚决定书

    ├── 行政复议 ──→ 上一级主管部门 / 本级人民政府
    │       │          （60日内申请）
    │       ├── 复议维持 ──→ 行政诉讼（15日内）
    │       └── 复议撤销/变更 ──→ 结束
    │
    ├── 行政诉讼 ──→ 有管辖权的人民法院
    │       │          （6个月内）
    │       └── 二审（15日内上诉）
    │
    └── 行政复议（前置？）
            │
            ├── 纳税争议：必须先复议 → 再诉讼
            └── 其他大多数处罚：复议或诉讼选择
```

### 4.2 行政复议

《行政复议法》(2023修订) `[PKULaw]`：

| 要点 | 内容 |
|------|------|
| 申请期限 | 知道该行政行为之日起 **60日**内（不可抗力除外） |
| 申请方式 | 书面或口头 |
| 审理方式 | 书面审查为主，可组织听证 |
| 审理期限 | 60日 + 可延长30日（复杂案件） |
| 停止执行 | 原则上不停止执行，但可申请停止（《行政复议法》第42条） |

### 4.3 行政诉讼

| 要点 | 内容 |
|------|------|
| 起诉期限 | 复议后：决定书送达之日起**15日**内；直接起诉：知道行政行为之日起**6个月**内 |
| 管辖法院 | 最初作出行政行为的机关所在地或复议机关所在地 |
| 举证责任 | 被告（行政机关）对行政行为的合法性承担举证责任 |
| 诉讼期间执行 | 原则上不停止执行 |

### 4.4 行政复议 vs. 行政诉讼 — 策略选择

| 因素 | 行政复议 | 行政诉讼 |
|------|---------|---------|
| 时间 | 60日+30日（快） | 6个月（一审）+2个月（二审） |
| 专业度 | 行业主管部门复议更有行业专业度 | 法官行政法专业但非行业专家 |
| 独立度 | 同一系统的上级机关（独立性有限） | 法院审判独立 |
| 调解可能性 | 复议中可调解（部分案件） | 行政案件原则上不适用调解（但有例外） |
| 管辖 | 无级别限制 | 中级法院管辖对县级以上政府的诉讼 |
| 成功率 | 较低（维持率高） | 更低（行政机关胜诉率高） |

## Step 5 — 处罚执行与信用修复

### 5.1 罚款缴纳与暂缓

| 情形 | 内容 |
|------|------|
| 缴纳期限 | 处罚决定书确定的15日（通常） |
| 逾期后果 | 每日按罚款数额的**3%**加处罚款（《行政处罚法》第72条） |
| 暂缓/分期缴纳 | 确有经济困难的，可申请暂缓或分期缴纳（《行政处罚法》第66条） |
| 强制执行 | 到期不缴纳，行政机关可申请法院强制执行 |

### 5.2 严重违法失信名单

《市场监督管理严重违法失信名单管理办法》`[PKULaw]`：

| 列入条件 | 公示期限 | 影响 |
|---------|---------|------|
| 受到较重行政处罚（情节严重/从重处罚等） | **3年** | 政府采购、工程招投标、银行贷款、上市融资等全方位受限 |

### 5.3 信用修复

| 条件 | 修复方式 |
|------|---------|
| 公示期满一定时间（≥6个月） | 申请提前停止公示 |
| 已履行法定义务、消除不良影响 | 申请信用修复 |
| 未再发生同类违法行为 | 缩短公示期限 |
| 作出信用承诺 | 可作为修复参考条件 |

## 注意事项

- 法定申请期限错过即丧失权利——60日复议申请期限和6个月诉讼期限是刚性的。把关键日期标注为最重要的事项。
- "首违不罚"是裁量性不予处罚，不是法定不予处罚——执法机关可以选择适用或不适用。在陈述申辩中充分论证为何符合"危害后果轻微"和"及时改正"。
- 听证权告知是处罚机关的法定职责——如果应告知而未告知听证权，处罚决定不成立。这是最常见的程序违法事项。
- 行政拘留由公安机关决定——其余大多数行政处罚不涉及人身自由限制。
- 对处罚决定书中的金额、期限、缴款账户要逐一核对——行政程序中也可能存在抄写错误或计算错误。
