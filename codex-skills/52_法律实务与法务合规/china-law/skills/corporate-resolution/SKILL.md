# /corporate-resolution

起草或审查公司决议文件，确保程序、内容、形式合法性。

## 指令

### 1. 事务上下文
- 公司类型？（有限责任公司/股份有限公司/外商投资企业）
- 决议类型？（股东会/董事会/监事会）
- 决议事项？（增资/减资/修改章程/选举董事/对外担保/合并分立/解散）

### 2. 核心法律框架（《公司法》(2023修订) `[PKULaw]`）

#### 2.1 股东会决议

**股东会职权（第59条）：**
选举和更换董事、监事；审议批准董事会/监事会报告；审议批准财务预/决算、利润分配和弥补亏损方案；增减注册资本；发行公司债券；合并、分立、解散、清算；修改公司章程

**表决比例:**
- 普通决议 → 过半表决权通过（章程可提高）
- 修改章程、增资或减资、合并、分立、解散或变更公司形式 → 三分之二以上表决权（第66条）

**会议通知（第64条）：**
定期会议 → 章程规定；临时会议 → 代表十分之一以上表决权的股东/三分之一以上董事/监事会提议 → 应召开

**不召开会议的直接决定（第65条）：**
全体股东以书面形式一致同意 → 可不召开股东会会议，直接作出决定，全体股东签名盖章

#### 2.2 董事会决议

**董事会组成（第68条）：**
- 有限责任公司董事会 3-13人（或1名执行董事，小型公司）
- 股份有限公司董事会 5-19人

**表决（第73条）：**
- 一人一票
- 出席会议的董事过半数通过
- 与决议事项有关联关系的董事 → 回避（不参加表决，不计入出席人数）

#### 2.3 公司决议效力

**决议不成立（公司法司法解释四第1条 `[PKULaw]`）：**
- 未召开会议（全体书面同意的除外）
- 会议未对决议事项进行表决
- 出席会议的人数或表决权未达到法定或章程要求
- 同意决议的表决权未达到法定或章程要求

**决议无效：**
决议内容违反法律、行政法规

**决议可撤销：**
- 召集程序、表决方式违反法律、行政法规或公司章程
- 决议内容违反公司章程
- 撤销权：自决议作出之日起60日内向法院提出（第26条）

#### 2.4 股东权利

- 知情权（第57条）：查阅、复制公司章程、股东名册、股东会会议记录、董事会会议决议、监事会会议决议、财务会计报告
- 利润分配请求权（第210条）
- 股东代表诉讼（第189条）

### 3. 决议文件清单

- 会议通知
- 会议签到表
- 表决票（如需书面表决）
- 会议记录
- 决议文件
- 授权委托书（如代理人出席）

### 4. 输出

决议程序合规性审查 + 决议草案/审查意见 + 表决权计算 + 程序注意事项 + 决策树。
