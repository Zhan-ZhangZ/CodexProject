# /diligence-report

1. 读取 `~/.claude/plugins/config/claude-for-legal/china-law/CLAUDE.md` → 管辖与法律框架、审查偏好。
2. 确定尽调范围和重点领域。
3. 逐项审查目标公司的法律状况。
4. 出具附风险分级和整改建议的尽调报告。

---

## Purpose

法律尽职调查是并购、投资和上市项目的核心环节——它回答"我们在买什么？有什么坑？"本技能提供中国法律框架下的系统性尽调方法和报告格式，将发现分级并辅以整改建议。

## 尽调类型

| 尽调类型 | 目的 | 特点 |
|---------|------|------|
| 并购尽调 | 买方全面了解目标公司的法律状况 | 范围最广，买方导向 |
| 投资尽调（少数股权投资） | 评估投资标的的法律风险 | 侧重关键风险，非全面审计 |
| 上市尽调 | 确保满足上市法律要求 | 规范性最强，监管导向 |
| 融资担保尽调 | 了解担保品的法律状况 | 侧重资产/股权权属和负担 |
| 合作项目尽调 | 了解合作方的法律资质和风险 | 侧重资质、履约能力和涉诉情况 |

---

## Step 1 — 确定尽调范围

一次性询问：

> 请提供：
>
> - **尽调类型：** 并购/投资/上市/融资担保/合作项目？
> - **目标公司**名称、统一社会信用代码、注册地址。
> - **尽调资料**（请上传或列出可提供的文件清单）：
>   - 营业执照 + 公司章程
>   - 工商登记档案（含历次变更）
>   - 股权结构图 + 股东名册
>   - 重大合同清单 + 合同文本
>   - 知识产权清单（商标/专利/著作权/域名）
>   - 不动产/重要资产清单及权属证书
>   - 员工名册 + 劳动合同模板 + 社保缴纳记录
>   - 诉讼/仲裁/行政处罚清单
>   - 资质证照/行政许可（如涉及特殊行业）
>   - 财务报表/审计报告（如有）
>   - 税务登记 + 纳税记录
>   - 环保批文/排污许可证（如涉及生产型/重污染企业）
> - **重点关切：** 有哪些特别关注的领域或已知风险？

## Step 2 — 逐项尽调

### 尽调事项清单

#### 一、主体资格与存续

| 审查项 | 审查内容 | 资料来源 |
|-------|---------|---------|
| 营业执照 | 经营范围、注册资本、成立日期、有效期 | 营业执照 + 国家企业信用信息公示系统 |
| 公司章程 | 章程内容是否与工商登记一致；是否有超级多数决/一票否决权等特殊安排；是否违反公司法强制性规定 | 工商备案章程 + 内部章程 |
| 工商变更记录 | 历史变更是否合规；是否存在未登记的实际变更（如股东变更未登记） | 工商登记档案 |
| 存续状态 | 是否正常经营；是否存在被吊销或注销风险 | 国家企业信用信息公示系统 |

#### 二、股权结构与股东

| 审查项 | 审查内容 | 资料来源 |
|-------|---------|---------|
| 现有股权结构 | 股东名录、持股比例、出资方式、出资时间 | 工商登记 + 股东名册 + 出资证明 |
| 出资情况 | 股东是否已履行全部出资义务（认缴→实缴） | 验资报告/银行进账单/财务报表 |
| 股权代持 | 是否存在股权代持安排及代持协议 | 询问管理层 + 审查协议 |
| 股权质押/冻结 | 是否存在股权质押、冻结或其他权利限制 | 工商登记档案 + 国家企业信用信息公示系统 |
| 优先权/特殊权利 | 是否存在优先购买权、回购权、反稀释、一票否决权等（融资交易文件） | 股东协议 + 增资协议 |
| 历史股权转让 | 历次转让的合规性（国有产权转让是否挂牌、是否经审批） | 工商变更记录 + 转让协议 |

#### 三、资产与知识产权

| 审查项 | 审查内容 | 资料来源 |
|-------|---------|---------|
| 不动产 | 土地使用权证/不动产权证书；是否存在抵押/查封 | 不动产权证书 + 不动产登记查询 |
| 租赁场所 | 租赁合同的关键条款（租金、租期、续租权、转租限制） | 租赁合同 |
| 主要设备/动产 | 所有权/融资租赁/抵押情况 | 购买合同 + 融资租赁协议 + 抵押登记查询 |
| 商标/专利/著作权 | 注册/登记状态、有效期、权利人一致、许可/质押情况 | 商标局/专利局/版权局查询 + 许可合同 |
| 域名/软件 | 注册人是否与公司一致、许可情况 | WHOIS查询 + 许可合同 |

#### 四、重大合同

| 审查项 | 审查内容 | 资料来源 |
|-------|---------|---------|
| 客户/销售合同 | 主要客户、收入依赖度、合同期限和终止条款 | 合同文件 |
| 供应商/采购合同 | 主要供应商、最低采购承诺、排他性条款 | 合同文件 |
| 融资/担保合同 | 贷款余额、担保范围、交叉违约条款 | 合同文件 + 征信报告 |
| 关联交易 | 是否存在关联交易及定价公允性 | 合同 + 审计报告 |
| 合同特殊条款 | 控制权变更条款、最惠国待遇、重大不利变更条款等 | 合同文件 |

#### 五、劳动用工

| 审查项 | 审查内容 | 资料来源 |
|-------|---------|---------|
| 劳动合同 | 签订率、合同期限、试用期合规性 | 劳动合同模板 + 花名册 |
| 社会保险/公积金 | 是否全员缴纳、缴费基数是否合规 | 社保局/公积金中心缴费记录 |
| 特殊工时制 | 综合工时制/不定时工时制是否获得审批 | 审批文件 |
| 劳务派遣/外包 | 派遣比例是否超过10%、外包是否为"假外包真派遣" | 派遣协议 + 外包合同 |
| 竞业限制/保密 | 核心员工是否签有竞业限制和保密协议 | 协议文本 |
| 劳动争议历史 | 是否有未决的劳动仲裁/诉讼 | 仲裁委/法院查询 + 管理层询问 |

#### 六、诉讼、仲裁与行政处罚

| 审查项 | 审查内容 | 资料来源 |
|-------|---------|---------|
| 未决诉讼/仲裁 | 案件清单、案件标的、案件进展 | 法院/仲裁委查询 + 管理层询问 |
| 行政处罚 | 是否存在行政处罚记录 | 国家企业信用信息公示系统 + 相关主管部门 |
| 被执行/失信 | 是否被列为被执行人或失信被执行人 | 中国执行信息公开网 |
| 潜在争议 | 是否有可能产生重大争议的事项（如产品质量问题、客户投诉、合同对方已发函等） | 管理层询问 |

#### 七、税务

| 审查项 | 审查内容 | 资料来源 |
|-------|---------|---------|
| 税务登记 | 税种核定、纳税人资格 | 税务登记证 |
| 纳税记录 | 近N年主要税种（增值税、企业所得税等）的申报和缴纳情况 | 纳税申报表 + 纳税凭证 |
| 税务处罚 | 是否存在税务处罚记录 | 税务机构查询 + 管理层询问 |
| 税收优惠 | 高新技术企业/研发加计扣除等是否存在被撤销风险 | 认定文件 + 合规审查 |
| 转让定价 | 关联交易是否存在转让定价风险 | 关联交易文件 + 转让定价文档 |

#### 八、行业资质与特许经营

| 审查项 | 审查内容 | 资料来源 |
|-------|---------|---------|
| 行业许可/牌照 | 所有必要的行业许可/牌照是否齐全、是否在有效期内 | 许可证照原件 |
| 许可条件 | 是否持续满足许可条件（人员、资本、技术设备等） | 许可条件清单 + 实际情况核对 |
| 外商投资限制 | 所从事行业是否涉及负面清单领域 | 外商投资准入特别管理措施（负面清单）（最新版） |

#### 九、环境合规（制造型/重污染企业必审）

| 审查项 | 审查内容 | 资料来源 |
|-------|---------|---------|
| 环评批复/竣工验收 | 建设项目是否取得环评批复和竣工环保验收 | 批复文件 |
| 排污许可证 | 是否持有有效排污许可证 | 排污许可证 |
| 环保处罚 | 是否存在环保处罚记录 | 环保部门查询 |
| 危废处置 | 危险废物管理是否合规 | 危废台账 + 处置合同 |

#### 十、数据合规

| 审查项 | 审查内容 | 资料来源 |
|-------|---------|---------|
| 个人信息处理 | 处理的个人信息类型和规模、是否有隐私政策 | 隐私政策 + 产品分析 |
| 数据出境 | 是否存在数据出境及是否完成安全评估/签署标准合同 | 数据处理记录 |
| APP/小程序合规 | 是否存在违规收集个人信息问题 | APP隐私合规检测 |
| 网络安全等级保护 | 是否完成等保定级/备案/测评 | 等保文件 |

## Step 3 — 风险分级

采用与 CLAUDE.md 统一的四级风险标准：

| 级别 | 说明 | 处理要求 |
|------|------|---------|
| 🔴 阻断性 | 发现可能导致交易中止或须重新定价的重大法律障碍 | 必须披露给决策层；交易前必须解决或接受 |
| 🟠 高 | 重大合规差距/法律风险/潜在重大赔偿责任 | 需在交割前或交割后限期内整改 |
| 🟡 中 | 需关注的合规事项或合同缺陷 | 在交易文件中纳入陈述与保证，交割后限期整改 |
| 🟢 低 | 轻微瑕疵/可接受的常规问题 | 可在交割后正常经营过程中解决 |

## 注意事项

- 本报告基于目标公司提供的文件出具——报告须明确说明文件来源和未提供的文件，并在关键发现的表述中注明"根据[文件名称]所述"或"根据管理层口头说明（未经文件核实）"。
- 公开渠道查询的信息（国家企业信用信息公示系统、中国执行信息公开网、商标局/专利局数据库等）与目标公司提供的信息之间存在差异的，须明确指出。
- 尽调报告是为委托人内部决策使用——除非另有约定，不对第三方承担责任。
- 涉及国有资产、上市公司或外资并购的尽调，须遵守特别规定（如涉及国有资产的需评估、挂牌；涉及上市公司的需信息披露；涉及外资安全审查的需申报）。
