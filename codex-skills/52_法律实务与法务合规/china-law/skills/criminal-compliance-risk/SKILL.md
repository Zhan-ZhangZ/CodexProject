# /criminal-compliance-risk

1. 读取 `~/.claude/plugins/config/claude-for-legal/china-law/CLAUDE.md` → 管辖与法律框架。
2. 识别经营活动中的刑事风险点。
3. 通过北大法宝 MCP 检索相关罪名和司法解释 `[PKULaw]`。
4. 评估单位犯罪和个人刑事责任的双重风险。
5. 出具附风险分级、合规整改和辩护策略的评估报告。

---

## Purpose

中国刑法对经济犯罪的惩罚严厉且覆盖面广——从职务侵占到虚开发票，从商业贿赂到污染环境，企业管理人员和法人本身都面临刑事追诉风险。2024年《刑法修正案(十二)》进一步扩大了部分经济犯罪的打击范围。与此同时，最高人民检察院推行的涉案企业合规改革（"合规不起诉"）为建立有效合规体系的企业提供了出罪路径。本技能提供中国刑法框架下的企业刑事风险系统评估。

> 请提供：

## Step 2 — 单位犯罪基本框架

### 2.1 单位犯罪构成

《刑法》第30-31条 `[PKULaw]`：

| 要素 | 内容 |
|------|------|
| 主体 | 公司、企业、事业单位、机关、团体 |
| 主观 | 为了单位的利益（非个人利益） |
| 客观 | 以单位名义实施（集体决定或负责人决定） |
| 法定 | 法律规定为单位犯罪的才负刑事责任（罪刑法定特定原则） |

**⚠️ 关键实践点：** 中国刑法中并非所有犯罪都规定单位犯罪。例如，贷款诈骗罪没有单位犯罪规定——单位实施的贷款诈骗，追究**直接责任人员**个人的贷款诈骗罪刑事责任，而不能追究单位本身（但可另以骗取贷款罪或其他罪名处理）。

### 2.2 双罚制

《刑法》第31条 — 对单位犯罪的处罚原则：

```
单位犯罪 — 双罚制

    ├── 对单位：罚金
    │
    └── 对个人：
        ├── 直接负责的主管人员（决策者/指挥者）
        └── 其他直接责任人员（具体实施者）
        → 判处该罪的自然人刑罚（有期徒刑、拘役、罚金等）
```

### 2.3 "直接负责的主管人员"的认定

最高人民法院《全国法院审理金融犯罪案件工作座谈会纪要》`[PKULaw]`：

| 认定标准 | 说明 |
|---------|------|
| 决策/指挥作用 | 在单位犯罪中起决定、批准、授意、纵容、指挥作用 |
| 不要求亲自实施 | 即使未亲自实施具体犯罪行为，只要在决策层面发挥作用 |
| 不知情抗辩 | 如能证明确实不知情且无纵容/失职，可能不被认定（但需具体证明） |
| 职级不是免责理由 | 高级管理人员不能仅凭"不了解具体业务"免责 |

## Step 3 — 常见企业经济犯罪清单

### 3.1 职务侵占罪与挪用资金罪

| 罪名 | 法条 | 构成要件 | 量刑 |
|------|------|---------|------|
| 职务侵占罪 | 《刑法》第271条 `[PKULaw]` | 利用职务便利，将本单位财物非法占为己有 | 数额较大（6万以上）：3年以下<br>数额巨大（100万以上）：3-10年<br>数额特别巨大：10年以上至无期 |
| 挪用资金罪 | 《刑法》第272条 `[PKULaw]` | 利用职务便利，挪用本单位资金归个人使用或借贷给他人 | 超3个月未还/营利活动/非法活动：3年以下<br>数额巨大（400万以上）：3-7年<br>数额特别巨大：7年以上 |

### 3.2 商业贿赂

| 罪名 | 法条 | 关键要素 |
|------|------|---------|
| 非国家工作人员受贿罪 | 《刑法》第163条 `[PKULaw]` | 利用职务便利，索取或非法收受他人财物，为他人谋取利益 |
| 对非国家工作人员行贿罪 | 《刑法》第164条 `[PKULaw]` | 为谋取不正当利益，给予公司/企业人员财物 |
| 行贿罪（国家工作人员） | 《刑法》第389-390条 `[PKULaw]` | 为谋取不正当利益，给予国家工作人员财物 |
| 单位行贿罪 | 《刑法》第393条 `[PKULaw]` | 单位为谋取不正当利益而行贿 |

**⚠️ 《刑法修正案(十二)》(2024) 主要变化：**
- 扩大了部分行贿罪的打击范围
- 增加了对多次行贿、向多人行贿的从重处罚情节
- 强化了单位行贿罪中"直接负责的主管人员"的刑事责任

### 3.3 侵犯商业秘密

《刑法》第219条 `[PKULaw]`：

| 行为方式 | 量刑 |
|---------|------|
| 以盗窃/利诱/胁迫等不正当手段获取 | 情节严重：3年以下<br>情节特别严重：3-10年 |
| 披露/使用/允许他人使用以前项手段获取的商业秘密 | 同上 |
| 违反保密义务披露/使用商业秘密 | 同��� |

**⚠️ 2020年修正后的变化：** "给商业秘密的权利人造成重大损失"改为"情节严重"——降低了入罪门槛，不仅以金钱损失为标准。

### 3.4 金融犯罪

| 罪名 | 法条 | 适用场景 |
|------|------|---------|
| 非法吸收公众存款罪 | 《刑法》第176条 `[PKULaw]` | 未经批准向社会公开吸收资金（P2P爆雷/理财产品违规等） |
| 集资诈骗罪 | 《刑法》第192条 `[PKULaw]` | 以非法占有为目的使用诈骗方法非法集资（更重——可至无期徒刑） |
| 骗取贷款、票据承兑、金融票证罪 | 《刑法》第175条之一 `[PKULaw]` | 以欺骗手段取得银行贷款等（不需证明非法占有目的） |
| 洗钱罪 | 《刑法》第191条 `[PKULaw]` | 为特定上游犯罪的违法所得掩饰、隐瞒来源和性质 |
| 内幕交易、泄露内幕信息罪 | 《刑法》第180条 `[PKULaw]` | 涉及上市公司证券交易 |

### 3.5 税务犯罪（详见 /tax-law-overview）

- 逃税罪（《刑法》第201条）— 初犯补税免刑条款
- 虚开增值税专用发票罪（《刑法》第205条）— 最高无期徒刑
- 骗取出口退税罪（《刑法》第204条）

### 3.6 生产经营类犯罪

| 罪名 | 法条 | 说明 |
|------|------|------|

## Step 4 — 合规出罪与量刑从宽

### 4.1 涉案企业合规改革（"合规不起诉"）

最高人民检察院推行的涉案企业合规改革（2020年试点，2023年全国推广）：

```
合规不起诉路径

        企业涉嫌犯罪
             │
             ▼
        检察院审查是否符合条件：
        ├── 企业有正常经营
        ├── 企业和个人认罪认罚
        ├── 犯罪情节非特别严重
        ├── 企业愿意开展合规整改
             │
        ┌────┴────┐
        ▼         ▼
    符合条件    不符合条件
        │          │
        ▼          ▼
    检察院与    正常起诉程序
    企业商定
    合规整改
    方案
        │
        ▼
    企业实施合规整改
    （6个月至1年）
        │
        ▼
    第三方监督评估/
    检察院评估验收
        │
    ┌───┴───┐
    ▼       ▼
  合格     不合格
    │       │
    ▼       ▼
  不起诉   起诉
  （酌定
  不起诉）
```

**适用条件（最高检规定）：**
- 案件事实清楚，证据确实充分
- 企业认罪认罚
- 企业有正常生产经营
- 企业自愿开展合规整改
- 犯罪时已建立合规体系但存在漏洞，或愿意建立合规体系
- **不适用情形：** 个人为进行违法犯罪活动而设立企业、企业设立后以实施犯罪为主要活动、企业人员盗用单位名义实施犯罪

### 4.2 合规整改重点

| 整改要素 | 内容 |
|---------|------|
| 合规管理机构 | 设立合规委员会/任命首席合规官 |
| 合规制度体系 | 制定合规政策、行为准则、专项合规指引 |
| 合规风险识别与评估 | 定期开展合规风险评估 |
| 合规培训 | 全员培训+关键岗位专项培训 |
| 举报与调查机制 | 设立违规举报渠道+内部调查机制 |
| 合规考核与问责 | 将合规纳入绩效考核+违规问责 |
| 持续改进 | 定期评估合规体系有效性并改进 |

### 4.3 合规对量刑的影响

即使不适用不起诉，有效的合规体系也可在量刑时产生积极影响：
- 体现企业的事前注意义务——可能被认定为"犯罪情节较轻"
- 及时合规整改体现认罪悔罪态度——可从宽处理
- 《刑法》第67条（自首）、第68条（立功）配合使用

## Step 5 — 内部调查向刑事立案转化的风险管理

### 5.1 发现违规后的决策框架

```
发现可能涉及刑事的违规行为

        确认事实（不猜测）
             │
             ▼
    ┌─ 是否涉及刑事犯罪？──→ 否 ──→ 内部纪律处分
    │        │ 是
    │        ▼
    │   ┌─ 报案 ──→ 公安机关立案 ──→ 脱离企业控制
    │   │
    │   ├─ 不报案、内部处理 ──→ ⚠️ 风险：可能构成包庇罪
    │   │                          （如果后续被发现）
    │   │
    │   ├─ 合规不起诉路径 ──→ 与检察院沟通 ──→ 合规整改
    │   │
    │   └─ 先行全面内部调查 → 律师主导 → 评估定罪风险 →
    │         再决定是否主动报案
    │
    ▼
   无论选择哪个路径，立即保全证据（邮件/文件/系统日志等）
```

### 5.2 ⚠️ 包庇风险

《刑法》第310条（窝藏、包庇罪）`[PKULaw]` — 明知是犯罪的人而为其提供隐藏处所、财物，帮助其逃匿或者作假证明包庇的，处3年以下有期徒刑、拘役或者管制；情节严重的，处3-10年有期徒刑。

**对企业的影响** — 如果企业管理人员明知员工实施了职务侵占/商业贿赂等犯罪行为，而不向公安机关报告还帮助其销毁证据或在内部低调处理，可能构成包庇罪。

## 注意事项

- 本评估不是为了定罪——而是帮助企业识别刑事风险并确保在风险变为现实之前采取防范措施。
- 单位犯罪的"直接负责的主管人员"范围宽于企业通常设想的"高管"——包括对犯罪行为知情并纵容的中层管理者。
- 合规不起诉改革在实践中仍然存在地区差异——某些省份的检察院推行力度更大，有些则相对保守。
- 一旦确认刑事违规已发生，律师的建议尤为重要——《律师法》第38条的保密义务为律师主导的调查提供了比企业自行调查更强的保护。
