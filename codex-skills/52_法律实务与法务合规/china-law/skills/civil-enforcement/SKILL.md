# /civil-enforcement

1. 读取 `~/.claude/plugins/config/claude-for-legal/china-law/CLAUDE.md` → 管辖与法律框架。
2. 检查执行依据和执行期限。
3. 通过北大法宝 MCP 检索相关司法解释和规定 `[PKULaw]`。
4. 制定执行策略：财产调查→执行措施→执行救济。

---

## Purpose

拿到胜诉判决只是第一步。如果对方不主动履行，需要启动强制执行程序。本技能覆盖从申请执行到执行终结的全流程——财产查控、失信惩戒、执行异议、执行和解、参与分配、"终本"案件的恢复执行。

## 适用场景

- 拿到生效判决/调解书，对方不履行
- 仲裁裁决需要法院强制执行
- 公证债权文书需要执行
- 执行过程中遇到执行异议
- 多个债权人对同一被执行人申请执行
- 被执行人无财产可供执行（"执行难"）

## 法律框架

**核心法律:**
- 《中华人民共和国民事诉讼法》(2023修正) 第三编 执行程序(第235-274条) `[PKULaw]`
- 《最高人民法院关于适用〈中华人民共和国民事诉讼法〉的解释》(2022修正) 执行程序部分 `[PKULaw]`
- 《最高人民法院关于人民法院执行工作若干问题的规定(试行)》(2020修正) `[PKULaw]`
- 《最高人民法院关于民事执行中变更、追加当事人若干问题的规定》(2020修正) `[PKULaw]`
- 《最高人民法院关于人民法院办理执行异议和复议案件若干问题的规定》(2020修正) `[PKULaw]`
- 《最高人民法院关于人民法院网络司法拍卖若干问题的规定》`[PKULaw]`
- 《最高人民法院关于限制被执行人高消费及有关消费的若干规定》(2015修正) `[PKULaw]`
- 《最高人民法院关于公布失信被执行人名单信息的若干规定》(2017修正) `[PKULaw]`
- 《最高人民法院关于人民法院民事执行中查封、扣押、冻结财产的规定》(2020修正) `[PKULaw]`
- 《最高人民法院关于民事执行中财产调查若干问题的规定》(2020修正) `[PKULaw]`
- 《中华人民共和国刑法》第313条（拒不执行判决、裁定罪）`[PKULaw]`

## Step 1 — 执行前审查

### 1.1 执行依据检查

可申请执行的生效法律文书：
- 人民法院作出的民事判决书、裁定书、调解书、支付令
- 仲裁机构作出的仲裁裁决书、调解书
- 公证机关赋予强制执行效力的债权文书
- 人民法院作出的承认和执行外国法院判决、外国仲裁裁决的裁定

### 1.2 申请执行期限（《民事诉讼法》第246条 `[PKULaw]`）

- 申请执行的期间为**2年**
- 起算：法律文书规定履行期间的最后一日起；分期履行的，从每次履行期间的最后一日起
- 申请执行时效中止、中断适用诉讼时效的规定
- ⚠️ 超过申请执行期限 → 法院可能不予受理

### 1.3 执行管辖（《民事诉讼法》第235条 `[PKULaw]`）

- 第一审人民法院或与第一审人民法院同级的被执行人住所地/财产所在地法院
- 仲裁裁决、公证债权文书 → 被执行人住所地或被执行的财产所在地中级人民法院（或指定基层法院）

## Step 2 — 申请执行

### 申请执行书要素

```
申请执行人：[姓名/名称]，住所地：[地址]，联系方式：[电话]
被执行人：[姓名/名称]，住所地：[地址]，联系方式：[电话]

执行依据：[判决/裁定/调解书/仲裁裁决] — [法院/仲裁机构] [案号]，[生效日期]

申请事项：
1. 请求强制执行被执行人支付人民币 [金额] 元及迟延履行期间的债务利息
2. 请求强制执行被执行人履行 [行为义务描述]
3. 请求对被执行人采取限制消费措施/纳入失信被执行人名单

事实与理由：
[简述执行依据内容和被执行人未履行情况]

被执行人的财产状况：
线索1: [财产类型 — 银行存款/不动产/机动车/股权/知识产权/债权/其他]，[线索描述]，[来源]
线索2: ...

此致
[法院名称]

申请执行人：[签名/盖章]
日期：YYYY-MM-DD
```

### 申请执行材料清单

- [ ] 申请执行书
- [ ] 生效法律文书副本
- [ ] 申请执行人的身份证明（营业执照/身份证复印件）
- [ ] 授权委托书（如委托代理人）
- [ ] 被执行人财产线索（如有）
- [ ] 法律文书生效证明（部分法院要求）

## Step 3 — 财产调查与执行措施

### 3.1 法院主动调查

法院受理执行申请后：
- 向被执行人发出执行通知书和报告财产令
- 通过网络执行查控系统（"总对总"、"点对点"）查询被执行人的银行存款、不动产、机动车、股权、证券等财产
- 被执行人拒绝报告或虚假报告的 → 罚款、拘留；纳入失信名单

### 3.2 申请人提供的财产线索

| 财产类型 | 查询途径 | 执行措施 |
|---------|---------|---------|
| 银行存款 | 日常交易中获取的账户信息 | 冻结、扣划 |
| 不动产 | 不动产登记中心查询（需律师调查令或法院协助查询） | 查封、拍卖、变卖、强制管理 |
| 机动车 | 车管所查询 | 查封、扣押、拍卖 |
| 股权/投资 | 国家企业信用信息公示系统 | 冻结、拍卖 |
| 知识产权 | 商标局/专利局查询 | 冻结、拍卖 |
| 工资/劳务收入 | 已知工作单位 | 扣留、提取（保留生活必需费用） |
| 应收账款/到期债权 | 已知债务人 | 冻结、通知向申请执行人履行 |
| 公积金 | 公积金管理中心 | 可执行（各地规定略有差异） |
| 理财产品/保险 | 已知购买渠道 | 冻结、扣划 |

### 3.3 执行措施

**金钱债权执行：**
1. 冻结银行存款 → 扣划
2. 查封不动产/动产 → 拍卖/变卖/以物抵债
3. 扣留提取收入
4. 冻结到期债权

**行为执行：**
1. 交付特定物
2. 强制迁出房屋/退出土地
3. 替代履行（由他人代为完成，费用由被执行人承担）+ 迟延履行金

**网络司法拍卖：**
- 首次拍卖：保留价不低于评估价或市价的70%
- 流拍后再次拍卖：保留价降低不超过20%
- 两次流拍后：变卖或以物抵债

## Step 4 — 执行惩戒

### 4.1 限制消费（"限高令"）

依据：《最高人民法院关于限制被执行人高消费及有关消费的若干规定》`[PKULaw]`

限制内容：
- 乘坐飞机（不得乘坐飞机，包括经济舱）、列车软卧、高铁全部席位、轮船二等以上舱位
- 在星级以上宾馆/酒店/夜总会/高尔夫球场等场所高消费
- 购买不动产、新建/扩建/高档装修房屋
- 租赁高档写字楼/宾馆/公寓等办公场所
- 购买非经营必需车辆
- 旅游、度假
- 子女就读高收费私立学校
- 支付高额保费购买保险理财产品
- 其他非生活和工作必需的消费行为

### 4.2 失信被执行人名单（"老赖名单"）

依据：《最高人民法院关于公布失信被执行人名单信息的若干规定》`[PKULaw]`

纳入条件（第1条）：
1. 有履行能力而拒不履行生效法律文书确定义务的
2. 以伪造证据、暴力、威胁等方法妨碍、抗拒执行的
3. 以虚假诉讼、虚假仲裁或者以隐匿、转移财产等方法规避执行的
4. 违反财产报告制度的
5. 违反限制消费令的
6. 无正当理由拒不履行执行和解协议的

**法律后果：**
- 在政府采购、招标投标、行政审批、政府扶持、融资信贷、市场准入、资质认定等方面受到信用惩戒
- 在征信系统记录
- 通过中国执行信息公开网、报纸、广播、电视、网络等方式公布

### 4.3 罚款、拘留

对拒不履行、妨碍执行的行为，可对个人处10万元以下罚款、15日以下拘留；对单位处5万-100万元罚款。

### 4.4 拒不执行判决、裁定罪（《刑法》第313条 `[PKULaw]`）

情节严重 → 三年以下有期徒刑、拘役或罚金；情节特别严重 → 三年以上七年以下有期徒刑，并处罚金。

移送公安机关立案侦查的条件：
- 有能力执行而拒不执行
- 情节严重（转移财产、隐藏财产、虚假诉讼逃债等）

## Step 5 — 执行异议与执行救济

### 5.1 执行行为异议（《民事诉讼法》第232条 `[PKULaw]`）

当事人、利害关系人认为执行行为违反法律规定的 → 向执行法院提出书面异议
- 法院收到后15日内审查
- 裁定驳回或理由成立→撤销/改正
- 对裁定不服 → 10日内向上一级法院申请复议

### 5.2 案外人异议（《民事诉讼法》第234条 `[PKULaw]`）

案外人对执行标的主张所有权/其他足以排除强制执行的权益 → 书面异议
- 法院收到后15日内审查
- 理由不成立 → 裁定驳回
- 理由成立 → 裁定中止执行
- 对裁定不服 → 提起执行异议之诉（15日内）

### 5.3 执行异议之诉

- **案外人执行异议之诉：** 案外人对驳回异议裁定不服 → 以申请执行人为被告起诉
- **申请执行人执行异议之诉：** 法院支持案外人异议 → 申请执行人以案外人为被告起诉
- **执行分配方案异议之诉：** 对参与分配方案有异议

### 5.4 变更、追加被执行人

依据：《最高人民法院关于民事执行中变更、追加当事人若干问题的规定》`[PKULaw]`

常见情形：
| 情形 | 变更/追加对象 |
|------|-------------|
| 被执行人死亡 | 继承人（在遗产范围内） |
| 被执行人合并 | 存续/新设法人 |
| 被执行人分立 | 分立后的法人（承担连带责任） |
| 个人独资企业不能清偿 | 出资人 |
| 合伙企业不能清偿 | 普通合伙人 |
| 一人有限公司股东不能证明财产独立 | 股东 |
| 股东未履行出资义务即转让股权 | 原股东 |
| 公司未经清算即办理注销 | 股东/实际控制人 |

对变更追加裁定不服 → 10日内向上一级法院申请复议（部分情形可提起执行异议之诉）

## Step 6 — 参与分配（多债权人）

被执行人为公民/其他组织（非企业法人）→ 多个债权人可申请参与分配

**分配顺序（《民事诉讼法》第236条 `[PKULaw]`）：**
1. 执行费用（评估费、拍卖费等）
2. 优先权（抵押权、质权等担保物权；建设工程价款优先受偿权；船舶/航空器优先权）
3. 普通债权 → 按债权比例分配

被执行人为企业法人 → 通过破产程序解决，不适用参与分配（但在未进入破产程序时可参照参与分配处理，按查封先后顺序）

---

## Step 7 — 执行和解

双方可自愿协商达成执行和解协议（《民事诉讼法》第237条 `[PKULaw]`）：

- 和解协议提交法院，法院可中止执行
- 完全履行 → 执行终结
- 被执行人不履行和解协议 → 申请执行人可申请恢复执行原生效法律文书（但扣除已履行部分）
- ⚠️ 就和解协议另行起诉的主张一般不被支持（《执行和解司法解释》第9条 `[PKULaw]`）

### 和解协议要点

```
1. 明确履行金额和方式
2. 设定履行期限和节点
3. 担保条款（第三人提供担保 → 不履行时可直接执行担保财产）
4. 违约条款：任何一期未按时履行 → 恢复执行原法律文书
5. 切勿写入"申请执行人放弃对剩余债权的追索"等放弃实体权利的条款，除非确实有此意愿
```

## Step 8 — 执行终结与恢复执行

### "终本"案件 — 终结本次执行程序

条件（《关于严格规范终结本次执行程序的规定(试行)》 `[PKULaw]`）：
- 已发出执行通知书和报告财产令
- 已对被执行人采取限制消费措施/纳入失信名单
- 已穷尽财产调查措施（网络查控+传统调查）
- 未发现可供执行的财产/发现的财产不能处置
- 申请执行人签字确认或法院组成合议庭审查核实

**终本后：**
- 申请执行人发现可供执行财产 → 可随时申请恢复执行（不受申请执行时效限制）
- 法院定期（每6个月）对终本案件进行财产统查
- 失信/限高措施继续有效

### 执行完毕

- 全部履行完毕 → 执行法院应当结案
- 以物抵债完成 → 结案
- 执行和解并履行完毕 → 结案

## 注意事项

- 执行程序是"最后一道防线"——判决易得，执行难求。财产调查是执行成功的关键。
- 法人被执行人的股东以认缴出资额为限承担有限责任——穿透至股东的路径有限且条件严格（出资加速到期、一人公司财产混同等）。
- 终本案件占比高是现实——"终本"不意味着放弃，发现财产可随时恢复执行。
- 网络司法拍卖通常比传统拍卖效率更高，评估价与起拍价的差距直接影响回款率。
