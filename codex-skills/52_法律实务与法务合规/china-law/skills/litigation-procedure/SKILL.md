# /litigation-procedure

提供中国民事诉讼程序指引，覆盖从起诉到执行的全程。

## 指令

### 1. 事务上下文
- 纠纷性质？（合同/侵权/物权/公司纠纷/劳动争议/知识产权/婚姻家庭/继承）
- 诉讼标的额？
- 管辖连接因素？（被告住所地/合同履行地/侵权地/标的物所在地/约定管辖）

### 2. 核心法律框架

#### 2.1 管辖（《民事诉讼法》第22-38条 `[PKULaw]`）

**一般地域管辖（第22条 `[PKULaw]`）：**
- 被告为公民 → 被告住所地法院管辖；住所地与经常居住地不一致 → 经常居住地
- 被告为法人或其他组织 → 被告住所地法院管辖

**合同纠纷管辖（第24条）：**
被告住所地或合同履行地

**侵权纠纷管辖（第29条）：**
侵权行为地或被告住所地

**协议管辖（第35条 `[PKULaw]`）：**
- 合同或其他财产权益纠纷 → 可书面协议选择
- 可选择范围：被告住所地、合同履行地、合同签订地、原告住所地、标的物所在地等与争议有实际联系的地点
- 不得违反级别管辖和专属管辖

**专属管辖：**
- 不动产纠纷 → 不动产所在地法院
- 港口作业纠纷 → 港口所在地法院  
- 遗产继承纠纷 → 被继承人死亡时住所地或主要遗产所在地法院

→ 管辖协议约定两个以上与争议有实际联系地点的，原告可向其中一个起诉（民事诉讼法司法解释(2022修正)第30条 `[PKULaw]`）

#### 2.2 诉讼时效（《民法典》第188-197条 `[PKULaw]`）

- **普通时效：** 3年，自权利人知道或应当知道权利受损及义务人之日起计算
- **最长时效：** 20年，自权利受损之日起，不适用中止/中断
- **中断事由（第195条）：** 提出履行请求 / 义务人同意履行 / 提起诉讼或仲裁 / 其他
- **中止事由（第194条）：** 不可抗力 / 无/限制民事行为能力人无法定代理人 / 其他

→ 民法典总则编司法解释第35条 `[PKULaw]`：3年时效可适用中止、中断，不适用延长；20年期间不适用中止、中断

#### 2.3 起诉条件（第122条）

- 原告是与本案有直接利害关系的公民、法人或其他组织
- 有明确的被告
- 有具体的诉讼请求和事实、理由
- 属于人民法院受案范围和受诉人民法院管辖

#### 2.4 证据规则

**证明责任（民事诉讼法司法解释第90条 `[PKULaw]`）：**
- 当事人对自己提出的诉讼请求所依据的事实或反驳对方诉讼请求所依据的事实 → 应当提供证据证明
- 判决前未能提供证据或证据不足以证明事实主张 → 由负有举证证明责任的当事人承担不利后果

**证据审核（第105条 `[PKULaw]`）：**
- 全面、客观审核证据
- 运用逻辑推理和日常生活经验法则
- 判断有无证明力和证明力大小
- 公开判断理由和结果

**质证（第103条 `[PKULaw]`）：**
- 证据应当在法庭上出示，由当事人互相质证
- 未经当事人质证的证据，不得作为认定案件事实的根据
- 涉及国家秘密、商业秘密、个人隐私的证据 → 不得公开质证

**法院调查取��（第96条）：**
涉及可能损害国家利益/社会公共利益的；涉及身份关系的；涉及恶意串通损害他人合法权益可能的；涉及依职权追加当事人、中止诉讼、终结诉讼、回避等程序性事项的

#### 2.5 保全措施

- **财产保全：** 判决可能难以执行或造成其他损害 → 查封、扣押、冻结
- **行为保全：** 侵害行为正在实施或即将实施 → 责令停止
- **证据保全：** 证据可能灭失或以后难以取得

#### 2.6 审级制度

- **一审：** 基层/中级/高级/最高（按标的额和案件类型确定级别）
- **上诉期限：** 判决书送达之日起15日内上诉；裁定书10日内
- **二审：** 事实审 + 法律审（终审）
- **再审：** 当事人申请/法院依职权/检察院抗诉

#### 2.7 执行（第247条 `[PKULaw]`）

- 发生法律效力的民事判决、裁定 → 当事人必须履行
- 一方拒绝履行 → 对方可向法院申请执行
- 执行法院：第一审法院或与一审法院同级的被执行财产所在地法院

#### 2.8 外国判决承认与执行（民事诉讼法司法解释第544条 `[PKULaw]`）

- 需先申请承认 → 法院审查 → 裁定承认后 → 依民事诉讼法第三编执行
- 仅申请承认而未同时申请执行的 → 仅对应否承认进行审查并裁���
- 依据：中国缔结或参加的国际条约或互惠原则
