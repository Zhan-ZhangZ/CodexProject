# amazon_sp_mcp — 跨境电商与店铺管理 MCP 服务

- **项目主页**: https://github.com/jay-trivedi/amazon_sp_mcp

## 功能说明

亚马逊 Selling Partner API (SP-API) 销售与运营数据集成 MCP。支持读取店铺每日销售额、财务报告、买家退货分析及订单列表，辅助进行多维度的店铺数据盘点与经营状况分析。
