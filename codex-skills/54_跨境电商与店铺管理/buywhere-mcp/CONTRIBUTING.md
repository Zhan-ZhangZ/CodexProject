# Contributing to BuyWhere MCP Server

Thanks for your interest in contributing! We welcome issues, PRs, and suggestions.

## How to Contribute

### Report Bugs
Open an issue using the [bug report template](.github/ISSUE_TEMPLATE/bug_report.md).

### Suggest Features
Open an issue using the [feature request template](.github/ISSUE_TEMPLATE/feature_request.md).

### Submit Code Changes

1. Fork the repo
2. Create a branch: `git checkout -b feature/your-feature`
3. Make your changes
4. Run `npm run build` to verify it compiles
5. Commit with clear messages
6. Push and open a Pull Request

### Pull Request Guidelines

- Keep PRs focused on a single change
- Update the README if your change affects usage
- Run `npm run build` before submitting
- Reference any related issues

## Questions?

Start a [GitHub Discussion](https://github.com/BuyWhere/buywhere-mcp/discussions) or open an issue.
