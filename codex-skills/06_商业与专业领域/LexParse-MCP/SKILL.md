# LexParse-MCP — 中国法律相关 MCP 服务与智能体技能

- **项目主页**: https://github.com/1wu-davy-2/LexParse-MCP

## 功能说明

LexParse MCP | Claude 法律文书结构化 MCP 服务。支持中国大陆与香港法律体系，一键把判决书、合同、起诉状等法律文件（PDF/Word/文本）转为结构化 JSON，自动提取当事人、案情事实、法律依据、判决结果和风险条款，并进行风险比对。
