# kicad-mcp-automation — 电气与智能化设计 MCP 服务

- **项目主页**: https://github.com/muammar-yacoob/kicad-mcp

## 功能说明

KiCad 电路设计与生产自动化审核 MCP。支持通过自然语言创建 PCB 项目、运行电气规则检查 (ERC)、设计规则检查 (DRC)、自动生成物料清单 (BOM) 并导出 Gerber 制造文件。
