# nexar-mcp — 电气与智能化设计 MCP 服务

- **项目主页**: https://github.com/Farad-Labs/nexar-mcp

## 功能说明

对接 Octopart/Nexar API 的电子元器件搜索引擎 MCP。支持元器件型号搜索、实时库存与多渠道报价查询、技术参数对比及数据手册 (Datasheet) 链接检索。
