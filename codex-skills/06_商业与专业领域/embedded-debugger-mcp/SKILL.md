# embedded-debugger-mcp — 电气与智能化设计 MCP 服务

- **项目主页**: https://github.com/Adancurusul/embedded-debugger-mcp

## 功能说明

基于 probe-rs 的底层硬件级在线调试 MCP。支持通过 J-Link、ST-Link 等仿真器直接连接 ARM Cortex-M 和 RISC-V 芯片，进行固件烧录、寄存器读写和断点调试。
