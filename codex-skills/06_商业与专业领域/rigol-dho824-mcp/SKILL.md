# rigol-dho824-mcp — 电气与智能化设计 MCP 服务

- **项目主页**: https://github.com/aimoda/rigol-dho824-mcp

## 功能说明

普源 Rigol DHO824 系列数字示波器远程控制 MCP。支持通过以太网 LXI 协议读取仪器状态、抓取当前屏幕截图、采集多通道测量数据，并将波形图导出给智能体分析。
