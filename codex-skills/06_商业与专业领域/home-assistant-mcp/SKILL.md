# home-assistant-mcp — 电气与智能化设计 MCP 服务

- **项目主页**: https://github.com/oleander/home-assistant-mcp-server

## 功能说明

Home Assistant 智能家居与场景控制 MCP 服务。支持通过自然语言查询智能硬件状态、调用服务控制家居设备（如灯光、开关、空调），支持本地 stdio 和远程 SSE 传输。
