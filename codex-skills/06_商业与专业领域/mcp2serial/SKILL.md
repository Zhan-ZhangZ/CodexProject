# mcp2serial — 电气与智能化设计 MCP 服务

- **项目主页**: https://github.com/mcp2everything/mcp2serial

## 功能说明

通用物理串口与控制台交互 MCP。支持直接访问并控制主机的物理串口，支持与各类电子实验板卡（如树莓派 Pico 等）进行直接控制台会话和数据流读写。
