# mqtt-mcp — 电气与智能化设计 MCP 服务

## 功能说明

轻量级 MQTT 消息网关与物联网调试 MCP。使智能体可以直接连接至本地或远程 MQTT 代理服务器（Broker），实现消息发布、主题订阅及实时硬件遥测数据流监听。
