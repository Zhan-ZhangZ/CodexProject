# AmazonSeller-mcp-server — 跨境电商与店铺管理 MCP 服务

- **项目主页**: https://github.com/mattcoatsworth/AmazonSeller-mcp-server

## 功能说明

亚马逊卖家中心产品管理与多仓库存同步 MCP。支持管理店铺商品的在线刊登、查询实时库存余量，并支持跨平台的库存信息自动对齐。
