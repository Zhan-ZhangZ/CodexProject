# MODBUS-MCP — 电气与智能化设计 MCP 服务

- **项目主页**: https://github.com/midhunxavier/MODBUS-MCP

## 功能说明

工业 Modbus 协议通讯控制 MCP。支持通过 Modbus TCP、UDP 或 Serial/RTU 串口方式与工业设备建立连接，执行读取/写入线圈和寄存器操作以控制 PLC、传感器和执行机构。
