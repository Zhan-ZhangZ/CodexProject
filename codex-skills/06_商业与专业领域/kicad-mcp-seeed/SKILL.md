# kicad-mcp-seeed — 电气与智能化设计 MCP 服务

- **项目主页**: https://github.com/Seeed-Studio/kicad-mcp-server

## 功能说明

Seeed Studio 官方开发的 KiCad 智能分析与设计助手 MCP。支持基于 KiCad 9.0+ 的原理图分析、PCB 连接关系追踪、元器件网络检查及生成嵌入式驱动代码。
