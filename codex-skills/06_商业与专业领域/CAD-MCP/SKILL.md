# CAD-MCP — 2D 平面工程制图自动化

- **项目主页**: https://github.com/daobataotie/CAD-MCP

## 功能说明

`CAD-MCP` 是一个通过 Windows COM 接口桥接大语言模型与主流商用 CAD 软件的 MCP Server。它使得大模型能够理解自然语言的制图需求，并直接在本地运行的 CAD 软件中绘制实体图纸。

### 适用软件
- AutoCAD
- 中望 CAD (ZWCAD)
- 浩辰 CAD (GstarCAD)

### 核心能力
- **基础绘图**：绘制直线、多段线、圆、圆弧、矩形等。
- **标注与文字**：添加单行/多行文字、线性/对齐标注。
- **图层管理**：新建图层、切换当前图层、设置颜色。
- **图案填充**：对封闭区域进行 Hatch 填充。
- **文件操作**：保存当前图纸为 DWG 文件。

## 使用条件

- **操作系统**：必须为 Windows（依赖 `pywin32` 和 COM 接口）。
- **环境**：Python 环境，需安装 `mcp`, `pywin32`, `pydantic` 等依赖。
- **CAD 软件**：本地必须已经安装并处于运行状态的支持的 CAD 软件。

## 典型场景

- "帮我画一个长 100 宽 50 的矩形，放到 '轮廓' 图层，颜色设为红色。"
- "在坐标 (50, 25) 处画一个半径为 10 的圆，并填充斜线图案。"
- "批量为图纸上所有的长方形添加长度标注。"
