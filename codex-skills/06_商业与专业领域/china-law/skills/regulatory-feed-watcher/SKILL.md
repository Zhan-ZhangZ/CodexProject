# /regulatory-feed-watcher

监控中国法律法规变化，过滤实质性更新，产生可操作的摘要。

## 指令

### 1. 读取配置

读取 `~/.claude/plugins/config/claude-for-legal/china-law/CLAUDE.md` → 实务领域配置和管辖覆盖范围。

### 2. 监控范围

#### 法律渊源监控层级

| 层级 | 来源 | 监控方式 |
|------|------|---------|
| 法律 | 全国人大及其常委会 | 北大法宝 + 全国人大网 |
| 行政法规 | 国务院 | 北大法宝 + 中国政府网 |
| 部门规章 | 国务院各部委 | 北大法宝 + 各部委网站 |
| 地方性法规 | 省级/设区的市人大 | 北大法宝 + 地方人大网 |
| 司法解释 | 最高法/最高检 | 北大法宝 + 最高法官网 |

#### 监控的实务领域（根据配置）

- 劳动用工：劳动合同法、社会保险法、劳动争议调解仲裁法相关修订和司法解释
- 商业合同：民法典合同编相关司法解释和指导意见
- 数据合规：PIPL/DSL/CSL配套规定、国家标准、部门规章
- 知识产权：专利法/商标法/著作权法修订和实施条例
- 公司治理/FIL：公司法司法解释、外商投资负面清单更新
- 争议解决：民事诉讼法修订、仲裁法修订

### 3. 实质性判断标准

**始终实质性（立即行动）：**
- 新颁布的法律、行政法规、司法解释
- 生效日期已确定的新规
- 对公司有直接合规义务的修订
- 行政处罚案例涉及本行业

**值得审阅（评估后决定）：**
- 征求意见稿/草案
- 指导意见/通知（非强制性）
- 对竞争者的执法行动
- 地方法规更新（如在当地经营）

**FYI（记录，无需行动）：**
- 法律学术评论
- 立法动态（尚未进入审议）
- 与本行业无关的法规变化

### 4. 检索操作

**北大法宝 MCP 检索 `[PKULaw]`：**

```
# 按时间检索最新法规
mcp__pkulaw-law-search__search_article
text: "[领域关键词] 2026"
```

**手动补充来源：**
- 全国人大网 (npc.gov.cn)
- 中国政府网 (gov.cn)
- 最高人民法院网 (court.gov.cn)
- 国家市场监督管理总局 (samr.gov.cn)
- 国家互联网信息办公室 (cac.gov.cn)

## 运行节奏

- **周报：** 每周一运行，检查上周变化（推荐默认）
- **日报：** 仅在重大立法活跃期使用

运行 `/china-law:regulatory-feed-watcher --since 2026-05-13` 检查自上周以来的变化。

## 注意事项

- 本摘要为AI生成的法律法规动态过滤——所引法规可能未被完全核实。在依赖任何法规变化作出决策前，请通过北大法宝或一手来源确认当前文本和生效状态。
- 标注 `[PKULaw]` 的引用已经北大法宝 MCP 检索确认；标注 `[模型知识 — 请核实]` 的引用依赖训练知识。

**参考法律渊源:**
- 《中华人民共和国立法法》(2023修正)第87-92条 `[PKULaw]`
- 《法规规章备案规定》`[PKULaw]`
