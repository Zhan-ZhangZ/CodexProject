## Purpose

中国税务法律体系是所有法律领域中变动最频繁的之一——增值税法于2024年底刚刚通过（2026年1月1日施行），《税收征收管理法》正在修订中，新的反避税规则和转让定价合规要求不断出台。本技能提供中国税法框架下的系统性法律分析，从交易税负评估到刑事责任边界界定。

## 分析类型

| 类型 | 适用场景 |
|------|---------|
| 交易税负评估 | 并购/重组/资产转让的税务成本和法律风险 |
| 筹划方案合法性评估 | 税务筹划方案是否构成"避税安排"或"逃税" |
| 转让定价/关联交易 | 关联交易定价是否公允、文档准备是否合规 |
| 税收优惠资格审查 | 高新技术企业/研发加计扣除/小微企业等优惠的资格存续审查 |
| 税务争议分析 | 税务稽查、纳税评估、行政复议、行政诉讼的前景分析 |
| 税务刑事风险评估 | 逃税罪、虚开增值税专用发票罪的刑事追诉风险 |

## Step 2 — 主要税种框架

### 2.1 企业所得税

《中华人民共和国企业所得税法》(2018修正) `[PKULaw]`：

| 项目 | 内容 |
|------|------|
| 法定税率 | **25%**（标准税率） |
| 高新技术企业 | **15%**（需取得科技厅认定） |
| 小型微利企业 | **20%**（且有额外减免） |
| 西部大开发鼓励类产业 | **15%** |
| 预提所得税（股息/利息/特许权使用费） | **10%**（税收协定可能降为5%/7.5%） |

**关键制度：**
- **居民企业 vs. 非居民企业** — 实际管理机构在中国境内即为居民企业（全球所得纳税）
- **关联交易与转让定价** — 独立交易原则（Arm's Length Principle）
- **受控外国企业（CFC）** — 境外子公司利润可能被"穿透"至境内母公司征税
- **反避税** — 一般反避税规则（GAAR），"不具有合理商业目的"的安排
- **特别纳税调整** — 补税+利息（非罚款），利率为基准利率+5个百分点

### 2.2 增值税

《中华人民共和国增值税法》(2024，2026年1月1日施行) `[PKULaw]`：

**⚠️ 增值税法2024年12月25日通过，于2026年1月1日正式施行——在这之前仍适用《增值税暂行条例》。**

| 项目 | 内容 |
|------|------|
| 一般纳税人税率 | **13%**（货物/加工修理修配/有形动产租赁）、**9%**（不动产/建筑/交通运输/农产品等）、**6%**（服务/无形资产） |
| 小规模纳税人征收率 | **3%**（现行减按1%） |
| 出口退税 | 零税率（退税率0%/6%/9%/13%不等） |
| 留抵退税 | 符合条件的可申请退还期末留抵税额 |

**增值税法主要变化（2026年起 vs. 原暂行条例）：**
1. 法律层级提升——增值税法取代暂行条例
2. 小规模纳税人标准进一步明确
3. 留抵退税制度写入法律
4. 不得抵扣项目缩减（餐饮服务有望抵扣）
5. 电子发票推广法定化

### 2.3 个人所得税

《中华人民共和国个人所得税法》(2018修正) `[PKULaw]`：

| 所得类型 | 税率 | 计税方式 |
|---------|------|---------|
| 工资薪金 | **3%-45%**（七级超额累进） | 综合所得（年度汇算） |
| 劳务报酬 | 并入综合所得 | 综合所得 |
| 经营所得 | **5%-35%**（五级超额累进） | 单独计税 |
| 股息红利 | **20%** | 分类所得 |
| 财产转让所得 | **20%** | 分类所得 |
| 偶然所得 | **20%** | 分类所得 |

**关键制度：**
- 综合与分类相结合的税制（2019年改革）
- 专项附加扣除（子女教育、继续教育、大病医疗、住房贷款利息/租金、赡养老人、3岁以下婴幼儿照护）
- 个人所得税反避税条款（第8条：关联交易/CFC/GAAR）
- 境外所得税收抵免

### 2.4 其他主要税种

| 税种 | 主要规定 | 关键税率 |
|------|---------|---------|
| 印花税 | 《印花税法》(2022年7月1日施行) `[PKULaw]` | 合同0.03‰-1‰；产权转移书据0.5‰；营业账簿0.25‰ |
| 契税 | 《契税法》(2021年9月1日施行) `[PKULaw]` | **3%-5%**（省市可在该幅度内确定具体税率） |
| 房产税 | 《房产税暂行条例》`[PKULaw]` | 从价：1.2%（房产余值）；从租：12% |
| 土地增值税 | 《土地增值税暂行条例》`[PKULaw]` | **30%-60%**（四级超额累进） |
| 城镇土地使用税 | 《城镇土地使用税暂行条例》 | 定额税率（因城市而异） |
| 消费税 | 《消费税暂行条例》 | 烟、酒、高档化妆品、贵重首饰、汽车等 |

## Step 3 — 转让定价与关联交易

### 3.1 关联关系认定

《企业所得税法实施条例》第109条 `[PKULaw]` + 国家税务总局公告2016年第42号：

| 关联关系类型 | 认定标准 |
|------------|---------|
| 股权控制 | 直接或间接持有**25%以上**股份 |
| 资金借贷 | 持有对方**10%以上**股份且借贷>注册资本50%或借贷>10%总资产 |
| 特许权 | 特许权许可对生产经营构成依赖 |
| 购销控制 | 购买/销售/劳务等受一方控制 |
| 人员控制 | 半数以上董事/高管由一方委派 |
| 其他 | 实质上具有其他共同利益的关系 |

### 3.2 转让定价方法

《特别纳税调整实施办法（试行）》`[PKULaw]`（国家税务总局公告2016年第42号更新）：

| 方法 | 适用场景 |
|------|---------|
| 可比非受控价格法（CUP） | 有可比非关联交易价格的场景 |
| 再销售价格法 | 分销型企业 |
| 成本加成法 | 制造/服务提供企业 |
| 交易净利润法（TNMM） | 最常用的方法，适合大多数场景 |
| 利润分割法 | 高度整合的关联交易（如联合研发） |

### 3.3 合规文档（同期资料）

| 文档类型 | 准备门槛 | 提交期限 |
|---------|---------|---------|
| **主体文档** | 年度关联交易总额**>10亿元** | 关联交易发生年度次年6月30日前 |
| **本地文档** | 关联交易>4000万元（有形资产）/1亿元（金融资产/无形资产）/全部关联交易合计>2亿元 | 同上 |
| **国别报告** | 集团合并收入>55亿元 | 次年12月31日前 |
| **特殊事项文档** | 成本分摊协议/资本弱化（债资比>2:1非金融企业或>5:1金融企业） | 次年6月30日前 |

**不合规后果：**
- 补缴税款 + 利息（利率为基准利率+5个百分点，而非一般滞纳金的万分之五/日）
- 被列为重点稽查对象
- 行政处罚（罚款）

## Step 4 — 税务筹划合法性边界

### 4.1 "避税安排" vs. "合理税务筹划"

《企业所得税法》第47条 + 《特别纳税调整实施办法》第10章 `[PKULaw]`：

| 判断标准 | 合理税收筹划 | 避税安排（可被调整） |
|---------|------------|-----------------|
| 商业目的 | 具有合理商业目的（非仅为税务目的） | 不具有合理商业目的 |
| 经济实质 | 具有经济实质 | 缺乏经济实质，仅为形式 |
| 交易形式 | 交易形式反映经济实质 | 形式与经济实质不符 |
| 税收结果 | 与商业目标一致 | 主要目的为获取税收利益 |

### 4.2 逃税罪边界

《刑法》第201条 `[PKULaw]`：

**逃税罪构成要件：**
- 采取欺骗、隐瞒手段进行虚假纳税申报或不申报
- 逃避缴纳税款**数额较大**（10万元以上）且占应纳税额**10%以上**

**⚠️ 免责条款（第201条第4款）：**
- 已受行政处罚（补税+滞纳金+罚款）
- **且**五年内未因逃税受过刑事处罚或被两次以上行政处罚
- → 可不予追究刑事责任

这一"初犯免刑"条款是有争议的——是税务筹划最后的安全网，但不是盾牌。

### 4.3 虚开增值税专用发票罪

《刑法》第205条 `[PKULaw]` — 这是中国税法中**最危险的刑事红线**：

| 量刑档次 | 虚开税额 | 刑罚 |
|---------|---------|------|
| 第一档 | 5万元以上 | 3年以下有期徒刑或拘役 + 2-20万罚金 |
| 第二档 | 50万元以上（数额较大） | 3-10年有期徒刑 + 5-50万罚金 |
| 第三档 | 500万元以上（数额巨大） | **10年以上有期徒刑或无期徒刑** + 5-50万罚金或没收财产 |

**⚠️ 关键判例变化：** 最高人民法院在2018年指导案例中明确——不以骗税为目的、未造成国家税收损失的"对开""环开"增值税发票行为，不构成虚开增值税专用发票罪。但司法解释仍存不确定，慎重对待。

## Step 5 — 税务争议解决

### 5.1 争议解决路径

```
税务争议解决路径

        税务稽查通知
             │
             ▼
        税务处理决定 / 税务行政处罚决定
             │
             ▼
    ┌─ 纳税争议（税额/税基/税率争议）──→ 必须先缴纳税款/提供担保
    │        │                           然后向上一级税务机关
    │        ▼                           申请行政复议
    │   行政复议（前置）
    │        │
    │        ▼
    │   行政诉讼（二审终审）
    │
    ├─ 处罚争议（罚款/没收/停供发票）──→ 行政复议或直接诉讼（并行）
    │
    └─ 其他行政行为──→ 行政复议或直接诉讼
```

### 5.2 税务行政复议

《税收征收管理法》第88条 + 《税务行政复议规则》`[PKULaw]`：

| 争议类型 | 复议前置 | 前提条件 |
|---------|---------|---------|
| 纳税争议（税额、税基、税率） | **是** | 必须先缴纳税款/滞纳金或提供相应担保 |
| 税务行政处罚 | 否（可直接诉讼） | — |
| 税收保全措施/强制执行措施 | 否（可直接诉讼） | — |
| 发票管理行为 | 否（可直接诉讼） | — |

**复议期限：** 知道该具体行政行为之日起 **60日**内。

### 5.3 税务行政诉讼

《行政诉讼法》+ 最高人民法院行政诉讼法司法解释 `[PKULaw]`：

- **起诉期限：** 复议决定送达之日起**15日**内（经复议后）；知道行政行为之日起**6个月**内（直接起诉）
- **举证责任：** 税务机关对其行政行为的合法性承担举证责任
- **诉讼期间不停止执行：** 税务处理决定在诉讼期间原则上继续执行

## 注意事项

- 税法变动频繁——本技能中引用的税率、门槛和数据须在出具意见前通过北大法宝检索确认是否为最新版本。特别是增值税法于2026年1月1日施行后，大量配套规则将陆续出台。
- 税收筹划的合法性边界是模糊的——"不具有合理商业目的"是一个可被反避税调整的标准。当一项筹划方案的主要或唯一目的是减少税负时，应标注 `[筹划风险 — 可能被视为避税安排]`。
- 虚开增值税专用发票罪是中国税法中最严重的刑事风险——指导案例缩小了定罪范围，但并未完全消除不确定性。涉及发票的任何安排都应极度谨慎。
- 纳税争议的复议前置和"先缴税后复议"规则是程序障碍——在规定期限内未缴纳税款可能丧失复议和诉讼权。标注时间期限至关重要。
- 跨国交易的税收协定适用、常设机构（PE）风险、混合错配安排等涉及国际税收规则（BEPS/MLI/支柱二），需结合具体税收协定分析。
