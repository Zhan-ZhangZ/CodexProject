# /trademark-search

中国商标在先权利检索——申请前排查、使用风险评估。

## 指令

### 1. 事务上下文

- 商标类型？（文字/图形/组合/声音/颜色组合）
- 拟使用的商品/服务类别？（尼斯分类第1-45类）
- 使用场景？（中国境内使用/跨境使用）
- 是否已投入使用？（申请前/使用中）

### 2. 核心法律框架（《商标法》(2019修正) `[PKULaw]`）

#### 2.1 注册原则

中国商标制度以注册为基础，先申请原则（第31条 `[PKULaw]`）：
- 同日申请 → 使用在先优先
- 在先注册商标 → 阻却在后相同/近似商标在相同/类似商品上的注册

#### 2.2 在先权利排查范围

**绝对理由（第9-12条）：**
- 仅有本商品的通用名称、图形、型号
- 仅直接表示商品的质量、主要原料、功能、用途等特点
- 缺乏显著特征
- 三维标志为商品自身性质产生的形状/为获得技术效果需要的形状/使商品具有实质性价值的形状

**相对理由（第13条、第15条、第32条）：**
- 与已注册驰名商标相同/近似（跨类保护）（第13条 `[PKULaw]`）
- 与未注册驰名商标相同/近似（同类保护）
- 代理人/代表人未经授权抢注（第15条）
- 损害他人在先权利（商号权、著作权、外观设计专利权、姓名权、肖像权）（第32条 `[PKULaw]`）
- 以不正当手段抢注他人已经使用并有一定影响的商标
- 与地理标志冲突

#### 2.3 近似判断标准（商标审查审理指南）

**文字商标近似：**
- 中文商标汉字构成相同，仅字体/读音/排列顺序不同 → 近似
- 外文商标由三个以上字母构成，仅个别字母不同 → 易使相关公众混淆的构成近似
- 商标文字读音相同/近似，字形不同 → 易使相关公众混淆的构成近似

**图形商标近似：**
- 构图和整体外观近似 → 易使相关公众混淆的构成近似
- 完整包含他人有显著性的图形商标 → 近似

**组合商标近似：**
- 商标文字/图形不同但排列组合方式近似 → 易使混淆的构成近似

#### 2.4 三年不使用撤销（第49条 `[PKULaw]`）

注册商标无正当理由连续三年不使用的 → 任何单位或个人可向商标局申请撤销

→ 存量商标稳定性判断的关键依据

### 3. 检索策略

#### 3.1 检索层级

1. **完全相同的在先注册/申请** → 阻断性
2. **高度近似（文字/读音/含义/整体外观）在同类别** → 高风险
3. **跨类别驰名商标** → 需判断驰名度及跨类混淆可能性
4. **已注册但可撤销的商标**（三年不使用） → 可能路径
5. **在先权利冲突**（商号/著作权/外观设计等） → 综合判断

#### 3.2 检索操作

**CNIPA商标数据库（通过CDP浏览器或WebFetch）：**
- 文字商标：精确检索 + 模糊检索
- 图形商标：图形编码检索（维也纳分类）
- 组合商标：分要素检索

**北大法宝案例检索 `[PKULaw]`：**
- 检索相关侵权判例："商标侵权 [商标名称] 近似"
- 检索撤销案例："三年不使用 撤销 [商标名称]"

**市场主体登记检索：**
- 检索商号登记（全国企业信用信息公示系统）
- 检索域名注册

### 4. 检索结果评估

#### 4.1 风险评估标准

| 严重度 | 标准 |
|-------|------|
| 🔴 阻断性 | 相同文字商标在相同类别已注册；驰名商标跨类明确保护 |
| 🟠 高 | 高度近似的文字/图形在同类别；在先使用并有一定影响的商标 |
| 🟡 中 | 读音相近但字形不同；跨类但非驰名商标；撤销可能 |
| 🟢 低 | 未发现冲突；显著性强；建议立即提交申请确权 |

#### 4.2 可注册性评估清单

| 审查项 | 状态 |
|-------|------|
| 相同类别在先相同/近似商标 | ✓/✗ |
| 类似类别在先近似商标 | ✓/✗ |
| 跨类驰名商标风险 | ✓/✗ |
| 三年不使用撤销可能性（存量商标） | ✓/✗ |
| 缺乏显著性风险 | ✓/✗ |
| 在先权利冲突（商号/著作权等） | ✓/✗ |
| 不良影响风险（第10条） | ✓/✗ |

### 5. 输出

检索报告 + 冲突商标清单 + 风险评估矩阵 + 可注册性判断 + 建议行动（申请/修改/取得共存协议/放弃）+ 决策树。
