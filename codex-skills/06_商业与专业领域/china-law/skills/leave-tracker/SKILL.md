# /leave-tracker

检查所有带法定硬性期限的员工假期，仅筛选需要决策或行动的事项。不是状态报告——告诉你需要做什么及为什么。

## 指令

1. 读取 `~/.claude/plugins/config/claude-for-legal/china-law/leave-register.yaml`。如果文件不存在，提示用户通过 `/china-law:log-leave` 添加假期记录或上传假期电子表格。

2. 仅对需要行动的假期发出预警。无问题的假期一行摘要。

3. 每周一运行——设置周一早间提醒调用 `/china-law:leave-tracker`。

## 中国法定假期类型与期限

### 年休假（《职工带薪年休假条例》第3条 `[PKULaw]`）

| 累计工作年限 | 年休假天数 |
|------------|----------|
| 1年以上不满10年 | 5天 |
| 10年以上不满20年 | 10天 |
| 20年以上 | 15天 |

**不享受当年年休假情形（第4条 `[PKULaw]`）：**
- 享受寒暑假天数多于年休假天数
- 请事假累计20天以上且单位不扣工资
- 累计工作满1年不满10年，请病假累计2个月以上
- 累计工作满10年不满20年，请病假累计3个月以上
- 累计工作满20年以上，请病假累计4个月以上

→ 年休假可跨1个年度安排（第5条 `[PKULaw]`）

### 病假/医疗期（《企业职工患病或非因工负伤医疗期规定》 `[PKULaw]`）

| 实际工作年限 | 本单位工作年限 | 医疗期 |
|------------|-------------|-------|
| 10年以下 | 5年以下 | 3个月 |
| 10年以下 | 5年以上 | 6个月 |
| 10年以上 | 5年以下 | 6个月 |
| 10年以上 | 5-10年 | 9个月 |
| 10年以上 | 10-15年 | 12个月 |
| 10年以上 | 15-20年 | 18个月 |
| 10年以上 | 20年以上 | 24个月 |

→ 医疗期届满可能触发《劳动合同法》第40条第1项的解除条件

### 产假（《女职工劳动保护特别规定》第7条 `[PKULaw]`）

- 基本产假：98天（产前15天 + 产后83天）
- 难产：+15天
- 多胞胎：每多1个 +15天
- 流产：未满4个月 → 15天；满4个月 → 42天
- 各地奖励假不等（上海30天、北京30-90天等）

→ 产假期满前返岗决策点

### 工伤停工留薪期（《工伤保险条例》第33条 `[PKULaw]`）

- 一般不超过12个月
- 伤情严重或情况特殊经鉴定委员会确认可延长不超过12个月
- 期间原工资福利待遇不变

→ 12个月和24个月为关键决策点

### 育儿假

各地规定不一，多数省市每年5-10天（子女3周岁前）。

### 陪产假

各地规定不一，多数省市7-15天。

## 预警触发条件

| 假期类型 | 预警类型 | 时间节点 |
|---------|---------|---------|
| 医疗期 | 届满预警 | 届满前30天 |
| 医疗期 | 届满后解除评估 | 届满日 |
| 年休假 | 跨年度截止 | 次年12月31日前30天 |
| 产假 | 返岗预警 | 产假期满前14天 |
| 工伤停工留薪期 | 12个月届满 | 届满前30天 |
| 工伤停工留薪期 | 24个月最终届满 | 届满前30天 |
