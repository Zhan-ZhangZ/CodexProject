# /investigation-report

1. 读取 `~/.claude/plugins/config/claude-for-legal/china-law/CLAUDE.md` → 管辖与法律框架、保密与工作成果保护。
2. 设计调查方案。
3. 审查证据材料，确定待调查事实。
4. 通过北大法宝 MCP 检索相关法律规定 `[PKULaw]`。
5. 出具调查报告——事实认定 + 法律评价 + 处分/整改建议。

---

## Purpose

企业内部调查处理不当会引发劳动争议（违法解除）、名誉侵权（对员工的错误指控）、证据灭失（未及时保全），甚至刑事风险（包庇犯罪）。本技能提供中国法律框架下的结构化调查方法和报告格式——确保调查在程序上站得住脚，在实体上有证据支撑。

## 常见调查事项

| 类型 | 典型行为 | 涉及法律 |
|------|---------|---------|
| 职务侵占/挪用资金 | 利用职务便利侵占/挪用公司财物 | 《刑法》第271-272条 `[PKULaw]` |
| 商业贿赂 | 收受回扣/贿赂/利益冲突交易 | 《刑法》第163-164条；《反不正当竞争法》第7条 `[PKULaw]` |
| 侵犯商业秘密 | 泄露/窃取/未经授权使用公司商业秘密 | 《反不正当竞争法》第9条；《刑法》第219条 `[PKULaw]` |
| 利益冲突/自我交易 | 未经披露的关联交易/同业竞争 | 《公司法》(2023修订)第180-185条（董事/高管忠实义务）`[PKULaw]` |
| 虚假报销/财务欺诈 | 虚报费用/伪造凭证/虚假账目 | 公司内部规章制度；严重者可涉刑事（职务侵占/诈骗） |
| 职场骚扰/歧视 | 性骚扰/职场霸凌/歧视 | 《民法典》第1010条；《女职工劳动保护特别规定》第11条 `[PKULaw]` |
| 竞业限制/保密违约 | 违约从事竞争性业务/泄露保密信息 | 《劳动合同法》第23-24条；《反不正当竞争法》第9条 `[PKULaw]` |
| 违规操作/违反内部制度 | 违反公司业务流程、审批制度等 | 公司内部规章制度；《劳动合同法》第39条第2项 `[PKULaw]` |

## Step 1 — 案件受理与初步评估

### 受理时需确认

- [ ] 举报来源（实名/匿名举报、管理层指令、日常监管发现）
- [ ] 举报是否具备基本可信度（并非所有举报都需要正式调查）
- [ ] 如果举报人是实名举报，是否已告知会对其身份保密
- [ ] 被举报人是否处于关键岗位（是否需要立即采取保护措施——如暂停系统权限、调整工作安排，但尚未定性）
- [ ] 是否需要保护证据（文件、邮件、系统日志、监控录像等可能正在被销毁）

🔴 **紧急保护措施**（在调查结论出来前采取的临时措施，不构成处分）：
- 暂停被调查人的系统访问权限
- 将被调查人暂时调离涉事岗位（非处分性调岗）
- 要求IT保全相关邮件和文件

## Step 2 — 调查方案设计

### 调查方案内容

```
调查方案 — [案件名称]

1. 调查范围：明确要查清的事项清单

2. 证据收集计划：
   - 文件证据：[需要获取的文件类型和来源]
   - 电子数据：[邮件、系统日志、即时通讯记录等的获取方式和范围]
   - 物证：[实物证据的获取和保管]
   - 证人：[需要访谈的人员及顺序]

3. 调查组成员：
   - 组长：[姓名/职务]
   - 成员：[姓名/职务]
   - 是否外部律师参与（如涉及复杂法律问题/高管被调查）

4. 时间安排：
   - [各阶段的时间节点]

5. 保密要求：
   - 调查组成员签署保密承诺
   - 不得向无关人员透露调查信息
   - 避免对被调查人造成不必要的声誉影响

6. 回避检查：
   - 调查组成员是否与被调查人或举报人存在利害关系
```

## Step 3 — 证据收集

### 证据类型与合法性

| 证据类型 | 收集方式 | 合法性注意事项 |
|---------|---------|-------------|
| 公司邮件/系统日志 | IT部门提取 | 公司设备/公司系统 → 属于公司财产，合法获取 |
| 个人手机/微信 | 经员工同意查看 | 未经同意不得强制查看个人手机（→ 可能侵犯隐私）|
| 财务凭证/合同 | 调取原件或复印件 | 正常的工作文件调取 |
| 监控录像 | 安保部门提供 | 公共区域监控 → 合法（需遵守《个人信息保护法》告知义务）；更衣室/卫生间 → 非法 |
| 录音 | 参与谈话的一方录制的录音 | 见证据规则skill——正常商谈中的秘密录音一般不视为非法（《民事证据规定》第90条 `[PKULaw]`） |
| 证人陈述 | 访谈记录 | 证人应自愿陈述，不得胁迫或诱导 |

### 证据保全

```
证据保全清单

| 序号 | 证据名称 | 证据形式 | 保管人 | 保管方式 |
|------|---------|---------|-------|---------|
| 1 | | | | 原件密封保存/复印件/电子文件MD5校验 |
```

## Step 4 — 访谈

### 访谈顺序

先外围后核心：
1. 举报人/信息来源（了解具体指控内容和背景）
2. 一般证人（了解相关事实，不涉及定性判断）
3. 被调查人的上级/同事（了解工作表现和相关背景）
4. 被调查人（最后访谈——让其有机会充分回应）

### 被调查人访谈要点

- 访谈前已掌握足够的事实材料——不要在证据不足时仓促访谈被调查人
- 告知被调查人：调查的目的和范围（但不透露全部证据）
- 听取被调查人的陈述和申辩——《劳动合同法》要求在作出处分决定前应当听取员工的陈述和申辩
- 记录访谈（最好由被调查人签字确认）

### 访谈记录格式

```
访谈记录

时间：YYYY年MM月DD日 HH:MM
地点：[会议室/线上]
访谈人：[姓名/职务]
记录人：[姓名/职务]
被访谈人：[姓名/职务/部门]

被访谈人确认：
本人已被告知本次访谈的目的，自愿就相关事项进行陈述。
本人保证所述内容真实。

访谈内容：
Q: [问题]
A: [回答]

[被访谈人补充说明]：

被访谈人签名：        日期：
访谈人签名：          日期：
```

## Step 5 — 事实认定

### 事实认定标准

| 认定 | 标准 |
|------|------|
| 查证属实 | 有充分、确凿的证据支持，证据之间能够相互印证，排除合理怀疑 |
| 无法认定 | 证据不足以支持指控，或正反证据相当 |
| 查证不属实 | 有充分证据证明指控不成立 |

### 逐一认定

对举报/调查涉及的每项事实逐一认定：

```
事实认定表

| 序号 | 指控事实 | 证据支撑 | 证据评价 | 认定结论 |
|------|---------|---------|---------|---------|
| 1 | [具体指控] | [证据1/证据2/...] | [充分/尚不充分/矛盾] | [属实/无法认定/不属实] |
```

## Step 6 — 法律与制度评价

对已认定属实的行为进行评价：

- **是否违反公司内部规章制度？** 引用制度的具体条款。
- **是否违反劳动合同/保密协议/竞业限制协议的约定义务？** 引用合同/协议的具体条款。
- **是否违反法律规定？** 引用法律的具体条款 `[PKULaw]`。
- **是否构成严重违纪/严重失职？** 《劳动合同法》第39条第2、3项 `[PKULaw]`——严重违纪或严重失职、营私舞弊，给用人单位造成重大损害的，可单方解除劳动合同且无需支付经济补偿。
- **是否需要向公安机关移送？** 如果涉及刑事犯罪，建议是否报案。

## 重要警示

- 调查报告本身在中国法下不享有美国式的"attorney work product"保护——如果在后续劳动争议或刑事程序中成为证据，报告内容可能被审查。报告的语言应客观、专业，不得包含主观揣测或侮辱性表述。
- 如果调查由外部律师主导并以律师名义出具——根据《律师法》第38条 `[PKULaw]` 保密义务的保护范围更广。在条件允许的情况下，建议由律师主导敏感调查以获取较强的保密保护。
- 调查过程和结论须遵守以下原则：
  - **程序正当：** 给被调查人充分的陈述和申辩机会
  - **比例原则：** 调查手段与被调查行为的严重程度相匹配
  - **保密原则：** 调查信息在调查期间和调查后均应严格保密
  - **客观公正：** 调查组应排除偏见和利益冲突

**参考法律渊源:**
- 《中华人民共和国劳动合同法》第39条 `[PKULaw]`
- 《中华人民共和国刑法》第163、164、219、271、272条 `[PKULaw]`
- 《中华人民共和国反不正当竞争法》第7、9条 `[PKULaw]`
- 《中华人民共和国公司法》(2023修订) 第180-185条 `[PKULaw]`
- 《中华人民共和国个人信息保护法》`[PKULaw]`
- 《最高人民法院关于民事诉讼证据的若干规定》(2019修正) `[PKULaw]`
