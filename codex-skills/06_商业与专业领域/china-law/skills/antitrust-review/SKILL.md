# /antitrust-review

1. 读取 `~/.claude/plugins/config/claude-for-legal/china-law/CLAUDE.md` → 管辖与法律框架。
2. 确定审查类型和适用标准。
3. 通过北大法宝 MCP 检索相关法律法规 `[PKULaw]`。
4. 进行反垄断风险评估。
5. 出具附风险分级和应对建议的审查报告。

---

## Purpose

中国反垄断执法已进入常态化——市场监管总局（SAMR）对经营者集中未依法申报的处罚力度持续加强（2024年多起处罚金额超百万元），垄断协议和滥用市场支配地位案件罚款可达上一年度销售额的1%-10%。本技能提供中国反垄断法律框架下的系统审查方法和风险应对方案。

## 审查类型

| 类型 | 适用场景 | 主要法律依据 |
|------|---------|-------------|
| 经营者集中审查 | 并购、合资、资产收购、通过合同等方式取得控制权 | 《反垄断法》第25-34条 `[PKULaw]` |
| 垄断协议审查 | 横向/纵向价格固定、市场分割、限制产量、联合抵制、转售价格维持 | 《反垄断法》第16-21条 `[PKULaw]` |
| 滥用市场支配地位 | 不公平高价/低价、掠夺性定价、拒绝交易、限定交易、搭售、差别待遇 | 《反垄断法》第22-24条 `[PKULaw]` |
| 行政性垄断 | 行政权力排除、限制竞争 | 《反垄断法》第39-45条 `[PKULaw]` |
| 反垄断合规 | 企业反垄断合规体系建设和风险评估 | 《经营者反垄断合规指南》`[PKULaw]` |
| 反垄断调查应对 | 应对SAMR的反垄断调查（黎明突击检查/正式调查） | 《反垄断法》第46-50条 `[PKULaw]` |

## Step 1 — 确定审查类型与收集信息

一次性询问：

> 请提供：
>
> - **审查类型：** 经营者集中/垄断协议/滥用市场支配地位/行政性垄断/合规体系建设/调查应对？
> - **交易/行为描述：** 涉及哪些当事方？什么行业？什么行为？
> - **营业额数据：** 各当事方上一会计年度的中国境内和全球营业额（经营者集中审查必需）。
> - **相关市场界定：** 相关产品市场和地域市场的初步描述。
> - **上传文件：** 交易文件/合资协议/合作协议/分销合同/定价政策/合规手册等。

## Step 2 — 经营者集中审查

### 2.1 申报标准

根据《国务院关于经营者集中申报标准的规定》(2024修订) 第3条 `[PKULaw]`：

| 标准 | 条件 | 是否触发申报 |
|------|------|------------|
| 标准一 | 参与集中的所有经营者上一会计年度**全球**营业额合计超过 **120亿元**，且其中至少两个经营者上一会计年度**中国境内**营业额均超过 **8亿元** | [是/否] |
| 标准二 | 参与集中的所有经营者上一会计年度**中国境内**营业额合计超过 **40亿元**，且其中至少两个经营者上一会计年度**中国境内**营业额均超过 **8亿元** | [是/否] |

**⚠️ 2024年修订新增:** 对于大型企业的"扼杀式收购"（killer acquisitions），即使未达到上述申报标准，如果集中一方上一会计年度中国境内营业额超过1000亿元，且集中可能对市场竞争产生排除、限制影响，SAMR可以要求申报。

### 2.2 控制权判断

判断是否构成"经营者集中"（《反垄断法》第25条 `[PKULaw]`）：

| 情形 | 示例 |
|------|------|
| 合并 | A公司与B公司合并为新公司C |
| 取得控制权 | A公司收购B公司51%股权 |
| 取得决定性影响 | A公司通过合同/人事安排对B公司的生产经营获得决定性影响（即使持股<50%） |

### 2.3 申报程序

```
经营者集中申报时间线

        签署交易文件
             │
             ▼
    ┌── 是否达到申报标准？ ──→ 否 ──→ 无需申报
    │        │ 是
    │        ▼
    │   准备申报材料（通常需要4-8周）
    │        │
    │        ▼
    │   向SAMR提交申报（正式立案）
    │        │
    │        ▼
    │   ┌─ 简易程序（30日，初步审查）
    │   │     │ 无竞争关切
    │   │     ▼ 或进入进一步审查
    │   ├─ 进一步审查（90日）
    │   │     │ 无竞争关切
    │   │     ▼ 或进入延期审查
    │   └─ 延期审查（60日，可再延长）
    │             │
    │             ▼
    │   SAMR决定：批准 / 附条件批准 / 禁止
    │
    ▼
   交割（不得在批准前交割 — "抢跑"罚则）
```

### 2.4 未依法申报的后果

| 处罚类型 | 依据 | 幅度 |
|---------|------|------|
| 罚款 | 《反垄断法》第58条 `[PKULaw]` | 处**500万元以下**罚款（具有排除限制竞争效果的，处上一年度销售额**10%以下**罚款） |
| 责令恢复原状 | 《反垄断法》第58条 | 责令停止实施集中、限期处分股份或者资产、限期转让营业等 |
| 声誉影响 | — | 处罚记录公开，影响后续交易审查 |

## Step 3 — 垄断协议审查

### 3.1 横向垄断协议（竞争者之间）

《反垄断法》第17条 `[PKULaw]` 禁止具有竞争关系的经营者达成以下协议：

| 类型 | 典型行为 | 红线程度 |
|------|---------|---------|
| 固定价格 | 统一涨价通知、价格同盟、最低价格协议 | 🔴 核心卡特尔 |
| 限制产量/销量 | 限产保价协议 | 🔴 核心卡特尔 |
| 分割市场 | 划分销售区域或客户群体 | 🔴 核心卡特尔 |
| 限制新技术/新产品 | 联合抵制新技术或新产品 | 🔴 核心卡特尔 |
| 联合抵制 | 共同拒绝与特定经营者交易 | 🔴 核心卡特尔 |

**⚠️ 核心卡特尔不适用"安全港"豁免，也不适用"效果轻微"抗辩。**

### 3.2 纵向垄断协议（上下游之间）

《反垄断法》第18条 `[PKULaw]`：

| 类型 | 法律态度 | 说明 |
|------|---------|------|
| 转售价格维持（固定转售价/最低转售价） | 🔴 原则禁止 | 经营者须证明协议不具有排除、限制竞争效果 |
| 限制销售区域/客户 | 🟡 适用合理原则 | 结合市场份额、市场结构评估 |
| 排他性交易 | 🟡 适用合理原则 | 结合市场份额评估（安全港：<15%） |

### 3.3 安全港制度

《禁止垄断协议规定》`[PKULaw]`：

| 协议类型 | 安全港标准 | 效果 |
|------|-----------|------|
| 横向协议（非核心卡特尔） | 各方合计市场份额 **<15%** | 推定不具有排除限制竞争效果 |
| 纵向协议 | 各方各自市场份额 **<15%** | 推定不具有排除限制竞争效果 |

**核心卡特尔（固定价格、限制产量、分割市场、联合抵制、限制新技术）不适用安全港。**

### 3.4 宽大制度（Leniency）

《反垄断法》第56条 + 《国务院反垄断委员会垄断案件经营者宽大制度适用指南》`[PKULaw]`：

| 申请顺序 | 减免幅度 | 条件 |
|---------|---------|------|
| 第一个申请 | 免除或**减少不低于80%** | 提供重要证据 + 全面配合 |
| 第二个申请 | 减少 **30%-50%** | 提供重要证据 |
| 第三个及以后 | 减少 **不超过30%** | 提供重要证据 |
| 胁迫其他经营者实施垄断协议的组织者 | 不适用免除，可酌情减轻 | — |

## Step 4 — 滥用市场支配地位评估

### 4.1 市场支配地位认定

《反垄断法》第23-24条 `[PKULaw]`：

**认定因素：**
- 经营者的市场份额和相关市场竞争状况
- 经营者控制市场的能力
- 经营者的财力和技术条件
- 其他经营者对该经营者的依赖程度
- 其他经营者进入相关市场的难易程度

**推定标准（可被推翻）：**

| 市场份额 | 推定 |
|---------|------|
| 一个经营者 ≥ **1/2** | 推定具有市场支配地位 |
| 两个经营者合计 ≥ **2/3** | 推定共同支配（但其中任一<10%的除外） |
| 三个经营者合计 ≥ **3/4** | 推定共同支配（但其中任一<10%的除外） |

### 4.2 滥用行为类型

| 滥用类型 | 典型表现 | 认定要点 |
|---------|---------|---------|
| 不公平高价/低价 | 价格远高于/低于合理水平 | 需进行价格比较分析 |
| 掠夺性定价 | 低于成本销售排挤竞争者 | 需证明排挤意图 + 低于成本 |
| 拒绝交易 | 无正当理由拒绝供应或接入 | 特别适用于必需设施 |
| 限定交易 | 要求交易相对人只能与其交易 | "二选一" |
| 搭售 | 捆绑销售无关产品 | 需证明无正当理由 |
| 差别待遇 | 对同等交易设置不同价格 | 大数据杀熟（平台经济特别关注） |

### 4.3 平台经济特别规定

《国务院反垄断委员会关于平台经济领域的反垄断指南》`[PKULaw]`：

- 平台经济相关市场界定更复杂（多边市场、网络效应、锁定效应）
- 算法合谋可构成垄断协议（即使无明示沟通）
- 数据垄断行为纳入滥用审查范围
- "二选一"、自我优待（self-preferencing）是执法重点

## Step 5 — 行政性垄断审查

《反垄断法》第39-45条 `[PKULaw]`：

| 行为类型 | 示例 | 救济途径 |
|---------|------|---------|
| 限定交易 | 地方政府指定特定供应商/服务商 | 向上级机关举报 + SAMR建议处理 |
| 地区封锁 | 限制外地商品进入本地市场 | 同上 |
| 排斥/限制外地经营者参加招投标 | 设定不合理的地域条件 | 同上 |
| 强制经营者从事垄断行为 | 行政指导行业协会统一定价 | 同上 |
| 制定含垄断内容的规章 | 印发含排除限制竞争内容的文件 | 公平竞争审查制度（前置） |

## Step 6 — 反垄断调查应对

### 6.1 调查类型

| 调查类型 | 触发方式 | 法律依据 |
|---------|---------|---------|
| 黎明突击检查（Dawn Raid） | SAMR自行发起 | 《反垄断法》第46条 `[PKULaw]` |
| 正式立案调查 | 举报/自行发现 | 《反垄断法》第46-49条 |
| 承诺中止调查 | 被调查人承诺整改 | 《反垄断法》第51条 |

### 6.2 黎明突击检查应对清单

```
黎明突击检查（Dawn Raid）应对清单

□ 检查人员出示执法证件和检查通知书
□ 核实执法人员身份（人数、单位）
□ 立即通知外部律师到场
□ 指派专人全程陪同执法人员（每名执法人员至少一名陪同）
□ 详细记录被检查/复制的文件和信息系统
□ 记录执法人员提出的问题和员工回答
□ 不得：销毁/藏匿文件、阻止进入、在律师到前拒绝检查
□ 不得：指导员工回答问题或要求员工保持沉默
□ 前台/安保人员：已培训，知道检查通知书的样貌
□ 建立"检查应对团队"名单（律师+法务+IT+行政）随时待命
```

### 6.3 承诺制度

《反垄断法》第51条 `[PKULaw]`：
- 被调查人可在调查期间向SAMR作出整改承诺
- SAMR可中止调查（non-settlement）——不是和解，是"整改承诺"
- 如被调查人履行承诺，SAMR可终止调查
- 不适用于核心卡特尔

### 风险分级

| 级别 | 经营者集中 | 垄断协议 | 滥用支配地位 |
|------|----------|---------|------------|
| 🔴 阻断性 | 应申报未申报且已交割；具有排除限制竞争效果 | 核心卡特尔；转售价格维持无豁免可能 | 市场支配地位+典型滥用行为+无正当理由 |
| 🟠 高 | 触发申报、有竞争关切 | 纵向协议超安全港+限制效果明显 | 可能认定支配地位+行为存疑 |
| 🟡 中 | 接近申报门槛需持续监控 | 合作协议可能被解释为横向协调 | 市场份额较高但滥用不确定 |
| 🟢 低 | 明显不达标 | 合作协议明确在安全港内 | 市场份额低、行为有正当理由 |

## 注意事项

- 经营者集中申报前不得交割（standstill obligation）——"抢跑"（gun-jumping）处罚日趋严厉（2024年起SAMR加强了这方面执法）。
- 营业额计算口径遵照《国务院关于经营者集中申报标准的规定》——注意关联企业营业额的合并计算规则。
- 简易程序并非"简化审查"——是"简易标准"，即SAMR初步认为交易明显不具有竞争关切，但这仍需要完整的申报材料。
- 垄断协议的宽大申请有"抢跑"效应——第一个申请者获得最大减免，须尽早决策。
- 涉及国有企业或敏感行业的经营者集中可能涉及国家安全审查，需与反垄断审查平行推进。
