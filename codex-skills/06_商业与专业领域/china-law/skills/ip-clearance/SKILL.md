# /ip-clearance

对拟使用的商标、技术方案或作品进行中国法下的在先权利冲突初步排查。

## 指令

### 1. 事务上下文
- IP类型？（商标/发明/实用新型/外观设计/著作权/商业秘密）
- 使用范围和地域？
- 相关行业和商品/服务类别？

### 2. 核心法律框架

#### 2.1 商标（《商标法》(2019修正)）

**注册原则与先申请原则（第4条、第31条 `[PKULaw]`）：**
- 中国商标制度以注册为基础，先申请原则
- 同日申请 → 使用在先优先

**不得注册的标志（第9-12条）：**
- 缺乏显著特征
- 仅有本商品的通用名称、图形、型号
- 仅直接表示商品的质量、主要原料、功能、用途等特点

**不予注册的相对理由（第13条、第15条、第32条）：**
- 复制、摹仿、翻译他人未在中国注册的驰名商标 → 相同/类似商品 → 不予注册
- 复制、摹仿、翻译他人已在中国注册的驰名商标 → 不相同/不类似商品 → 不予注册（第13条）
- 代理人/代表人未经授权抢注
- 损害他人在先权利（商号权、著作权、外观设计专利权、姓名权、肖像权）
- 以不正当手段抢注他人已经使用并有一定影响的商标

**三年不使用撤销（第49条 `[PKULaw]`）：**
注册商标无正当理由连续三年不使用的 → 任何单位或个人可向商标局申请撤销

→ 对于存量商标的稳定性判断意义重大

#### 2.2 专利（《专利法》(2020修正)）

**专利权类型（第2条 `[PKULaw]`）：**
- 发明：对产品、方法或其改进提出的新的技术方案
- 实用新型：对产品的形状、构造或其结合提出的适于实用的新的技术方案
- 外观设计：对产品的整体或局部的形状、图案或其结合以及色彩与形状、图案的结合所作出的富有美感并适于工业应用的新设计

**新颖性（第22-23条）：**
- 发明和实用新型：不属于现有技术，无抵触申请
- 外观设计：不属于现有设计，无抵触申请

**强制许可（第53-54条 `[PKULaw]`）：**
- 专利权人自授权之日起满3年且申请日起满4年，无正当理由未实施或未充分实施
- 国家紧急状态或非常情况或为公共利益目的

**诉讼时效（专利纠纷司法解释修正后第23条 `[PKULaw]`）：**
侵犯专利权诉讼时效为3年

#### 2.3 著作权（《著作权法》(2020修正)）

**保护范围（第2条 `[PKULaw]`）：**
- 中国公民、法人或非法人组织的作品 → 不论是否发表，依本法享有著作权
- 外国人、无国籍人的作品 → 根据所属国或经常居住地国与中国签订的协议或共同参加的国际条约

**保护对象（第3条）：**
文学、艺术和科学领域内具有独创性并能以一定形式表现的智力成果

**侵权赔偿（第54条 `[PKULaw]`）：**
- 实际损失 / 违法所得 / 权利使用费 → 赔偿
- 故意 + 情节严重 → 1倍以上5倍以下赔偿
- 以上均难以计算 → 法定赔偿500元至500万元
- 含合理开支（维权费用）

#### 2.4 知识产权惩罚性赔偿（惩罚性赔偿司法解释 `[PKULaw]`）

**故意的认定（第3条）：**
- 经通知、警告后仍继续侵权
- 被告或其法定代表人与原告有关联关系且接触过涉案知识产权
- 存在劳动、合作、许可等关系且接触过
- 实施盗版、假冒注册商标行为

**情节严重的认定（第4条）：**
- 因侵权被处罚后再次侵权
- 以侵害知识产权为业
- 伪造、毁坏或隐匿侵权证据
- 侵权获利或权利人受损巨大

### 3. 检索策略

1. **商标:** 通过CNIPA商标数据库和市场主体登记系统检索相同/近似标识
2. **专利:** 通过专利数据库检索相关技术领域的现有专利
3. **著作权:** 检索作品登记记录和公开的同类作品
4. **案例:** 通过北大法宝检索相关侵权判例 `[PKULaw]`

### 4. 严重度评估

| 严重度 | 标准 |
|-------|------|
| 🔴 阻断性 | 发现高度近似的在先注册商标（相同类别）或有效专利；存在已有侵权判决 |
| 🟠 高 | 发现可能构成混淆的近似商标；技术方案可能落入他人专利权保护范围 |
| 🟡 中 | 存在未知的在先权利；需进一步专业检索 |
| 🟢 低 | 未发现冲突；建议注册/申请以确权 |

### 5. 输出

排查结果 + 风险分级 + 法律依据 + 建议行动（申请/修改/取得许可/放弃）+ 决策树。
