# /environmental-compliance

1. 读取 `~/.claude/plugins/config/claude-for-legal/china-law/CLAUDE.md` → 管辖与法律框架。
2. 确定审查范围（全面/项目/处罚应对）。
3. 通过北大法宝 MCP 检索适用法规 `[PKULaw]`。
4. 逐项审查环境合规状况。
5. 出具附风险分级和整改建议的合规审查报告。

---

## Purpose

中国环境法律体系在过去十年中发生了结构性变化——从2015年《环境保护法》修订引入"按日连续处罚"开始，到排污许可制改革、环境影响评价制度改革、碳排放交易制度建设、环境公益诉讼制度成熟。环境刑事责任门槛持续降低（污染环境罪由结果犯向行为犯转型）。环境合规已从"软问题"变为并购和投资中的"硬条件"。本技能提供中国环境法律框架下的系统合规审查方法和风险评估。

## 审查类型

| 类型 | 适用场景 |
|------|---------|
| 全面环境合规审查 | 企业年度合规自检、ESG审计环境部分 |
| 并购/投资尽调环境部分 | 目标公司环境法律风险评估 |
| 新建项目环境合规 | 建设项目EIA、排污许可、竣工环保验收 |
| 环保处罚应对 | 收到环保处罚后的应对策略分析 |
| 碳合规审查 | 碳排放配额管理、碳交易合规 |

## Step 2 — 环境合规审查清单

### 2.1 环境影响评价（EIA）

《环境影响评价法》(2018修正) `[PKULaw]`：

| 审查项 | 合规要点 | 依据 |
|-------|---------|------|
| 是否需要环评 | 建设项目是否属于《建设项目环境影响评价分类管理名录》中的报告书/报告表/登记表类别 | 《环评法》第16条 |
| 环评文件等级 | 是否编制了对应等级的环评文件（报告书>报告表>登记表） | 《环评法》第16条 |
| 环评批复 | 是否取得了生态环境部门的环评批复（报告书/报告表项目须审批） | 《环评法》第22条 |
| 重大变动 | 建设过程中如有重大变动，是否重新报批环评 | 《环评法》第24条 |
| 5年失效期 | 环评批复后超过5年未开工建设的，是否重新审核 | 《环评法》第24条 |

**⚠️ "未批先建"后果：**
- 责令停止建设 + 罚款（总投资额**1%-5%**）
- 可责令恢复原状（《环评法》第31条）
- 直接负责的主管人员和其他直接责任人员可被处**5-20万元**罚款（2020年修正新增）

### 2.2 竣工环保验收

《建设项目竣工环境保护验收暂行办法》`[PKULaw]`：

| 要点 | 内容 |
|------|------|
| 验收时间 | 竣工后、正式投产前 |
| 验收方式 | **企业自主验收**（2017年改革后，由审批制改为企业自主验收制） |
| 验收合格标准 | 实际排放符合环评和批复标准 |
| 信息公开 | 验收报告应在全国建设项目竣工环保验收信息平台公示 |
| 未验先投后果 | 责令限期改正 + 罚款**20-100万元**；直接责任人**5-20万元** |

### 2.3 排污许可

《排污许可管理条例》(2021) `[PKULaw]`：

| 审查项 | 合规要点 |
|-------|---------|
| 持证排污 | 是否取得排污许可证（所有固定污染源均须持证排污） |
| 许可证有效期 | 是否在有效期内（5年） |
| 排放口合规 | 排污口的数量、位置、排放方式是否与许可证一致 |
| 排放标准 | 排放浓度和排放量是否在许可证核定范围内 |
| 自行监测 | 是否按许可证要求开展自行监测并保存原始记录 |
| 执行报告 | 是否按时提交季度/年度执行报告 |
| 信息公开 | 是否公开自行监测数据 |

**无证排污后果：** 罚款**20-100万元**；情节严重的，责令停业、关闭。

### 2.4 水污染防治

《水污染防治法》(2017修正) `[PKULaw]`：

| 审查项 | 要点 |
|-------|------|
| 废水排放标准 | 是否符合行业排放标准或综合排放标准 |
| 排污口设置 | 是否在饮用水水源保护区设置排污口（禁止） |
| 水污染物总量控制 | 是否超过核定的水污染物排放总量 |
| 雨水/污水分离 | 厂区是否实现雨污分流 |
| 初期雨水收集 | 是否收集初期雨水（前15分钟） |
| 地下水监测 | 是否按要求开展地下水水质监测 |

### 2.5 大气污染防治

《大气污染防治法》(2018修正) `[PKULaw]`：

| 审查项 | 要点 |
|-------|------|
| 大气污染物排放标准 | 是否符合排放标准（有组织排放 + 无组织排放） |
| VOCs治理 | 含VOCs（挥发性有机物）物料的储存、转移和使用是否有密闭和收集措施 |
| 重污染天气应急 | 是否制定重污染天气应急减排方案 |
| 在线监测 | 重点排污单位是否安装大气污染物自动监测设备并与环保部门联网 |

### 2.6 固体废物与危险废物

《固体废物污染环境防治法》(2020修订) `[PKULaw]` — 2020年修正大幅加强处罚力度：

| 审查项 | 合规要点 |
|-------|---------|
| 危废名录 | 产生的废物是否属于《国家危险废物名录》 |
| 危废贮存 | 贮存场所是否满足"三防"（防渗漏、防流失、防扬散）+ 标识 |
| 危废转移联单 | 每次转移是否填写危险废物转移联单 |
| 危废处置合同 | 是否与有资质的危废处置单位签订处置合同 |
| 危废管理计划 | 是否制定并向环保部门备案 |
| 危废台账 | 是否如实记录危废产生、贮存、转移、处置的台账 |
| 危废贮存期限 | 贮存期限是否超过1年 |
| 一般工业固废 | 贮存、处置、综合利用是否合规 |

**⚠️ 2020年修订后的处罚升级：**
- 非法倾倒、堆放、丢弃、遗撒固体废物：罚款原为"1-10万元" → 现为所需处置费用**3-5倍**（所需处置费用不足20万元的，按20万元计算 — 即最低罚款60万元）
- 无证从事危废经营活动：**100-500万元**罚款
- 危废提供或委托给无证经营者：**60-300万元**罚款

### 2.7 土壤污染防治

《土壤污染防治法》(2019) `[PKULaw]`：

| 审查项 | 要点 |
|-------|------|
| 土壤污染重点监管单位 | 是否在名录中（化工/冶金/电镀/印染等行业） |
| 土壤自行监测 | 重点监管单位是否按要求开展土壤自行监测 |
| 土壤隐患排查 | 是否建立土壤污染隐患排查制度 |
| 有毒有害物质排放 | 是否报告年度排放情况 |
| 拆除活动 | 拆除设施前是否制定土壤污染防治工作方案 |

### 2.8 噪声污染防治

《噪声污染防治法》(2022年6月5日施行) `[PKULaw]`：

| 审查项 | 要点 |
|-------|------|
| 厂界噪声 | 是否符合《工业企业厂界环境噪声排放标准》 |
| 夜间生产 | 夜间（22:00-06:00）噪声是否符合夜间标准 |

## Step 3 — 环境刑事责任风险

《刑法》第338-346条 `[PKULaw]`：

| 罪名 | 法条 | 量刑 | 关键门槛 |
|------|------|------|---------|
| 污染环境罪 | 第338条 | 情节严重：3年以下；情节特别严重：3-7年；情节特别严重+致多人伤亡等：**7年以上** | "严重污染环境"（2020年修正后降低了入罪门槛） |
| 非法处置进口的固体废物罪 | 第339条 | 5年以下/5-10年/10年以上 | "将境外的固体废物进境倾倒、堆放、处置" |
| 污染环境单位犯罪 | 第346条 | 对单位罚金 + 直接责任人个人刑罚 | 双罚制 |

**⚠️ 污染环境罪入罪门槛持续降低：** 2016年司法解释将"严重污染环境"细化为18种具体情形，包括排放特定重金属超标3倍以上、在饮用水水源保护区等特定区域排放污染物等。

## Step 4 — 按日连续处罚制度

《环境保护法》第59条 `[PKULaw]` + 《环境保护主管部门实施按日连续处罚办法》：

```
按日连续处罚触发条件

    环保部门作出处罚决定 + 责令改正
              │
              ▼
    环保部门在30日内复查
              │
         ┌────┴────┐
         ▼         ▼
     已改正      未改正
         │         │
         ▼         ▼
      结束     按日连续处罚
               处罚金额 = 原处罚金额 × 复查确认
               未改正之日起至改正之日止的天数

⚠️ 无上限！单日罚款 × 持续天数 → 可达巨额金额
```

| 情形 | 处罚金额计算 |
|------|------------|
| 原处罚10万元，30天未改正 | 10万 × 30 = 300万元 |
| 原处罚20万元，180天未改正 | 20万 × 180 = 3600万元 |

## Step 5 — 环境公益诉讼风险

| 诉讼类型 | 原告 | 法律依据 |
|---------|------|---------|
| 环境民事公益诉讼 | 社会组织（在设区的市级以上民政部门登记+专门从事环保公益≥5年+无违法记录）+ 检察院（在无适格社会组织时） | 《环境保护法》第58条 `[PKULaw]` |
| 生态环境损害赔偿诉讼 | 省级/市地级政府（通过生态环境部门） | 《生态环境损害赔偿制度改革方案》`[PKULaw]` |

**公益诉讼救济范围：** 停止侵害、排除妨碍、消除危险、恢复原状、赔偿损失（含生态环境修复费用、服务功能损失等）、赔礼道歉。

## Step 6 — 碳排放与碳交易合规

《碳排放权交易管理暂行条例》(2024年5月1日施行) `[PKULaw]`：

| 审查项 | 要点 |
|-------|------|
| 纳入碳市场 | 企业是否属于全国碳市场覆盖行业（目前：发电行业；即将扩展至石化、化工、建材、钢铁、有色、造纸、航空） |
| 碳排放配额 | 是否获得碳排放配额并按时足额清缴 |
| 温室气体排放报告 | 是否每年编制并提交排放报告 |
| 排放核查 | 报告是否经第三方机构核查 |
| 未足额清缴后果 | 罚款：未清缴部分的市场均价**5-10倍** |

## 注意事项

- 环境法规的强制执行力度持续加强，且环境行政处罚信息依法公开——不仅是金钱成本，还有企业声誉和后续融资、上市的影响。
- 并购环境尽调中，历史遗留的土壤和地下水污染是最容易遗漏且成本最高的问题——做过电镀、化工、冶金的地块，无论停产多久，都应进行土壤和地下水调查。
- 危废管理中的"委托给无证经营者"责任是严格责任——即便委托时有证书，如发现是假证或过期证，产生者仍要承担责任。须核实受托方资质的真实性和有效性。
- 环境公益诉讼的被告不仅是污染者，也可能是对污染有监督责任或疏忽的母公司、实际控制人。
