# /legal-opinion

1. 读取 `~/.claude/plugins/config/claude-for-legal/china-law/CLAUDE.md` → 管辖与法律框架、实务领域。
2. 通过北大法宝 MCP 检索相关法律、行政法规、司法解释、部门规章 `[PKULaw]`。
3. 根据意见书类型确定分析框架。
4. 检索相关案例和监管实践。
5. 出具附引用来源和不确定标注的法律意见书初稿。

---

## Purpose

法律意见书是中国律师的核心输出物——它是律师基于法律知识，对特定事项的法律状态、法律风险和法律后果作出的专业判断。与律师函（对外发送，要求对方作为或不作为）不同，法律意见书通常是应委托人要求出具的内部或交易用文书。

## 法律意见书类型

| 类型 | 用途 | 典型场景 |
|------|------|---------|
| 交易结构法律意见 | 分析交易安排的法律可行性和风险 | 并购、投资、资产转让、重组 |
| 合规评估意见 | 评估特定行为的合规性 | 数据合规、出口管制、劳动用工、环境合规 |
| 争议解决方案分析 | 分析争议的法律态势和解决方案 | 诉讼/仲裁前景分析、和解策略 |
| 上市/融资法律意见 | 就公司上市/融资出具法律意见 | IPO、增发、债券发行 |
| 基准法分析 | 新型业务模式的法律适用分析 | 新业务的法律定性、监管适用判断 |
| 合同法律意见 | 就合同条款的法律效果出具意见 | 复杂交易的合同结构和条款分析 |
| 咨询答复意见 | 就委托人提出的具体法律问题答复 | 日常法律咨询的正式书面答复 |

## Step 1 — 明确委托事项

一次性询问：

> 请提供：
>
> - **意见书类型：** 交易/合规/争议/上市/基准法/合同/咨询答复？
> - **委托事项：** 需要回答的具体法律问题是什么？（一个问题为核心，如果有多项请排序）
> - **背景事实：** 相关交易/事项的基本事实——谁、什么、何时、怎样的结构？
> - **已有文件：** 上传交易文件/合同/相关文件。
> - **时间要求：** 是否有时限？
> - **意见书用途：** 内部参考/提供给合作方/提交监管机构/作为交易文件附件？

## Step 2 — 法律检索

### 2.1 检索计划

根据委托事项确定检索范围：
- **法律渊源层级：** 法律 → 行政法规 → 部门规章 → 司法解释 → 地方性法规
- **检索关键词：** 从委托事项中提取核心法律概念
- **实务领域：** 跨领域的须逐领域检索

### 2.2 法律检索清单

通过北大法宝 MCP 逐一检索 `[PKULaw]`：

```
检索记录：
1. 检索词："[关键词]" → 结果：[N]条，相关法律：[列出]，核实状态：[已核实/待核实]
2. 检索词："[关键词]" → 结果：[N]条，相关司法解释：[列出]，核实状态：[已核实/待核实]
...
```

### 2.3 监管实践和案例参考

（如北大法宝 MCP 支持案例搜索或法律法规以外的内容）：
- 检索相关的行政处罚案例——了解监管机构的执法口径
- 检索相关的司法案例——了解法院的裁判倾向

如无法检索到，标注 `[监管/司法实践未检索 — 基于法条文字分析]`。

## Step 3 — 分析框架

根据意见书类型选择分析框架：

### 交易结构分析框架

| 分析维度 | 要点 |
|---------|------|
| 法律定性 | 交易的法律性质（买卖/投资/担保/信托/其他） |
| 主体资格 | 各方是否具有签署和履行交易的民事权利能力和行为能力 |
| 交易标的 | 标的是否合法、可转让、无权利负担 |
| 交易程序 | 是否需要审批/核准/备案/通知/公告 |
| 合同效力 | 交易文件是否存在无效/可撤销情形（《民法典》第144-154条 `[PKULaw]`） |
| 担保安排 | 担保是否有效设立（保证/抵押/质押） |
| 税务考量 | 增值税/企业所得税/印花税/契税等影响 |
| 监管影响 | 安全审查（外商投资/国家安全）/经营者集中审查/行业监管 |

### 合规评估分析框架

| 分析维度 | 要点 |
|---------|------|
| 适用法规识别 | 哪些法律法规适用于该行为 |
| 合规差距分析 | 当前状态与法规要求之间的差距 |
| 合规路径 | 达到合规状态的步骤和时间表 |
| 行政处罚风险 | 不合规可能面临的处罚（罚款幅度/责令改正/停业整顿/吊销许可） |
| 刑事责任风险 | 是否存在刑事追诉可能（单位犯罪/个人责任） |
| 补救措施 | 可以采取的纠正和减轻措施 |
| 持续合规建议 | 合规建设的制度性建议 |

### 争议分析框架

| 分析维度 | 要点 |
|---------|------|
| 法律关系 | 争议的法律定性 |
| 权利义务 | 各方权利和义务的法律来源 |
| 违约/侵权 | 是否存在违约或侵权行为，构成要件是否满足 |
| 抗辩事由 | 对方可能主张的抗辩 |
| 举证责任 | 各方承担举证责任的事项 |
| 诉讼/仲裁前景 | 胜败诉可能性的分析（不做百分比预测，只分析有利和不利因素） |
| 救济范围 | 可能获得的救济类型和范围 |
| 时效/期限 | 诉讼时效/仲裁时效/除斥期间检查 |

## Step 4 — 起草法律意见书

### 法律意见书标准结构

```
[律所LOGO/名称]

法律意见书

[年份]XX法意字第[编号]号

致：[委托人名称]

[委托人名称]（以下简称"贵司"）就 [委托事项概述] 事宜，
委托本所出具法律意见。本所根据贵司提供的文件资料（详见附件一：
审阅文件清单），依据中华人民共和国现行有效的法律法规，出具
本法律意见书。

一、引言与声明

（一）本法律意见书的范围限于 [限定范围——如：仅就中国法律问题
出具意见，不涉及其他司法管辖区的法律]。

（二）本所审阅了贵司提供的以下文件：
[审阅文件清单]

（三）本法律意见书基于以下假设出具：
1. 贵司提供的所有文件均为真实、完整、有效，复印件与原件一致；
2. 所有文件上的签字和印章均为真实的；
3. [其他必要假设]。

（四）本法律意见书仅供贵司 [内部参考/向XX提交/用于XXX目的]
使用，未经本所事先书面同意，不得向第三方披露（法律法规另有规定者
除外），且不得被用作其他目的。

二、背景事实

基于贵司提供的信息，本所了解到的基本事实如下：

[简要梳理事实]

三、适用法律

与本委托事项相关的主要法律法规包括：

1. 《[法律名称]》（[修正年份]修正，以下简称"XX法"）`[PKULaw]`
   第X条规定：...
2. 《[行政法规/司法解释]》`[PKULaw]`
   第Y条规定：...
3. ...

四、法律分析

（按委托事项的顺序逐项分析，每项分析对应一个问题，结构如：）

（一）关于[第一个问题]

1. 法律依据
[引用具体法条]

2. 事实适用
[将法律规则适用于已知事实]

3. 分析结论
[得出明确的法律判断或指出不确定性]

[如是/否判断，给出明确答案。如是带有不确定性的判断，指出不确定性
的来源和范围。]

（二）关于[第二个问题]

...

五、总体结论

综合上述分析，[概述主要结论和建议。使用明确的表述，对不确定之处
予以标注。]

六、关于重大不确定性的提示

[如果存在重大法律不确定性、法规不清晰、事实不完整等情况，
必须在此节明确提示。]

**建议**：[如果委托人需要采取进一步行动，在此给出具体建议。]

本法律意见书一式 [N] 份，具有同等法律效力。

[律所名称]（盖章）
律师：[律师姓名]（签名/签章）

日期：YYYY年MM月DD日
```

### 起草要求

1. **每个结论有法律出处。** "根据《XX法》第X条 `[PKULaw]`，..." 不能只说"根据相关法律规定"。
2. **不确定性须明确标注。** "由于[法规不清晰/事实不完整/缺乏司法实践]的原因，这一结论存在不确定性 `[审查]`。"
3. **区分法律判断和商业判断。** "这是一个法律上允许的安排。商业上的利弊不在本法律意见书的分析范围内。"
4. **不在法律意见中夹带事实。** 不确定的事实 → "如贵司所述"；确认的事实 → "根据[文件]所载"。
5. **意见书范围明确。** "本法律意见书仅就上述具体问题出具，不涵盖委托事项可能涉及的其他法律问题。"

## Step 5 — 内部审查

在提交委托人前，法律意见书须经以下内部审查：

```
法律意见书内部审查清单

□ 引用的所有法律条款均已通过北大法宝或其他一手来源核实
□ 法律条文为现行有效文本（未被修正/废止）
□ 所有结论均有法律依据支持 — 无法条支持的观点须标注 [分析性意见，无直接法条支持]
□ 不确定之处已充分披露 — [N]处不确定已标注
□ 意见书范围声明清晰完整
□ 假设前提已明确列出
□ 意见书限定用途声明已包含
□ 审阅文件清单已完整附录
□ 律师签名/律所盖章
□ 编号已登记归档
```

## 注意事项

- 法律意见书不是保险——律师不对意见书中的判断承担结果担保责任，但须尽到勤勉尽责和专业审慎的义务。
- 对于涉及非中国大陆法律的问题，须明确指明不在本意见范围内，并建议咨询对应法域的执业律师。
- 对于上市/融资等涉及第三方依赖的法律意见，须明确意见书出具基础、免责声明和限制条件。
- 一份法律意见书的权威性建立在其引用的法律条文准确、分析逻辑严密的基础上——不要用"我们认为""应当"等模糊用语掩盖分析空白。

**参考法律渊源:**
- 《中华人民共和国律师法》(2017修正) `[PKULaw]`
- 中华全国律师协会《律师执业行为规范》(2009修订) `[PKULaw]`
- 《律师事务所从事证券法律业务管理办法》`[PKULaw]`
