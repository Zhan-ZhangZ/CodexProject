# /labor-arbitration-guide

中国劳动争议仲裁程序指引——劳动仲裁是诉讼的强制前置程序。

## 指令

### 1. 事务上下文

- 争议类型？（确认劳动关系/劳动报酬/工伤待遇/解除争议/竞业限制/社保争议）
- 争议标的额？
- 是否经过调解？（企业调解/乡镇街道调解）
- 是否已申请仲裁？（时效/管辖确认）

### 2. 核心法律框架

#### 2.1 仲裁前置原则（《劳动争议调解仲裁法》第5条 `[PKULaw]`）

劳动争议处理程序：调解（可选） → 劳动仲裁（强制前置） → 法院诉讼

未经劳动仲裁，除法定例外情形外不得直接向法院起诉。

#### 2.2 仲裁时效（第27条 `[PKULaw]`）

- **一般时效：** 1年，自当事人知道或应当知道权利被侵害之日起计算
- **中断事由：** 主张权利 / 请求救济 / 对方同意履行 → 时效重新计算
- **中止事由：** 不可抗力或其他正当理由 → 时效继续计算
- **劳动关系存续期间拖欠劳动报酬：** 不受1年时效限制；但劳动关系终止后应在1年内提出

#### 2.3 仲裁管辖（第21条）

- 劳动合同履行地 或 用人单位所在地
- 双方分别向两地申请 → 劳动合同履行地优先

#### 2.4 仲裁程序

**申请与受理（第29条 `[PKULaw]`）：**
仲裁委员会收到申请后5日内决定受理/不予受理；不予受理或逾期未决定 → 可向法院起诉

**开庭与裁决（第43条）：**
受理后45日内作出裁决；案情复杂经批准延长不超过15日；逾期未裁决 → 可向法院起诉

**先予执行（第44条 `[PKULaw]`）：**
追索劳动报酬、工伤医疗费、经济补偿或赔偿金的案件，根据当事人申请可裁定先予执行

#### 2.5 一裁终局（第47条 `[PKULaw]`）

下列裁决为终局裁决，自作出之日起生效：
1. 追索劳动报酬、工伤医疗费、经济补偿或赔偿金，不超过当地月最低工资标准12个月金额
2. 因执行国家劳动标准在工作时间、休息休假、社会保险等方面发生的争议

**终局裁决的救济：**
- 劳动者不服 → 收到裁决书15日内向法院起诉（第48条）
- 用人单位不服 → 不可起诉，但可自收到裁决书30日内向仲裁委所在地中级法院申请撤销（第49条）

#### 2.6 用人单位申请撤销终局裁决（第49条 `[PKULaw]`）

理由限于：
1. 适用法律、法规确有错误
2. 劳动争议仲裁委员会无管辖权
3. 违反法定程序
4. 裁决所根据的证据是伪造的
5. 对方当事人隐瞒了足以影响公正裁决的证据
6. 仲裁员在仲裁该案时有索贿受贿、徇私舞弊、枉法裁决行为

撤销后 → 劳动者可在收到裁定书15日内向法院起诉

#### 2.7 举证责任（第6条 `[PKULaw]`）

与争议有关的证据属于用人单位掌握管理的 → 用人单位应当提供；不提供 → 承担不利后果

劳动争议司法解释(一)第44条 `[PKULaw]`：用人单位对解除劳动合同决定、减少劳动报酬决定、计算劳动者工作年限承担举证责任

#### 2.8 诉讼衔接（第50条 `[PKULaw]`）

非终局裁决 → 收到裁决书15日内向法院起诉 → 期满不起诉则裁决生效

### 3. 仲裁 vs. 诉讼关键差异

| 差异点 | 劳动仲裁 | 法院诉讼 |
|-------|---------|---------|
| 时效 | 1年 | 3年（民法典第188条） |
| 审限 | 45天（+15天延长） | 一审6个月（简易3个月） |
| 终局性 | 小额案件一裁终局 | 二审终审 |
| 举证 | 用人单位部分倒置 | 谁主张谁举证 |
| 费用 | 免费 | 按标的额收费 |

### 4. 输出

仲裁时效判断 + 管辖确定 + 一裁终局适用分析 + 举证责任分配 + 程序指引 + 裁决执行/起诉建议 + 决策树。
