# /social-insurance-qa

回答社会保险（五险一金）相关的法律问题。

## 指令

### 1. 事务上下文
- 问题涉及哪种保险？（养老/医疗/工伤/失业/生育/公积金）
- 用人单位所在地？（省级/设区的市级——各地缴费比例和基数标准不同）
- 劳动者工资水平？（是否在缴费基数上下限范围内？）

### 2. 核心法律框架

#### 2.1 五险制度（《社会保险法》第2条 `[PKULaw]`）

国家建立基本养老保险、基本医疗保险、工伤保险、失业保险、生育保险等社会保险制度，
保障公民在年老、疾病、工伤、失业、生育等情况下依法获得物质帮助的权利。

#### 2.2 缴费责任

| 保险种类 | 用人单位 | 劳动者 | 法律依据 |
|---------|--------|-------|---------|
| 基本养老保险 | ✓ | ✓ | 社保法第11条 |
| 基本医疗保险 | ✓ | ✓ | 社保法第23条 |
| 工伤保险 | ✓ | ✗ | 社保法第33条 |
| 失业保险 | ✓ | ✓ | 社保法第44条 |
| 生育保险 | ✓ | ✗ | 社保法第53条 |
| 住房公积金 | ✓ | ✓ | 住房公积金管理条例 |

*具体比例由各省级/设区的市级政府规定。*

#### 2.3 缴费基数

- 上限：当地职工月平均工资的300%
- 下限：当地职工月平均工资的60%（部分地区为当地最低工资标准）
- 基数：按劳动者本人上年度月平均工资确定

#### 2.4 工伤保险制度（《工伤保险条例》(2010修订) `[PKULaw]`）

**认定工伤的情形:**
- 在工作时间和工作场所内，因工作原因受到事故伤害
- 工作时间前后在工作场所内，从事与工作有关的预备性或收尾性工作受到事故伤害
- 在工作时间和工作场所内，因履行工作职责受到暴力等意外伤害
- 患职业病
- 因工外出期间，由于工作原因受到伤害或者发生事故下落不明
- 在上下班途中，受到非本人主要责任的交通事故或者城市轨道交通、客运轮渡、火车事故伤害
- 法律、行政法规规定应当认定为工伤的其他情形

**视同工伤:**
- 在工作时间和工作岗位，突发疾病死亡或者在48小时之内经抢救无效死亡
- 在抢险救灾等维护国家利益、公共利益活动中受到伤害
- 职工原在军队服役，因战、因公负伤致残，已取得革命伤残军人证，到用人单位后旧伤复发

**不得认定工伤:**
- 故意犯罪
- 醉酒或者吸毒
- 自残或者自杀
- 法律、行政法规规定的其他情形

**停工留薪期（《工伤保险条例》第33条 `[PKULaw]`）:**
- 一般不超过12个月
- 伤情严重或情况特殊可延长，但≤12个月（需经市级劳动能力鉴定委员会确认）
- 期内原工资福利待遇不变，由用人单位按月支付

**伤残待遇:**
- 1-4级：保留劳动关系，退出工作岗位，享受伤残津贴
- 5-6级：保留劳动关系，安排适当工作或发伤残津贴
- 7-10级：合同期满或劳动者提出解除 → 支付一次性工伤医疗补助金和一次性伤残就业补助金

**因工死亡待遇:**
- 丧葬补助金（6个月的统筹地区上年度职工月平均工资）
- 供养亲属抚恤金（按职工本人工资一定比例）
- 一次性工亡补助金（上一年度全国城镇居民人均可支配收入的20倍）

#### 2.5 未依法缴纳的法律后果

- **补缴 + 滞纳金:** 用人单位未按时足额缴纳 → 责令限期缴纳或补足，自欠缴之日起按日加收万分之五的滞纳金
- **行政罚款:** 逾期仍不缴纳 → 处欠缴数额1倍以上3倍以下罚款（《社会保险法》第86条）
- **赔偿损失:** 未缴纳工伤保险费 → 用人单位按工伤保险待遇标准支付费用（《工伤保险条例》第62条 `[PKULaw]`）
- **劳动者解除权:** 用人单位未依法为劳动者缴纳社会保险费 → 劳动者可解除劳动合同并要求经济补偿（《劳动合同法》第38条、第46条 `[PKULaw]`）

### 3. 输出

事实分析 + 适用法条 + 计算说明（如涉及待遇金额） + 合规建议/补救措施 + 决策树。
