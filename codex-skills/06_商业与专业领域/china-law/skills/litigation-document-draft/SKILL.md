# /litigation-document-draft

1. 读取 `~/.claude/plugins/config/claude-for-legal/china-law/CLAUDE.md` → 管辖与法律框架。
2. 通过北大法宝 MCP 检索相关实体法和程序法依据 `[PKULaw]`。
3. 确定文书类型和适用程序。
4. 收集案件信息并验证法律依据。
5. 起草诉讼文书初稿。

---

## Purpose

中国的民事诉讼文书有严格的格式要求和必备要素——遗漏一项可能导致立案被退回。本技能起草各类核心诉讼文书，确保格式合规、法律依据准确、事实陈述有证据支撑。

## 支持的文书类型

| 文书类型 | 适用程序 | 法律依据 |
|---------|---------|---------|
| 民事起诉状 | 一审 | 《民事诉讼法》第122-124条 `[PKULaw]` |

| 反诉状 | 一审 | 《民事诉讼法》第54条 `[PKULaw]` |
| 上诉状 | 二审 | 《民事诉讼法》第171-173条 `[PKULaw]` |

## Step 1 — 确定文书类型与收集信息

### 一次性询问

> 请提供：
>
> - **文书类型：** 起诉状/答辩状/反诉状/上诉状/再审申请书/保全申请？
> - **案件类型：** 合同纠纷/侵权纠纷/劳动争议/知识产权/公司纠纷/家事/其他？
> - **当事人信息：** 原告/被告/第三人（自然人：姓名、性别、出生年月、身份证号/职业、住所地、联系方式；法人：名称、统一社会信用代码、住所地、法定代表人、联系方式）
> - **管辖法院：** 是否已确定管辖法院？
> - **核心事实：** 发生了什么？关键时间和节点？
> - **诉讼请求：** 希望通过诉讼达到什么结果？（逐项列明）
> - 上传相关合同、证据材料等文件。

## Step 2 — 法律检索与管辖确认

### 2.1 实体法依据检索

通过北大法宝 MCP 检索案件涉及的实体法条款 `[PKULaw]`，确认：
- 现行有效文本
- 最新修正日期
- 与案件事实的适用关系

### 2.2 管辖确认

- **级别管辖：** 基层/中级/高级/最高法院？争议金额和案件类型是否满足级别管辖标准？
- **地域管辖：** 一般地域管辖（被告住所地）/专属管辖（不动产/港口作业/继承遗产）/协议管辖？
- **协议管辖检查：** 合同中是否有管辖条款约定了特定法院？

## Step 3 — 起草

### 3.1 民事起诉状

```
民事起诉状

原告：[姓名/名称]，[性别]，[出生年月日]，[身份证号/统一社会信用代码]，
      [职业/经营范围]，住所地：[地址]，联系方式：[电话]
     （如系法人或其他组织，应写明法定代表人/负责人姓名、职务）

被告：[同上格式]

第三人（如有）：[同上格式]

诉讼请求：
1. 请求判令被告[具体履行行为，如：支付货款人民币XX元]；
2. 请求判令被告[具体行为，如：承担违约金人民币XX元（按日万分之五
   自YYYY年MM月DD日计算至实际清偿之日止）]；
3. 请求判令被告承担本案全部诉讼费用[及其他维权费用，如律师费XX元、
   保全费XX元等]。
   （多被告时注明各被告承担的责任类型——连带/按份/补充/单独）

事实与理由：

一、基本事实

（按时间顺序叙述案件事实，每项关键事实后标注证据出处）

YYYY年MM月DD日，原告与被告签订了《XX合同》（证据1），约定：...

YYYY年MM月DD日，被告...（描述违约/侵权/其他行为）

...

二、法律依据

根据《中华人民共和国民法典》第XX条 `[PKULaw]`...

（如有多项法律依据，逐项列出并关联对应的事实和诉求）

综上所述，原告的诉讼请求有事实和法律依据，恳请贵院依法支持原告的全部诉讼请求。

此致
[法院名称]

附件：
1. 本起诉状副本[N]份；
2. 证据清单及证据材料（共[N]份）；
3. 原告身份证明文件复印件。

具状人：[签名或盖章]
日期：YYYY年MM月DD日
```

### 3.2 民事答辩状

```
民事答辩状

答辩人：[姓名/名称]，住所地：[地址]，联系方式：[电话]

因[原告姓名/名称]诉答辩人[案由]纠纷一案（案号：[...]），
现就原告的起诉提出答辩如下：

一、关于原告陈述的事实

（逐条回应原告的陈述，指出哪些事实属实、哪些不属实、哪些需要
补充完整——每项回应附证据）

二、关于原告主张的法律适用

（指出原告引用的法律条款与本案事实的差异，或提供不同的法律解释）

三、答辩意见

综上所述，[简要总结答辩观点]，恳请贵院查明事实，依法驳回原告
的[全部/部分]诉讼请求。

此致
[法院名称]

附件：
1. 本答辩状副本[N]份；
2. 证据清单及证据材料（共[N]份）；
3. 答辩人身份证明文件复印件。

答辩人：[签名或盖章]
日期：YYYY年MM月DD日
```

### 3.3 财产保全申请书

```
财产保全申请书

申请人：[姓名/名称]，住所地：[地址]，联系方式：[电话]
被申请人：[姓名/名称]，住所地：[地址]，联系方式：[电话]

申请事项：
请求贵院查封/扣押/冻结被申请人名下价值人民币[金额]元的财产。
具体为：
1. 冻结被申请人在[银行名称]的银行账户（账号：[已知账号或待法院查询]）；
2. 查封被申请人名下位于[地址]的不动产（不动产权证书号：[...]）；
3. [其他具体财产]。

事实与理由：
[简述案件基本事实和被申请人可能转移财产的情况]
若不对被申请人的财产及时采取保全措施，可能导致申请人的合法权益
受到难以弥补的损害/可能导致将来的判决难以执行。

因此，依据《中华人民共和国民事诉讼法》第103-105条之规定，
申请人特申请贵院对上述财产采取保全措施。申请人愿以[自有/第三方
提供]的[财产/现金/保函]提供担保。

此致
[法院名称]

附件：
1. 拟申请保全的财产线索；
2. [担保方式及证明文件]。

申请人：[签名或盖章]
日期：YYYY年MM月DD日
```

### 3.4 上诉状

```
民事上诉状

上诉人（原审[原告/被告]）：[姓名/名称]，住所地：[地址]
被上诉人（原审[被告/原告]）：[姓名/名称]，住所地：[地址]

上诉人因与被上诉人[案由]纠纷一案，不服[原审法院名称]于YYYY年
MM月DD日作出的[案号]民事判决书/裁定书，现依法提出上诉。

上诉请求：
1. 请求撤销[原审法院名称]（YYYY）XX民初字第XX号民事判决/裁定；
2. 请求依法改判[具体改判内容]或发回重审；
3. 本案一、二审诉讼费用由被上诉人承担。

上诉理由：

一、原审判决/裁定认定事实错误

（指出原审判决/裁定在事实认定上的具体错误，附证据说明）

二、原审判决/裁定适用法律错误

（指出原审判决/裁定法律适用上的错误，引用正确法律条款）

三、原审程序违法（如适用）

（如原审存在程序违法——应回避未回避、未合法传唤即缺席判决、
　超越诉讼请求裁判等——逐项列明）

综上所述，原审判决/裁定在事实认定和法律适用上均存在错误，
恳请贵院查明事实，支持上诉人的全部上诉请求。

此致
[上一级人民法院]

附件：
1. 本上诉状副本[1]份；
2. 原审判决书/裁定书复印件。

上诉人：[签名或盖章]
日期：YYYY年MM月DD日
```

## Step 4 — 证据目录编制

每份诉讼文书应附证据清单：

```
证据清单

| 序号 | 证据名称 | 证据来源 | 证据形式 | 证明对象 | 页码 |
|------|---------|---------|---------|---------|------|
| 1 | 《XX合同》 | 原告存档 | 书证（复印件） | 证明原被告之间存在合同关系及约定的权利义务 | 1-20 |
| 2 | 银行转账凭证 | XX银行打印 | 书证（复印件） | 证明原告于XXXX年XX月XX日向被告支付人民币XX元 | 21 |
| 3 | 微信聊天记录 | 原告手机截图 | 电子数据 | 证明被告于XXXX年XX月XX日承认欠款事实 | 22-23 |

证据提交人：[签名或盖章]
日期：YYYY年MM月DD日
```

## 注意事项

- 起诉状的事实陈述是后续诉讼中不能随意更改的立场——"禁反言"原则适用。
- 劳动争议必须先经过劳动仲裁（仲裁前置），对仲裁裁决不服方可向法院起诉。
- 起诉时应同时考虑是否需要申请财产保全——在送达起诉状副本给被告之前完成保全对防止转移财产至关重要。
- 起诉后诉讼时效中断——仅起诉但后来撤诉的，诉讼时效不中断（最高人民法院关于审理民事案件适用诉讼时效制度若干问题的规定）。
