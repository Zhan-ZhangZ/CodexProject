# /arbitration-guide

中国仲裁程序指引——从仲裁协议的效力到裁决的执行与挑战。

## 指令

### 1. 事务上下文
- 仲裁类型？（国内/涉外/外国裁决承认执行）
- 仲裁机构？
- 争议事项可仲裁性？
- 是否有仲裁协议？（书面/电子/合同条款/单独协议）

### 2. 核心法律框架

#### 2.1 仲裁协议要件（《仲裁法》(2025修订)）

仲裁协议应具备：
- 请求仲裁的意思表示
- 仲裁事项
- 选定的仲裁委员会

**书面形式要求：** 合同中的仲裁条款或以其他书面方式达成的请求仲裁的协议

**无效仲裁协议：**
- 约定的仲裁事项超出法律规定的仲裁范围
- 无/限制民事行为能力人订立的
- 一方采取胁迫手段订立的

#### 2.2 主要仲裁机构

- 中国国际经济贸易仲裁委员会（CIETAC）—— 总部北京
- 上海国际经济贸易仲裁委员会（SHIAC）
- 深圳国际仲裁院（SCIA）
- 北京仲裁委员会（BAC）
- 各地仲裁委员会

#### 2.3 仲裁裁决的撤销（《仲裁法》第71条 `[PKULaw]`）

当事人可向仲裁机构所在地的中级人民法院申请撤销裁决，理由限于：
1. 没有仲裁协议
2. 裁决的事项不属于仲裁协议的范围或仲裁机构无权仲裁
3. 仲裁庭的组成或仲裁的程序违反法定程序
4. 裁决所根据的证据是伪造的
5. 对方当事人隐瞒了足以影响公正裁决的证据
6. 仲裁员在仲裁该案时有索贿受贿、徇私舞弊、枉法裁决行为

→ 法院认定裁决违背公共利益的 → 应当裁定撤销

**涉外仲裁裁决的撤销（第83条 `[PKULaw]`）：**
- 没有仲裁协议
- 被申请人未得到指定仲裁员或进行仲裁程序的通知，或其他原因未能陈述意见
- 仲裁庭的组成或仲裁程序与仲裁规则不符
- 裁决事项不属于仲裁协议范围或仲裁机构无权仲裁
- 认定违背公共利益的 → 应当撤销

#### 2.4 仲裁裁决的不予执行

**报核制度（仲裁司法审查报核规定(2021修正)第3条 `[PKULaw]`）：**
非涉外涉港澳台仲裁司法审查案件 → 如拟以违背社会公共利益为由不予执行或撤销 → 应当向最高人民法��报核 → 待审核后方可裁定

**涉外仲裁不予执行的报核 →** 同样需要层报最高人民法院

#### 2.5 中止执行（仲裁裁决执行规定第7条 `[PKULaw]`）

- 被执行人申请撤销仲裁裁决并已被法院受理 → 应裁定中止执行
- 被执行人或案外人提出不予执行申请并提供适当担保 → 应裁定中止执行
- 中止执行期间 → 停止处分性措施
- 申请执行人提供充分有效担保请求继续执行的除外

#### 2.6 外国仲裁裁决的承认与执行（《纽约公约》 `[PKULaw]`）

中国是《承认及执行外国仲裁裁决公约》（1958年纽约公约）缔约国。

**中国对纽约公约的互惠保留声明：**
对在另一缔约国领土内作出的仲裁裁决的承认和执行适用该公约

**拒绝承认和执行的事由（NY Convention Article V）：**
- 仲裁协议当事人无行为能力或协议依适用法律无效
- 被申请人未获适当通知或未能申辩
- 裁决处理事项超出仲裁协议范围
- 仲裁庭组成或仲裁程序与协议或仲裁地法律不符
- 裁决尚未生效或已被撤销/停止执行
- 争议事项依中国法律不可仲裁
- 承认或执行违反中国公共政策

→ 法院审查范围限于纽约公约第5条（全国法院涉外商事海事审判工作座谈会纪要第106条 `[PKULaw]`）：对被申请人未主张或超出第5条第1款规定的法定事由 → 不予审查；第5条第2款 → 依职权审查

#### 2.7 国内仲裁 vs. 涉外仲裁（关键差异）

| 差异点 | 国内仲裁 | 涉外仲裁 |
|-------|--------|--------|
| 裁决撤销理由 | 6项法定理由（含证据伪造、隐瞒证据） | 仅限于程序问题（没有仲裁协议、未获通知/未能申辩、程序不符、超范围） |
| 证据实体审查 | 可审查（证据伪造、隐瞒证据） | 不可审查实体问题 |
| 撤销报核 | 拟撤销须报高院→最高院 | 拟撤销须报高院→最高院（维护涉外仲裁裁决的终局性） |
