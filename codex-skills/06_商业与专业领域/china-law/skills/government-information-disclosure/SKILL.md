# /government-information-disclosure

1. 读取 `~/.claude/plugins/config/claude-for-legal/china-law/CLAUDE.md` → 管辖与法律框架。
2. 确定政府信息公开事项类型和角色。
3. 通过北大法宝 MCP 检索适用法规 `[PKULaw]`。
4. 分析信息公开申请的可获性/第三方商业秘密保护/答复合法性。
5. 出具附策略建议和救济路径的分析报告。

---

## Purpose

《政府信息公开条例》(2019修订) 是中国政务公开和公民知情权的核心制度依据。企业运用政府信息公开作为尽调工具（了解目标企业的行政处罚、奖励补贴、资质许可）、监管动向（了解行业执法口径）、争议准备（获取对方当事人的相关许可与处罚信息）已在实践中被广泛接受。同时，当企业自身的商业秘密可能因公开申请而被披露时，第三方的异议和诉讼权也至关重要。本技能提供政府信息公开制度的系统分析方法。

## 适用场景

| 场景 | 角色 |
|------|------|
| 申请政府信息公开 | 企业/个人作为申请人，希望获取特定政府信息 |
| 对答复不服 | 申请被拒/部分公开/逾期不答复→复议或诉讼策略 |
| 第三方商业秘密保护 | 企业作为第三方，反对政府机关向申请人公开含商业秘密的信息 |
| 政府信息公开诉讼 | 信息公开行政诉讼的策略和论证 |
| 合规尽调 | 通过信息���开获取目标企业的合规信息 |

## Step 2 — 政府信息���开基本制度

### 2.1 公开主体

《政府信息公开条例》(2019修订) `[PKULaw]`：

| 主体 | 义务范围 |
|------|---------|
| 行政机关 | 制作或获取的政府信息（履行职责过程中制作或获取的） |
| 法律/法规授权的具有管理公共事务职能的组织 | 同上（如证监会/银保监会/行业协会） |
| 教育/卫生健康/供水/供电/供气/供热/环境保护/公共交通等与人民群众利益密切相关的**公共企事业单位** | 参照本条例执行（提供社会公共服务过程中制作/获取的信息） |

### 2.2 公开方式

| 方式 | 内容 | 特点 |
|------|------|------|
| 主动公开 | 行政法规/规章/规范性文件；机关职能/机构设置；财政预决算；行政事业性收费；行政许可/处罚/强制；公务员招考等 | 无需申请，机关自行公开 |
| 依申请公开 | 公民/法人/其他组织向行政机关申请获取相关政府信息 | 需提交申请，20个工作日答复（可延长20个工作日） |

## Step 3 — 申请的提出与答复

### 3.1 申请条件

《政府信息公开条例》第29条 `[PKULaw]`：

| 要素 | 要求 |
|------|------|
| 申请主体 | 公民、法人或其他组织（不限国籍——外国人在中国也可申请） |
| 申请目的 | **不要求**说明用途（除申请数量/频次明显超过合理范围外） |
| 申请形式 | 书面（含数据电文）或口头（由受理机关代为填写） |
| 申请内容 | 名称/文号 或 便于查询的特征性描述（不能仅凭关键词模糊描述） |

### 3.2 答复时限

| 阶段 | 时限 |
|------|------|
| 初步答复 | 收到申请之日起 **20个工作日**内 |
| 延长答复 | 需经负责人同意+告知申请人，延长不超过**20个工作日** |
| 征求第三方意见 | 征求第三方意见的时间不计入答复期限 |
| 补正 | 申请内容不明确需补正的，自收到申请之日起**7个工作日**内一次性告知申请人补正 |

### 3.3 答复类型

| 答复类型 | 适用情形 |
|---------|---------|
| 予以公开 | 信息不属于不予公开范围 |
| 部分公开 | 信息含有不应公开的内容但可作区分处理（可分割性原则） |
| 不予公开 | 信息属于法定不予公开范围（见Step 4） |
| 无法提供 | 信息不存在/不属于本机关制作或获取 |
| 不予处理 | 申请内容不明确经补正后仍不明确/重复申请/申请数量频次明显超过合理范围 |

## Step 4 — 不予公开的法定例外

### 4.1 不予公开的类型

《政府信息公开条例》第13-17条 `[PKULaw]`：

| 类型 | 内容 | 是否绝对 | 法条 |
|------|------|---------|------|
| **国家秘密** | 依法确定为国家秘密的信息 | 🔴 **绝对**不予公开 | 第14条 |
| **法律行政法规禁止公开** | 法律、行政法规禁止公开的政府信息 | 🔴 **绝对**不予公开 | 第14条 |
| **"三安全一稳定"** | 公开后可能危及国家安全、公共安全、经济安全、社会稳定的 | 🔴 **绝对**不予公开 | 第14条 |
| **商业秘密** | 涉及商业秘密，公开后可能损害第三方合法权益 | 🟡 **相对**不予公开（第三方同意或公共利益需要除外） | 第15条 |
| **个人隐私** | 涉及个人隐私，公开后可能损害第三方合法权益 | 🟡 **相对**不予公开（第三方同意或公共利益需要除外） | 第15条 |
| **内部事务信息** | 人事管理/后勤管理/内部工作流程等内部事务信息 | ✅ 可以不予公开 | 第16条 |
| **过程性信息** | 讨论记录/过程稿/磋商信函/请示报告等过程性信息 | ✅ 可以不予公开（但行政执法案卷信息除外） | 第16条 |
| **行政执法案卷信息** | 行政执法的案卷材料（当事人已获查阅权利） | ✅ 可以不予公开（法律/法规/规章另有规定除外） | 第16条 |
| **重复申请** | 同一申请人向同一机关就同一内容重复申请 | ❌ 无答复义务 | — |

### 4.2 商业秘密审查机制

```
第三方商业秘密保护程序

    申请人向行政机关申请公开某信息
              │
              ▼
    行政机关初步审查：信息是否可能涉及第三方商业秘密？
              │
         ┌────┴────┐
         ▼         ▼
     不涉及       可能涉及
         │         │
         │         ▼
         │   行政机关书面征求第三方意见
         │   （第三方在15个工作日内答复）
         │         │
         │    ┌────┴────┐
         │    ▼         ▼
         │  同意公开   不同意公开
         │    │         │ 但行政机关认为
         │    ▼         │ 不公开可能对公共
         │  予以公开    │ 利益造成重大影响
         │             │         │
         │        ┌────┴────┐    │
         │        ▼         ▼    │
         │    同意不公开  公共利益  │
         │        │       >隐私    │
         │        ▼         │    │
         │    不予公开       ▼    │
         │             予以公开   │
         │             （告知第三方 │
         │              理由和救济）│
         ▼                         │
    直接予以公开 ←─────────────────┘
```

**⚠️ 第三方应对策略：** 第三方在收到征求意见书后15个工作日内须提出书面意见，说明商业秘密的理由和依据（包括商业秘密的界定、公开后可能造成的损害）。答复越详细和有法律依据，行政机关采纳的概率越高。同时，应明确提出如果机关决定公开，请提前告知以给第三方留出申请行政复议/诉讼的时间。

## Step 5 — 对信息公开答复不服的救济

### 5.1 救济路径

```
对信息公开答复不服的救济路径

    ├── 投诉（向上一级行政机关或同级政府信息公开主管部门）
    │
    ├── 行政复议（60日内，向上一级行政机关）
    │       │
    │       └── 对复议决定不服→行政诉讼（15日内）
    │
    └── 行政诉讼（6个月内，直接向法院起诉）
```

### 5.2 行政复议和诉讼的常见诉点

| 诉点类型 | 具体主张 |
|---------|---------|
| 答复超期 | 行政机关超过20个工作日（+20日延长）未答复→违法的行政不作为 |
| 不予公开理由不成立 | 行政机关以"国家秘密/商业秘密/内部事务"等为由不予公开但未提供实质性论证 |
| 部分公开不合理 | 应分割的信息未分割，应将可公开部分予以公开 |
| 公开前未征求意见 | 涉及商业秘密/个人隐私的未征求第三方意见 |
| 告知"信息不存在"未尽检索义务 | 未进行合理检索即宣称信息不存在 |
| 告知"不属于本机关"未履行指引义务 | 未告知应向哪个机关申请 |
| 拒绝信息的形式 | 要求提供纸质文件被以电子邮件形式搪塞（或反之） |

## Step 6 — 企业运用信息公开的策略

### 6.1 尽调用途

| 信息类型 | 获取途径 |
|---------|---------|
| 行政处罚记录 | 向市场监管/税务/环保/应急等部门申请 |
| 行政许可/资质信息 | 向发证机关申请 |
| 政府补贴/奖励信息 | 向财政/科技/发改等部门申请 |
| 食品安全抽检结果 | 向市场监管部门申请 |
| 环境监测/排污数据 | 向生态环境部门申请 |
| 项目审批/备案信息 | 向发改/规划/住建等部门申请 |

### 6.2 争议准备用途

- 对方当事人的行政许可/资质信息 → 用于判断履约能力
- 对方的行政处罚/涉诉信息 → 用于尽调和争议策略
- 相关政府的规划/政策文件 → 用于法律分析中的背景

## 注意事项

- 政府信息公开是制度性权利，不是行政恩赐——《政府信息公开条例》赋予公民、法人和其他组织获取政府信息的法定权利。行政机关拒绝公开不是行政裁量，而是必须基于法定例外的法律判断。
- 商业秘密保护是双边的——企业在申请获取竞争对手/合作方的信息时，对方可主张商业秘密；企业自身也可能在第三方的公开申请中被通知。建议企业建立商业秘密清单，在对外提供信息给行政机关时标注商业秘密。
- "三安全一稳定"（国家安全、公共安全、经济安全、社会稳定）是弹性条款——行政机关自由裁量空间大，司法审查强度有限。有效挑战需论证具体而不仅是抽象的"三安全一稳定"。
- 过程性信息和内部事务信息在2019年修订中成为"可以不予公开"——增加了这两种例外类型，减少了行政机关需在个案中论证的压力。
- 行政执法案卷信息虽然行政机关可以不予公开，但相关当事人（被处罚人、被许可人）在行政程序中有案卷查阅权——不可混淆两者的申请主体。
