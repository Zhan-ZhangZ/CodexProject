# /demand-letter-draft

1. 读取 `~/.claude/plugins/config/claude-for-legal/china-law/CLAUDE.md` → 管辖与法律框架、保密与工作成果保护。
2. 通过北大法宝 MCP 检索相关法律依据 `[PKULaw]`。
3. 确认函件类型并收集必要信息。
4. 审查事实准确性、法律依据适当性，避免刑事风险（诽谤、敲诈勒索、损害商业信誉）。
5. 起草律师函初稿。

---

## Purpose

律师函是中国法律实务中最常用的对外文书之一——它是一封有法律分量的信。写得好，对方在收到后主动履行或谈判解决；写得不好，可能构成诽谤、名誉侵权甚至敲诈勒索（《刑法》第274条 `[PKULaw]`）。本技能在确保法律依据准确、事实陈述可靠的前提下起草函件，并在发送前标记所有需要律师确认的风险点。

## 律师函类型

| 类型 | 用途 | 核心法律依据 |
|------|------|------------|
| 催告函/催收函 | 催告履行付款义务、返还原物、履行合同 | 《民法典》合同编；债务到期催告中断诉讼时效 |
| 违约通知函 | 通知对方违约并要求限期改正或承担违约责任 | 《民法典》第577-594条 `[PKULaw]` |
| 侵权警告函/停止侵权函 | 要求停止侵权行为（知识产权、名誉权等） | 《民法典》侵权责任编；《专利法》《商标法》《著作权法》相关条款 `[PKULaw]` |
| 竞业限制违约函 | 通知前员工/前合作方违反竞业限制义务 | 《劳动合同法》第23-24条；《反不正当竞争法》第9条 `[PKULaw]` |
| 声明函/澄清函 | 就特定事项发表正式声明，澄清事实或法律立场 | 视具体事项而定 |
| 回复函 | 回应对方的律师函或主张 | 视具体事项而定 |

## Step 1 — 接收案件信息

一次性询问：

> 请提供以下信息：
>
> - **函件类型：** 催收/违约通知/侵权警告/竞业限制/声明函/回复函？
> - **我方当事人**（委托人）**名称和基本信息**。
> - **对方当事人名称、地址**。
> - **核心事实：** 发生了什么？什么时间？涉及哪些关键文件或证据？
> - **法律依据：** 你认为适用的主要法律条款是什么？
> - **诉求：** 你要求对方做什么？（付款/停止侵权/履行合同/赔礼道歉...）
> - **截止期限：** 你希望给对方多长时间回应？
> - **发送方式：** 快递/电子邮件/两者？
> - 上传相关合同、协议、侵权证据或其他文件。

## Step 2 — 法律检索与核实

### 2.1 法律依据检索

- 通过北大法宝 MCP 检索相关法律条款 `[PKULaw]`
- 确认法律条款的现行有效性和最新修正情况
- 如北大法宝不可用，以 `[模型知识 — 请核实]` 标注

### 2.2 律师函法律风险自检

**⚠️ 发送律师函本身可能产生法律后果——在起草前检查以下风险：**

| 风险 | 法律依据 | 防范措施 |
|------|---------|---------|
| 诽谤/名誉侵权 | 《民法典》第1024-1028条 `[PKULaw]` | 事实陈述须有证据支持；不使用侮辱性、贬损性语言；不在不必要范围内扩大传播 |
| 敲诈勒索 | 《刑法》第274条 `[PKULaw]` | 诉求须有合法权利基础；不得以"不给钱就举报/曝光"为要挟；威胁手段不能超出法律救济范围 |
| 损害商业信誉/商品声誉 | 《刑法》第221条 `[PKULaw]` | 不捏造并散布虚伪事实；寄送范围限于对方及相关方 |
| 虚假诉讼 | 《刑法》第307条之一 `[PKULaw]` | 不基于捏造的事实提起民事诉讼；律师函中引用的证据须真实 |
| 侵犯商业秘密 | 《反不正当竞争法》第9条 `[PKULaw]` | 律师函中不披露己方掌握的对方商业秘密 |
| 诉讼时效 | 《民法典》第195条 `[PKULaw]` | 催告函可以中断诉讼时效——发函时机本身就是策略 |
| 商业诋毁 | 《反不正当竞争法》第11条 `[PKULaw]` | 不编造传播虚假信息或误导性信息损害竞争对手的商业信誉 |

**🔴 关键红线：**
1. 不得以刑事举报为要挟索取财物 → 敲诈勒索
2. 不得向与争议无关的第三方（媒体、客户、合作伙伴）广泛散发 → 可能构成商业诋毁或名誉侵权
3. 不得在无证据支持的情况下使用"诈骗""犯罪""违法"等刑事定性词汇

## Step 3 — 事实审查

### 逐条核验

将律师函中的每一项事实主张与证据对照：

- [ ] 主张1：[事实] — 证据来源：[合同/邮件/微信/转账记录/其他]，页码/截图位置：[...]
- [ ] 主张2：...
- [ ] ...

无证据支持的事实主张 → 降低确定性表述（"据我方了解""初步显示"），或删除。

## Step 4 — 起草律师函

### 标准律师函结构

```
[律所LOGO/名称]
[律所地址、电话、邮箱]

律师函

[年份]XX律函字第[编号]号

致：[对方全称]
地址：[对方地址]

本律师受[委托人全称]（以下简称"委托人"）的委托，就贵司
[争议事由概要]事宜，在审查委托人提供的相关材料后，依据
中华人民共和国相关法律法规，特致此函。

一、基本事实

[以经核实的事实为基础，按时间顺序陈述。每项关键事实标注证据来源。
不使用主观臆断性语言，不使用侮辱性词语，不使用刑事定性（除非已
有生效刑事判决）。]

二、法律分析

[引用具体法律条款，结合事实进行分析。须通过北大法宝检索确认
或标注 [模型知识 — 请核实]。]

三、委托人诉求

[逐项列出具体要求。诉求须具体、可执行、有法律依据。]

1.
2.
3.

四、法律后果提示

[提示如对方不满足诉求可能面临的法律后果——但须限于法律救济
范围，不应构成胁迫或恫吓。]

请贵司于收到本函之日起 [N] 日内以书面方式回复，并就上述
事项与委托人协商解决。

[如希望对方直接联系律师：]
如有任何疑问或希望进一步沟通，请联系本律师：
电话：[律师电话]
邮箱：[律师邮箱]

特此函告。

[律所名称]（盖章）
律师：[律师姓名]（签名/签章）

日期：YYYY年MM月DD日
```

### 起草要点

1. **律师函编号：** 律所统一编号管理，便于归档和后续追溯。
2. **事实陈述：** 客观、准确，不夸大、不臆断。每一句事实陈述都能回溯到具体证据。
3. **法律引用：** 准确引用法条编号和内容，通过北大法宝确认 `[PKULaw]`。
4. **诉求明确：** 不说"希望你们依法处理"——说"请于X日内支付Y元至Z账户"。
5. **期限设定：** 合理期限（催款一般7-15天，违约整改一般15-30天，侵权停止一般7-14天）。
6. **语气把控：** 催款函宜先礼后兵；侵权警告需坚定但不过激；给对方留出场机会对后续谈判有利。

### 不写入律师函的内容

- ❌ "如不付款，我们将向公安机关举报你涉嫌诈骗"（→ 可能构成敲诈勒索）
- ❌ "我们将向媒体公开你的侵权行为"（→ 可能构成商业诋毁）
- ❌ 对对方人格的贬损性评价
- ❌ 超出法律救济范围的威胁
- ❌ 未经证实的刑事指控
- ❌ 委托人的保密信息或商业秘密

## Step 5 — 发送前检查清单

```
律师函发送前检查清单

□ 所有事实陈述有证据支持
□ 引用的法律条款已经核实（或已标注 [模型知识 — 请核实]）
□ 诉求合法、明确、可执行
□ 无可能构成敲诈勒索的表述
□ 无可能构成诽谤/名誉侵权的表述
□ 公司/委托人已审批函件内容
□ 律师函编号已登记
□ 发送方式已确认（快递/电子邮件/两者）
□ 快递面单/送达凭证须妥善保留（用于证明中断诉讼时效）
□ 如对方为多人，收件人列表完整

⚠️ 发送律师函是不可逆行为——发送后即产生法律效果（如中断诉讼时效、可能触发对方起诉等）。
确认无误后再发送。
```

## 注意事项

- 律师函以律师事务所名义发出（加盖律所公章），律师个人签名/签章。公司自行发送的函件不是"律师函"。
- 发送律师函的费用（快递费等）由委托人承担。
- 律师函本身不产生诉权——仅发送律师函不能阻止对方先起诉（如确认不侵权之诉、确认债务不存在的诉讼）。
- 《民法典》第195条规定，权利人主张权利（包括发送律师函）可中断诉讼时效——发函前确认时效状态，发函后保留送达凭证。

**参考法律渊源:**
- 《中华人民共和国民法典》合同编、侵权责任编 `[PKULaw]`
- 《中华人民共和国刑法》第221条、第274条、第307条之一 `[PKULaw]`
- 《中华人民共和国律师法》(2017修正) 第38条 `[PKULaw]`
- 中华全国律师协会《律师执业行为规范》(2009修订) `[PKULaw]`
