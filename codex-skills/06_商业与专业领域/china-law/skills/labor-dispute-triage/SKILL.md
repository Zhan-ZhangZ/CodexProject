# /labor-dispute-triage

对劳动争议进行快速分流，确定适用的争议解决路径、时效风险及举证责任。

## 指令

### 1. 争议信息
- 争议类型？（劳动报酬/解除争议/社保/工伤/竞业限制/确认劳动关系/其他）
- 争议金额？（与一裁终局适用相关）
- 关键时间节点？（争议发生日期、解除日期等）

### 2. 法律框架

#### 2.1 劳动争议范围（《劳动争议调解仲裁法》第2条 `[PKULaw]`）

- 因确认劳动关系发生的争议
- 因订立、履行、变更、解除和终止劳动合同发生的争议
- 因除名、辞退和辞职、离职发生的争议
- 因工作时间、休息休假、社会保险、福利、培训以及劳动保护发生的争议
- 因劳动报酬、工伤医疗费、经济补偿或者赔偿金等发生的争议
- 法律、法规规定的其他劳动争议

#### 2.2 争议解决路径

**仲裁前置原则（《劳动法》第79条 `[PKULaw]`）：**
劳动争议 → 调解（可选）→ 劳动仲裁（必须）→ 法院诉讼

**例外：直接可诉情形：**
- 劳动者以用人单位的工资欠条为证据，诉讼请求不涉及劳动关系其他争议 → 可直接起诉（司法解释一第15条 `[PKULaw]`）

#### 2.3 仲裁时效（《劳动争议调解仲裁法》第27条 `[PKULaw]`）

- **一般时效:** 1年，自当事人知道或应当知道权利被侵害之日起计算
- **中断事由:** 向对方主张权利 / 请求权利救济 / 对方同意履行义务 → 时效重新计算
- **中止事由:** 不可抗力 / 其他正当理由 → 中止原因消除后继续计算
- **劳动报酬特殊时效:** 劳动关系存续期间不受1年限制；劳动关系终止的，自终止之日起1年内提出

#### 2.4 一裁终局（《劳动争议调解仲裁法》第47条 `[PKULaw]`）

终局裁决（对用人单位有约束力）适用于：
- 追索劳动报酬、工伤医疗费、经济补偿或赔偿金，不超过当地月最低工资标准12个月金额
- 因执行国家的劳动标准在工作时间、休息休假、社会保险等方面发生的争议

劳动者对终局裁决不服 → 15日内可起诉（第48条）
用人单位 → 仅能向中级法院申请撤销（第49条 `[PKULaw]`），理由限于：
- 适用法律、法规确有错误
- 仲裁委无管辖权
- 违反法定程序
- 证据伪造
- 对方隐瞒足以影响公正裁决的证据
- 仲裁员有索贿受贿、徇私舞弊、枉法裁决行为

#### 2.5 举证责任

- **一般原则：** 谁主张谁举证
- **用人单位举证责任倒置情形（《劳动争议调解仲裁法》第6条）：**
  - 作出开除、除名、辞退、解除劳动合同、减少劳动报酬、计算劳动者工作年限等决定 → 用人单位举证
  - 工资支付记录 → 用人单位保管，用人单位举证
  - 加班事实 → 劳动者有初步证明义务，但用人单位掌握加班审批记录时应提供

### 3. 严重度评估

| 严重度 | 情形 |
|-------|------|
| 🔴 阻断性 | 仲裁时效已过且无中断/中止事由；一裁终局案件缺乏撤销理由 |
| 🟠 高 | 关键证据缺失；争议类型复杂涉及多重法律关系 |
| 🟡 中 | 证据部分缺失；争议金额较小但程序复杂 |
| 🟢 低 | 证据充分；有明确法律规定支持；和解可能性高 |

### 4. 输出

争议类型判定 + 适用程序路径 + 时效分析 + 举证责任分配 + 严重度评估 + 和解/仲裁/诉讼建议 + 决策树。
