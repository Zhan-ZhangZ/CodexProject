# /foreign-investment-review

审查外商投资项目的合规性——负面清单、审查程序、外汇合规、投资保护。

## 指令

### 1. 事务上下文
- 外国投资者类型？（外国自然人/外国企业/其他组织）
- 投资方式？（新设/并购/股权投资/新建项目/VIE）
- 投资行业与经营范围？
- 投资金额及持股比例？

### 2. 核心法律框架

#### 2.1 基本制度（《外商投资法》第4条 `[PKULaw]`）

国家对外商投资实行**准入前国民待遇加负面清单管理制度**：
- 准入前国民待遇 → 投资准入阶段给予外国投资者及其投资不低���本国投资者及其投资的待遇
- 负面清单 → 特定领域对外商投资实施的准入特别管理措施
- 负面清单之外 → 给予国民待遇（内外资一致原则）
- 负面清单由国务院发布或批准发布

#### 2.2 负面清单制度

**禁止投资领域（第28条 `[PKULaw]`）：**
外国投资者不得投资 → 违规后果（第36条）：
- 责令停止投资活动
- 限期处分股份、资产或采取其他必要措施恢复到实施投资前的状态
- 没收违法所得
- 依法承担相应法律责任

**限制投资领域：**
必须符合负面清单规定的条件（如持股比例上限、需合资、需中方控股等）

**负面清单以外的领域:**
按内外资一致原则实施管理

**外商投资企业定义（第2条）：**
全部或部分由外国投资者投资，依照中国法律在中国境内登记注册设立的企业

→ 2019年外商投资法出台后，VIE架构的法律地位仍存在灰色地带 `[审查]`

#### 2.3 投资保护（第20-24条）

- 外国投资者的投资、收益和其他合法权益 → 依法保护
- 征收条件：公共利益需要 + 法定程序 + 及时公平合理的补偿
- 知识产权保护：不得强制技术转让
- 行政机关及其工作人员对其知悉的商业秘密保密

#### 2.4 外汇管理

外商投资企业的资本项目外汇收支 → 遵守外汇管理条例
出资、增资、减资、转股、利润汇出 → 需经外汇登记/变更登记

#### 2.5 负面清单分析清单

| 审查项 | 参考来源 | 状态 |
|-------|--------|------|
| 投资领域是否在负面清单禁止类 | 最新版负面清单 | ✓/✗ |
| 投资领域是否在负面清单限制类 | 最新版负面清单 | ✓/✗ |
| 限制类条件是否满足（持股比例等） | 负面清单具体限制措施 | ✓/✗ |
| 是否触发国家安全审查 | 外商投资法第35条 | ✓/✗ |
| 是否构成经营者集中需反垄断申报 | 《反垄断法》 | ✓/✗ |
| 是否涉及特殊行业许可 | 行业监管规定 | ✓/✗ |

#### 2.6 外商投资纠纷（司法解释）

**隐名投资（外商投资企业纠纷司法解释(一)(2020修正) `[PKULaw]`）：**
- 负面清单禁止投资领域 → 隐名股东显名化请求不予支持
- 负面清单以外 → 如其他股东过半数以上同意 → 可判决名义股东履行变更登记义务
- 负面清单限制投资领域 → 需同时办理报批手续
