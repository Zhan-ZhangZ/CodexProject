# /nda-review

审查保密协议的条款完整性、公平性和法律风险。

## 指令

### 1. 事务上下文
- 保密协议方向？（单向披露/单向接收/双向互惠）
- 交易背景？（商业合作谈判/尽职调查/雇佣关系/投资评估/技术合作）
- 商业秘密范围？（何种信息需要保护）

### 2. 核心审查清单

#### 2.1 保密信息定义

| 审查点 | 立场 | 最佳实践 |
|-------|------|--------|
| 保密信息范围 | 披露方→宽；接收方→窄 | 不可过分宽泛导致履行不能 |
| 口头信息的书面确认 | 披露方→不需要；接收方→要求 | 口头保密信息需在X日内书面确认 |
| 排除条款 | 接收方→需要 | 已公开/独立开发/第三方合法获取/法律要求披露 |

#### 2.2 保密义务

- 使用限制：保密信息仅可用于约定的目的
- 保密期限：约定保密期限 vs. 商业秘密永久保密
- 披露限制：仅向"需要知悉"的相关人员披露
- 安全保障：合理的保密措施

#### 2.3 残余信息

- 接收方人员记忆中的一般知识、技术、经验是否纳入保密范围？
- 接收方立场 → 应明确排除残余信息

#### 2.4 返还原件/销毁

- 保密期结束时是否要求返还或销毁所有保密信息和副本？
- 是否有合规审查例外（保留1份用于合规存档）？
- 电子备份是否豁免？

#### 2.5 竞业禁止条款

**⚠️ 注意：部分保密协议中隐含变相竞业限制条款。**
- 真正的保密义务 vs. 实质上的不竞争义务
- 如实质构成竞业限制 → 需要单独补偿
- 《反不正当竞争法》第9条 `[PKULaw]`：商业秘密侵权需证明"不正当手段获取"

#### 2.6 违约救济

- 禁令救济：是否有权申请行为保全？
- 违约金还是实际损失赔偿？
- 《反不正当竞争法》(2025修订)第22条 `[PKULaw]`：侵犯商业秘密法定赔偿最高500万元

### 3. 严重度分类

| 严重度 | 情形 |
|-------|------|
| 🔴 阻断性 | 保密范围无限扩大至非保密信息；竞业限制变相条款；缺乏排除条款 |
| 🟠 高 | 保密期限过长（超过商业周期）；使用限制缺失；无返还义务 |
| 🟡 中 | 部分条款模糊；残余信息未排除 |
| 🟢 低 | 格式优化；定义精确度 |

### 4. 输出

审查者说明 + NDA关键条款评估 + 严重度发现清单 + 修改建议 + 决策树。

**参考法律渊源:**
- 《民法典》合同编（保密义务：第501条 `[PKULaw]`）
- 《反不正当竞争法》(2025修订)第10条、第22条 `[PKULaw]`
- 《最高人民法院关于审理侵犯商业秘密民事案件适用法律若干问题的规定》第10条 `[PKULaw]`
- 《劳动合同法》第23条、第24条（竞业限制边界）`[PKULaw]`
