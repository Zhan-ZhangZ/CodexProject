# /real-estate-construction

1. 读取 `~/.claude/plugins/config/claude-for-legal/china-law/CLAUDE.md` → 管辖与法律框架。
2. 确定房地产/建设工程事项类型。

4. 进行法律风险评估。
5. 出具附风险分级和应对建议的分析报告。

## Purpose

中国的房地产和建设工程法律是法律渊源的复合体——从《民法典》物权编的基础性规定，到《城市房地产管理法》《土地管理法》《建筑法》等单行法，再到大量的行政法规、部门规章和司法解释。特别是建设工程领域，最高人民法院的司法解释（如《建设工程施工合同司法解释(一)》）在实践中具有至少等同于法条的影响力。本技能提供中国房地产和建设工程法律框架下的系统分析方法。

## 审查类型

| 类型 | 适用场景 |
|------|---------|
| 土地使用权取得 | 出让/划拨/转让/租赁土地使用权的法律路径分析 |
| 房地产开发合规 | 开发全流程合规审查（立项→土地→规划→施工→预售→竣工→交付） |
| 建设工程合同 | EPC/施工总承包/分包合同的签订和履行风险 |
| 工程价款纠纷 | 工程款支付、工程价款优先受偿权、结算争议 |
| 商品房预售/现售 | 预售许可、认购书/预售合同合规、交付条件 |
| 商业地产租赁 | 租赁合同审查、租约稳定性、转租/转让限制 |
| 不动产抵押 | 抵押设立、抵押物处置、多抵押权顺位 |
| 物业管理 | 物业管理合规、业委会与物业公司关系 |

## Step 2 — 土地使用权取得

### 2.1 土地权属体系

《土地管理法》(2019修正) `[PKULaw]` + 《民法典》物权编 `[PKULaw]`：

```
中国土地权利体系

土地所有权
├── 国家所有（城市土地 + 法律规定属于国家的农村土地）
└── 农民集体所有（农村宅基地、农用地、集体建设用地）

土地使用权（用益物权）
├── 国有建设用地使用权 ← 房地产市场核心
├── 集体建设用地使用权（2020年改革后可入市）
├── 宅基地使用权
├── 土地承包经营权
└── 地役权
```

### 2.2 国有建设用地使用权取得方式

| 方式 | 适用情形 | 特点 |
|------|---------|------|
| 出让 | 经营性用地（商业/旅游/娱乐/商品住宅） | 招拍挂（招标/拍卖/挂牌），有偿，有期限 |
| 划拨 | 公共利益/军事/国家机关/城市基础设施 | 无偿，无期限，但转让受限 |
| 转让 | 从已有使用权人处取得 | 需满足25%开发投入等条件 |
| 租赁 | 短期用地 | 租金按年缴纳，期限较短 |

### 2.3 土地使用期限

《城镇国有土地使用权出让和转让暂行条例》第12条 `[PKULaw]`：

| 用途 | 最高年限 |
|------|---------|
| 居住用地 | **70年** |
| 工业用地 | **50年** |
| 教育/科技/文化/卫生/体育用地 | **50年** |
| 商业/旅游/娱乐用地 | **40年** |
| 综合/其他用地 | **50年** |

**⚠️ 续期问题：** 住宅建设用地使用权期限届满的，**自动续期**（《民法典》第359条）。续期费用的缴纳或减免，依照法律、行政法规的规定办理——**目前具体规定尚未出台**，存在制度不确定性。

### 2.4 集体建设用地入市（2020年改革）

《土地管理法》(2019修正) 第63条 `[PKULaw]`：
- 集体经营性建设用地可出让、出租给集体经济组织以外的单位或个人
- 须经本集体经济组织成员的村民会议**三分之二以上成员或三分之二以上村民代表**同意
- 与国有建设用地"同权同价"

## Step 3 — 建设工程合同

### 3.1 合同类型

| 合同类型 | 适用场景 | 主要法律 |
|---------|---------|---------|
| 施工总承包合同 | 发包人将全部施工任务发包给一个施工总承包单位 | 《民法典》第788-808条 `[PKULaw]` |
| EPC总承包 | 工程总承包（设计-采购-施工一体化） | 住建部《房屋建筑和市政基础设施项目工程总承包管理办法》 |
| 专业分包 | 总承包人将专业工程分包给专业承包单位 | 《建筑法》第29条 |
| 劳务分包 | 分包劳务作业（不含主要材料和设备） | 司法解释(一)第5条 |

### 3.2 合同效力与无效情形

《民法典》第153条 + 《建设工程施工合同司法解释(一)》第1-7条 `[PKULaw]`：

| 无效情形 | 法律依据 | 后果 |
|---------|---------|------|
| 承包人未取得建筑业企业资质或超越资质等级 | 司法解释第1条第1项 | 合同无效，但工程验收合格→可参照合同约定折价补偿 |
| 没有资质的实际施工人借用有资质的建筑施工企业名义（"挂靠"） | 司法解释第1条第2项 | 合同无效 |
| 必须招标而未招标或中标无效 | 司法解释第1条第3项 | 合同无效 |
| 转包 | 司法解释第1条 | 合同无效 |
| 违法分包 | 司法解释第1条 | 合同相关条款无效 |

**⚠️ 合同无效但工程质量合格的特别规则：** 建设工程施工合同无效，但建设工程经验收合格的，可以参照合同关于工程价款的约定折价补偿承包人——这是建设工程合同特有的"无效合同有效化处理"规则。

### 3.3 违法发包、转包、违法分包和挂靠

住建部《建筑工程施工发包与承包违法行为认定查处管理办法》`[PKULaw]`：

| 行为 | 定义 | 典型表现 |
|------|------|---------|
| 违法发包 | 建设单位将工程发包给个人/无资质单位/应招标未招标等 | 肢解发包、指定分包 |
| 转包 | 承包单位不履行合同约定的责任和义务，将全部工程转给他人 | 以内部承包名义的转包 |
| 违法分包 | 分包给无资质单位/分包单位再分包/主体结构分包等 | 劳务分包中包含材料款 |
| 挂靠 | 无资质单位借用有资质单位名义承揽工程 | "管理费"模式 |

### 3.4 工程价款优先受偿权

《民法典》第807条 `[PKULaw]` + 《建设工程施工合同司法解释(一)》第35-42条：

| 要点 | 内容 |
|------|------|
| 权利主体 | 与发包人订立建设工程施工合同的承包人（不含实际施工人） |
| 优先范围 | 工程价款（含工作人员报酬、材料款等实际支出费用） |
| 顺位 | **优先于抵押权和其他债权**（但劣后于商品房消费者的购房款返还请求权） |
| 行使期限 | **自发包人应当给付工程价款之日起18个月内**（最长除斥期间，不适用中止/中断） |
| 行使方式 | 协议折价或申请法院拍卖 |
| 放弃限制 | 放弃损害建筑工人利益的，放弃无效 |

## Step 4 — 商品房预售/现售

### 4.1 预售条件

《城市房地产管理法》第45条 `[PKULaw]` + 《城市商品房预售管理办法》：

| 条件 | 内容 |
|------|------|
| 已交付全部土地使用权出让金，取得土地使用权证书 | 土地使用合法 |
| 持有建设工程规划许可证和施工许可证 | 建设手续完备 |
| 投入开发建设资金达工程建设总投资的**25%以上** | 形象进度要求 |
| 确定施工进度和竣工交付日期 | 可预期交付 |
| 取得《商品房预售许可证》 | 必须持证预售 |

**⚠️ 无证预售后果：** 预售合同可被认定无效（《最高人民法院关于审理商品房买卖合同纠纷案件适用法律若干问题的解释》第2条 `[PKULaw]`）——但起诉前取得预售许可的，可以认定有效。

### 4.2 商品房买卖合同关键条款

| 审查项 | 要点 |
|-------|------|
| 计价方式 | 按建筑面积/套内建筑面积/套（单元）计价 |
| 面积误差处理 | 绝对值3%以内据实结算；超出3%→买受人有权退房（司法解释第13条） |
| 交付条件 | 竣工验收备案 + 《住宅质量保证书》+《住宅使用说明书》 |
| 逾期交房 | 违约金标准（通常为已付房款的日万分之1-3） |
| 逾期付款 | 违约金 + 超过合理期限出让人可解除合同 |
| 办证期限 | 交付后90日（现房）/90日（期房）内办理不动产权证 |

### 4.3 购房款返还请求权的超级优先权

《最高人民法院关于商品房消费者权利保护问题的批复》(2023) `[PKULaw]`：

```
商品房消费者权利顺位

1. 商品房消费者的房屋交付请求权
   （以居住为目的 + 已支付全部价款，或仅支付部分价款且一审
    法庭辩论终结前已支付剩余价款）
2. 建设工程价款优先受偿权
3. 抵押权
4. 其他普通债权

⚠️ 如果房屋不能交付且无实际交付可能：
   → 商品房消费者的购房款返还请求权优先于工程价款优先受偿权
```

## Step 5 — 商业地产租赁

### 5.1 《民法典》租赁合同基本规定

| 要点 | 内容 | 法条 |
|------|------|------|
| 最长租期 | **20年**（超过部分无效） | 第705条 |
| 买卖不破租赁 | 租赁物在租赁期间发生所有权变动的，不影响租赁合同的效力 | 第725条 |
| 承租人优先购买权 | 出租人出卖租赁房屋的，应在出卖前合理期限内通知承租人，承租人享有以同等条件优先购买的权利 | 第726条 |
| 转租 | 须经出租人同意；未经同意转租的，出租人可以解除合同 | 第716条 |
| 续租权 | 租赁期限届满，房屋承租人享有以同等条件优先承租的权利 | 第734条 |

### 5.2 商业租赁特殊关注

| 审查项 | 要点 |
|-------|------|
| 免租期 | 明确免租期范围（是否含物业费） + 提前解约时免租期租金的返还义务 |
| 转租/分租 | 是否允许（商业运营中分租多为常态） |
| 业态限制 | 是否限制特定业态或禁止与某竞争性业态共存 |
| 营业额提成租金 | 混合租金结构（底租+提成）→ 营业额定义和审计权 |
| 恢复原状 | 退租时的装修处理（拆除/保留/折价） |
| 续租权 | 是否赋予续租期权及续租租金确定机制 |

## 注意事项

- 工程价款优先受偿权是18个月的除斥期间（不变期间）——错过即丧失，不适用诉讼时效中止、中断规则。工程项目应建立优先权行使期限追踪机制。
- 商品房消费者的超级优先权（2023年司法解释）是房地产企业的重大风险——在项目烂尾时，即使有工程价款优先受偿权和抵押权的权利人，也要排在购房消费者的购房款返还请求权之后。
- 挂靠经营在建设工程领域普遍存在但法律明确禁止——被挂靠企业对挂靠人的工程质量和对外债务承担连带责任的安排日趋严格。
- 集体建设用地入市改革进行中——地方实施细则差异大，在具体项目上须查询当地政策。
