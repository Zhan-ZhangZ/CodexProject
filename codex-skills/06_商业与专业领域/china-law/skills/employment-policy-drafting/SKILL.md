# /employment-policy-drafting

起草或审查劳动规章制度，确保其合法性和程序合规性。

## 指令

### 1. 事务上下文
- 制度类型？（员工手册/考勤制度/薪酬制度/奖惩制度/保密制度/竞业管理制度/培训制度）
- 已有草案还是从零起草？
- 公司所在地？（地方规定可能有特殊要求）
- 是否有工会或职工代表大会？

### 2. 核心法律框架

#### 2.1 规章制度的效力要件（《劳动合同法》第4条 `[PKULaw]`）

涉及劳动者切身利益的规章制度或重大事项：
- 应经**职工代表大会或全体职工讨论**，提出方案和意见
- 与**工会或职工代表**平等协商确定
- 应当**公示或告知**劳动者

→ 未经民主程序和公示的规章制度 → 不能作为解除劳动合同等处理的依据（司法解释一第50条 `[PKULaw]`）

#### 2.2 内容合法性审查清单

| 审查项 | 常见违法风险 |
|-------|------------|
| 试用期规定 | 期限超出法定上限；重复约定 |
| 加班制度 | 强制加班、不支付加班费、以调休取代法定节假日加班费 |
| 休假制度 | 年假天数低于法定标准；限制婚假/产假 |
| 奖惩制度 | 罚款超出合理性（中国法律限制用人单位罚款权）；体罚或变相体罚 |
| 薪酬制度 | 工资低于当地最低工资标准；工资结构不透明 |
| 保密制度 | 保密范围过宽；无补偿 |
| 竞业限制 | 人员范围超出高管/高技/保密人员；未约定补偿 |
| 离职程序 | 违法扣留档案或社保；违法约定违约金 |

#### 2.3 严重违纪认定标准

规章制度中"严重违纪"的界定直接影响解除劳动合同的合法性：
- 应量化标准（如"连续旷工X天"或"年度内累计旷工X天"）
- 不应笼统规定"视情节轻重处理"
- 处罚应与过错程度成比例

#### 2.4 程序文件要求

需保留的程序证明文件：
- 职工代表大会/职工大会讨论记录
- 工会或职工代表协商记录
- 公示证明（签收记录/公告栏照片/内部OA系统公告截图）
- 劳动者签收确认

### 3. 严重度分类

| 严重度 | 情形 |
|-------|------|
| 🔴 阻断性 | 内容违反法律强制性规定；未经过民主程序和公示 |
| 🟠 高 | 程序存在瑕疵（如仅公示未经民主讨论）；关键条款缺失 |
| 🟡 中 | 表述不清晰可能导致争议；条款与模板不一致 |
| 🟢 低 | 格式优化；条款顺序调整 |

### 4. 输出

内容合法性检查清单 + 程序合规性评估 + 修改建议（带具体条款语言）+ 程序操作指引 + 决策树。

**参考法律渊源:**
- 《劳动合同法》第4条 `[PKULaw]`
- 《最高人民法院关于审理劳动争议案件适用法律问题的解释(一)》第50条 `[PKULaw]`
- 《劳动合同法》第38条、第39条（关于规章制度作为解除依据）`[PKULaw]`
