# arduino-cli-mcp — 电气与智能化设计 MCP 服务

- **项目主页**: https://github.com/Oliver0804/arduino-cli-mcp

## 功能说明

封装 Arduino CLI 工具链的嵌入式开发 MCP。支持管理板卡开发包与第三方依赖库，编译并直接上传固件至 Arduino、ESP8266/ESP32 等开发板，且提供串口输出读取功能。
