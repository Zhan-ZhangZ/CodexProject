# shopify-mcp — 跨境电商与店铺管理 MCP 服务

- **项目主页**: https://github.com/GeLi2001/shopify-mcp

## 功能说明

Shopify 独立站后台全功能管理 MCP。基于 GraphQL Admin API 深度封装，支持产品 (Products)、订单 (Orders) 和客户 (Customers) 的全生命周期管理，支持更新 Metafields 扩展属性及库存修改。
