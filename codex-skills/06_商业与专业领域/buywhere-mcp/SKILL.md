# buywhere-mcp — 跨境电商与店铺管理 MCP 服务

- **项目主页**: https://github.com/BuyWhere/buywhere-mcp

## 功能说明

东南亚跨境电商多平台竞品比价与选品 MCP。支持在 Shopee、Lazada 等主流电商品牌上跨渠道对比同款商品的技术参数、价格分布及实时库存，辅助爆款发掘与竞品定价决策。
