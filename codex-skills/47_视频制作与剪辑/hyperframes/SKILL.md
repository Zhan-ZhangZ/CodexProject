# hyperframes

- **项目主页**: https://github.com/heygen-com/hyperframes

## 功能说明
一句话生成动效视频。告诉 Codex 视频主题，它可以使用 HTML、CSS 和动画制作产品介绍、动态海报、知识视频和 PPT 风格视频，最后渲染为 MP4。很适合把文章、推文、产品介绍快速改成视频。
