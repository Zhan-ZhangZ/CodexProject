# OPCUA-MCP — 电气与智能化设计 MCP 服务

- **项目主页**: https://github.com/midhunxavier/OPCUA-MCP

## 功能说明

标准工业 OPC UA 客户端 MCP。支持自动浏览现场设备节点命名空间，高频读取/写入设备数据项，并可直接调用 OPC UA 服务器内部暴露的方法进行状态切换。
