# pcbparts-mcp — 电气与智能化设计 MCP 服务

- **项目主页**: https://github.com/Averyy/pcbparts-mcp

## 功能说明

跨平台多源元器件与引脚封装搜索 MCP。整合了嘉立创 (JLCPCB)、Mouser 和 DigiKey 数据源，支持器件的参数化搜索、多渠道选型，并可拉取 EasyEDA 的原理图符号引脚定义。
