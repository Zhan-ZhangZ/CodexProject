# Basics of Noise Countermeasures [Lesson 6] Common Mode Choke Coils

## Common mode choke coils separate noise from signals using conduction modes

Rather than using differences in frequency, common mode choke coils separate noise from signals by differences in conduction mode.

## Common modes and differential modes

In the electrical circuit of a circuit board, the current flowing out from a certain part reaches another circuit through the load, and returns to the origin via a different route (often the ground plane). This type of flow is called differential mode (or normal mode).

Another conduction route also exists. A tiny amount of stray capacitance is generated between wires on the circuit board and the reference ground surface, creating a conduction route where current flows commonly through all wires and returns via the reference ground surface. This route is called common mode.

Although stray capacitance is quite small, impedance drops as frequency rises, so common mode current flows more easily at high frequencies. If a cable is externally connected to the circuit, common mode current will also flow through it. As it has electrical potential with respect to ground, it will be released as noise radio waves.

## Working principles

Common mode choke coils are made of two conducting wires wrapped around a single core (ferrite for high-frequency applications). They have four terminals. The wires are wrapped in opposite directions. When common mode currents flow, flux generated by each coil reinforces (same direction), increasing inductor action. Differential mode currents generate opposing flux that cancels out — it no longer acts as an inductor. Result: acts as inductor for common mode only, transparent to differential mode.

## Advantages

1. Even when signal and noise frequencies overlap, their different conduction modes enable suppression of only noise.
2. Performance does not decrease even with large differential mode current, as the core does not become saturated.

Common mode choke coils can separate signals from noise using transmission modes, so high-speed differential signals (USB, SATA, HDMI) are not affected. The problematic noise is mostly common mode, so these coils effectively suppress it without affecting high-speed signals.

For power lines with large currents: inductor-type filters that act on differential modes (ferrite beads, normal mode choke coils) suffer magnetic saturation. Common mode choke coils don't, because differential mode flux cancels out.

## Precautions

In reality, some flux leaks and is not cancelled, resulting in small differential mode inductance. This must be considered for extremely high-frequency signals (Display Port, USB 3.0). Chip-type common mode choke coils have been developed that suppress differential mode impedance.
