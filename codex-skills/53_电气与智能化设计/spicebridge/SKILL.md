# spicebridge — 电气与智能化设计 MCP 服务

- **项目主页**: https://github.com/clanker-lover/spicebridge

## 功能说明

基于 ngspice 的电路设计与仿真分析 MCP。支持使用自然语言描述电路结构，自动生成标准 SPICE 网表并运行 AC、DC 或瞬态仿真，可分析提取波形数据甚至导出为 KiCad 格式。
