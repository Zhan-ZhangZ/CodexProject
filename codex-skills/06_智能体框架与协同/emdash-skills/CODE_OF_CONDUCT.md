# Code of Conduct

## The Short Version

Be respectful. Be constructive. Ship code, not drama.

## Standards

**Do:** give constructive feedback, respect differing viewpoints, focus on what's best for the project, show empathy.

**Don't:** harass, troll, use sexualized language, publish others' private info, or engage in conduct that would be inappropriate in a professional setting.

## Enforcement

Instances of unacceptable behavior may be reported to **hey@megabyte.space**. All complaints will be reviewed. Maintainers who don't follow the Code of Conduct may be removed.

Adapted from the [Contributor Covenant](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).
