# 小红书热点营销推文 Skill 说明文档

## 一、产品简介

本 Skill 用于辅助生成符合当前小红书平台风向的营销推文。

它的核心能力不是直接“凭空写文案”，而是先结合小红书当日热门内容，提炼当前平台流行的选题方向、表达结构和内容包装方式，再根据客户指定的产品与写作风格，生成更贴近平台生态的营销文案。

适用场景：
- 品牌方进行小红书内容营销
- 运营团队快速产出热点跟风内容
- 产品推广阶段需要批量形成种草文案思路
- 代运营团队进行日常内容策划与文案生成

## 二、Skill 可完成的工作

该 Skill 可完成以下完整流程：

1. 启动本地小红书 MCP 服务
2. 在首次使用时辅助完成账号登录
3. 获取当前热门小红书内容
4. 总结当前热门笔记的内容风向
5. 主动询问需要推广的产品或服务
6. 主动询问需要采用的写作风格
7. 输出完整的小红书营销推文

最终输出通常包含：
- 标题
- 正文
- 推荐配图方向
- 推荐话题标签

## 三、Skill 的核心价值

### 1. 更贴近平台风向

传统文案生成往往只关注产品卖点，容易写成通用广告文案。

本 Skill 会先观察当前小红书热门内容，再进行创作，因此输出的文案更接近平台真实流行表达，更容易贴合用户阅读习惯。

### 2. 兼顾热点与产品转化

它不是只做热点总结，也不是只做产品广告，而是将两者结合：
- 先理解平台当下流行什么
- 再把客户产品嵌入到适合当前风向的内容表达中

### 3. 降低内容策划成本

原本需要运营人员手动查看热点、整理风向、构思角度、设计标题、写正文。

现在可以由 Skill 协助完成热点分析和初稿生成，明显缩短内容生产时间。

## 四、标准工作流程

### 第一步：准备运行环境

Skill 会检查本地是否存在小红书 MCP 所需程序。

如果本地没有对应程序，系统可按配置自动尝试：
- 下载适配当前平台的预编译版本
- 或从指定代码仓库拉取源码并构建

### 第二步：首次登录账号

如果是首次使用，或当前账号会话已失效，Skill 会先引导完成小红书账号登录。

只有在登录成功后，才会继续进行热门内容分析。

### 第三步：获取热门内容

Skill 会调用小红书 MCP 服务，读取当前热门或最新热点内容，并提炼：
- 高频主题
- 标题钩子
- 常见表达结构
- 常见内容包装形式
- 用户容易被打动的触发点

### 第四步：输出“热门风向”总结

在正式写推文之前，Skill 会先给出一份简要的热门风向分析，帮助客户理解当前平台流行方向。

### 第五步：追问产品与风格

在完成风向总结后，Skill 会主动追问两个必要信息：
- 需要推广什么产品或服务
- 希望采用什么风格来写

例如：
- 种草风
- 专业测评风
- 真实自述风
- 情绪共鸣风
- 高端克制风
- 爆款标题风

### 第六步：输出最终营销推文

在明确产品和风格后，Skill 会生成最终文案，确保内容尽量符合当前小红书平台语境。

## 五、输出结果说明

Skill 输出的内容一般包括以下几个部分：

### 1. 标题

用于吸引点击，尽量贴近当前平台热门标题结构。

### 2. 正文

结合产品卖点、使用场景、信任证明和行动引导，生成完整的推广文案。

### 3. 推荐配图方向

提供建议的图片内容方向，方便后续设计或拍摄。

### 4. 推荐话题

提供适合发布时使用的话题标签建议。

## 六、适合的客户类型

本 Skill 特别适合以下客户：
- 小红书品牌投放客户
- 电商产品商家
- 美妆、护肤、家居、母婴、食品等高内容化品类
- 小红书代运营团队
- 需要高频产出种草内容的营销团队

## 七、使用前提

为保证 Skill 正常运行，通常需要具备以下条件：
- 可运行本地程序的环境
- 可访问的小红书 MCP 程序
- 首次使用时完成账号登录
- 具备基础网络连接能力

如果客户侧需要长期稳定使用，建议提前完成：
- 环境部署
- MCP 程序准备
- 登录状态确认
- 账号使用规范确认

## 八、交付边界说明

本 Skill 的主要职责是：
- 获取热点内容
- 总结平台风向
- 生成营销推文

本 Skill 默认不包含以下内容，除非后续单独扩展：
- 自动发布到小红书
- 自动生成配图
- 自动评论、点赞、互动
- 多账号投放管理
- 数据回收与投放效果分析

如有需要，这些能力可以作为后续扩展方向继续开发。

## 九、注意事项

1. 热点内容具有时效性，不同日期生成的风向总结可能不同。
2. 若账号未登录或登录失效，Skill 无法正常获取平台内容。
3. 若平台接口或运行环境发生变化，可能需要调整配置或更新 MCP 程序。
4. 文案生成结果建议由运营人员做最终审核后再正式发布。

## 十、总结

这个 Skill 的价值，在于把“小红书热点观察”和“营销文案生成”合并为一条自动化流程。

它能够帮助客户更快地产出贴近平台风向的推广内容，减少人工搜热点、想角度、写文案的时间成本，同时提升内容与平台生态的匹配度。

如果客户后续有更高阶需求，也可以在此基础上继续扩展成完整的小红书内容运营工作流。
