# Hello World 验证技能

## 概述

此技能作为你创建的自定义 Codex 插件市场的测试技能。如果用户看到此技能被成功激活并能够运行，说明你的自定义插件市场仓库已经被 Codex 成功导入并解析。

## 工作流程

### 1. 友好致意

当用户发送包含 "Say hello from my custom plugin" 或类似的验证请求时，使用此技能响应。
向用户发出热情的问候，格式如下：
> 🎉 **恭喜！你的自定义 Codex 插件市场已成功导入！**

### 2. 显示当前环境信息

为了向用户证明你正在与之交互，请输出以下信息：
*   **当前本地时间**：获取并显示当前系统的本地时间。
*   **运行平台**：显示用户的操作系统类型（例如 macOS）。
*   **工作空间路径**：显示当前工作空间路径（`/Users/zz/codexproject`）。

### 3. 提供下一步操作建议

向用户简单介绍如何在此插件市场仓库中添加更多插件或技能：
1.  在 `plugins/codex/plugins/` 下创建新的插件文件夹。
2.  在 `.agents/plugins/marketplace.json` 中注册新插件的名称和相对路径。
3.  在插件的 `.codex-plugin/plugin.json` 中配置插件元数据，并在 `skills/` 下添加你的技能 Markdown 文件。
